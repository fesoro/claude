# İngilis Dili Öyrənmə Materialları

Bu layihə Azərbaycandilli istifadəçilər üçün sıfırdan C2 səviyyəsinə qədər əhatəli ingilis dili öyrənmə resursu kimi hazırlanmışdır. Bütün izahatlar Azərbaycanca yazılmış, nümunələr isə ingilis dilindədir.

---

## Hara başlamaq lazımdır?

### Tam yeni başlayanam
→ [A1 səviyyəsi](levels/a1/README.md) ilə başlayın  
→ Sonra [Öyrənmə Bələdçisi](learning-guide.md) oxuyun

### Əvvəl öyrənmişəm, amma unutmuşam
→ [Ümumi Roadmap](roadmap.md) vasitəsilə öz səviyyənizi müəyyən edin  
→ Uyğun [Levels](levels/) qovluğuna keçin

### Qrammatikaya fokuslanmaq istəyirəm
→ [Qrammatika Roadmap](grammar/roadmap.md) ilə başlayın  
→ [Qrammatika Mövzuları](grammar/README.md) siyahısına baxın

## Bu Materiallardan Necə İstifadə Etmək

1. **Səviyyənizi müəyyən edin** → [roadmap.md](roadmap.md)
2. **Həmin səviyyənin README-sini oxuyun** → gündəlik plan qurun
3. **Hər gün 1 unit keçin** → qrammatika faylını oxuyun
4. **Lüğət öyrənin** → Anki ilə spaced repetition edin
5. **Tərəqqinizi izləyin** → hər səviyyənin `progress-test.md` faylı ilə
