# Video Call English (Remote Work)

Real phrases for video calls — the ones that feel awkward if you don't know them.

---

## 1. Zəngi Açarkən (Joining a Call)

### Standart Açılış

```
Hi everyone, can you hear me?
```
```
Good [morning/afternoon] — am I audible?
```
```
Hey, can you see and hear me okay?
```

---

### Gec Qoşulduqda

```
Sorry I'm a couple minutes late — what did I miss?
```
```
Apologies for being late. Please carry on — I'll catch up.
```
```
Sorry for the delay. Are you still on [topic]?
```

---

### Hamı Gəlibmi?

```
Am I the last one?
```
```
Are we still waiting on anyone?
```
```
I think we're all here — should we get started?
```

---

### Zəngi Başlatmaq

```
Alright, let's get started if everyone's ready.
```
```
Let's kick off — we've got a lot to cover.
```
```
Okay, I'll share my screen in a moment. First, quick agenda check:
```

---

## 2. Texniki Problemlər (Technical Issues)

### Audio Problemi — Başqası Mute-dadır

```
I think you're on mute.
```
```
We can't hear you — you might be muted.
```
```
Your mic seems to be off.
```

---

### Audio Kəsilir

```
Your audio is cutting out.
```
```
You're breaking up a bit — could you repeat that?
```
```
I'm having trouble hearing you — there's a lot of static.
```
```
Sorry, you dropped out — could you say that again from [word]?
```

---

### Öz Audio Probleminiz

```
Can everyone hear me clearly?
```
```
Is my audio okay?
```
```
My headset just disconnected — give me a second.
```
```
Let me switch to a different audio input — one moment.
```

---

### Video Problemi

```
My video isn't working — I'll stay audio-only for now.
```
```
Sorry, my camera is having issues — I'm here though.
```
```
Your video is frozen.
```
```
Your video quality is very low — are you on a bad connection?
```

---

### Ekran Paylaşma Problemi

```
Can you see my screen?
```
```
I'm not seeing your screen — could you try sharing again?
```
```
Let me stop and reshare — something looks off on my end.
```

---

### Connection Problemi

```
I'm having connection issues — I'll rejoin in a moment.
```
```
Sorry, I dropped — what did I miss?
```
```
My internet is unstable — if I drop out, I'll be back shortly.
```
```
Let me switch to mobile hotspot — give me 30 seconds.
```

---

## 3. Sözü Almaq / Başqasını Kəsmək (Taking the Floor / Interrupting Politely)

### Söz Gözləmək

```
If I could just add something here...
```
```
I have a quick point when you get to a stopping place.
```
```
Can I jump in for a second?
```

---

### Nəzakətlə Kəsmək

```
Sorry to interrupt, but I think this is important —
```
```
Just to add to that point —
```
```
If I can just quickly say —
```
```
Before we move on, I wanted to mention —
```

---

### Sizi Kəsdilər

```
Let me just finish this thought —
```
```
If I could just finish —
```
```
Sorry, I'll be quick — [finish point] — okay, you were saying?
```

---

### Fikirlərinizi Geri Almaq

```
Sorry, where was I? Ah yes —
```
```
Right, so as I was saying —
```
```
Back to my point —
```

---

### Başqasına Söz Vermək

```
[Name], what do you think about this?
```
```
I'd like to hear [Name]'s take on this.
```
```
[Name], you were about to say something?
```

---

## 4. Ekranı Paylaşmaq (Screen Sharing)

### Başlamaq

```
Let me share my screen.
```
```
I'll pull up the diagram — one second.
```
```
Can everyone see that?
```
```
Is the screen sharing working?
```

---

### Naviqasiya Edərkən

```
I'll zoom in so it's easier to read.
```
```
Let me scroll down to the relevant part.
```
```
If you look at the left side here —
```
```
This part is the key piece — I'll highlight it.
```
```
Bear with me — I need to switch tabs.
```

---

### Bitirmək

```
Let me stop sharing.
```
```
I'll hand back the screen.
```
```
Okay, I'm done — who wants to share next?
```

---

### Başqasından Ekranını Paylaşmasını İstəmək

```
Could you share your screen so we can see the error?
```
```
Would you mind pulling up the PR?
```
```
Can you show us what you're seeing on your end?
```

## 5. Anlamadığınız Zaman (When You Don't Understand)

Non-native speaker olaraq — bu çox vacibdir. Normal bir şeydir, utanmaq lazım deyil.

### Connection / Audio Problem Bahanəsi

```
Sorry, could you repeat that? You were breaking up.
```
```
I missed the last part — could you say that again?
```
```
Sorry, my audio glitched — what were you saying after [word]?
```

**Niyə bu faydalıdır:** Connection problem real olur — bu özündən əmin görünməyə imkan verir.

---

### Açıqca Soruşmaq

```
Could you slow down a bit? I want to make sure I understand correctly.
```
```
Could you explain what you mean by [term]?
```
```
I'm not familiar with [X] — could you give a quick overview?
```

---

### Yazılı Sorğu

```
Could you put that in the chat for me?
```
```
Could you drop the link in the chat?
```
```
Would you mind sending me that after the call? I want to make sure I have it right.
```

---

### Anladığını Yoxlamaq

```
Just to confirm — you're saying [restate their point]?
```
```
So if I understand correctly, the plan is [X]?
```
```
Let me make sure I've got this right: [restate].
```

---

### Ümumi Qayda

Non-native speaker olaraq, suallar vermək **professional görünür**, zəiflik deyil.
"I want to make sure I understand correctly" — bu cümləni işlət. Hər kəs bunu eşitməyi xoşlayır.

---

## 6. Görüşü Bitirmək (Ending the Call)

### Wrapping Up

```
I think we've covered everything.
```
```
Let's wrap up — we're coming up on time.
```
```
I'll let everyone go — we're at the hour.
```
```
Any final questions before we close?
```

---

### Summary / Next Steps

```
I'll send a summary in Slack.
```
```
Let me recap the action items before we end:
```
```
I'll update the Notion doc with what we decided.
```
```
[Name] — you're taking [X], right? And I'll handle [Y]?
```

---

### Tez Ayrılmaq

```
I need to drop at [time] — hard stop.
```
```
I'll have to jump off in 5 — just a heads up.
```
```
I've got another call — I'll catch up on the recording.
```

---

### Görüşdən Sonra

```
Thanks everyone — talk soon.
```
```
Good call — speak later.
```
```
Cheers, bye.
```
```
Thanks for the time — I'll follow up by EOD.
```
