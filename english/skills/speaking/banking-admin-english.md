# Banking & Admin English

Yeni ölkədə bank hesabı açmaq, sənədləşmə, poçt, dövlət qurumları — bunlar olmadan normal həyata başlamaq mümkün deyil.

---

## 1. Bank hesabı açmaq

### Banka gedəndə

```
"I'd like to open a current account, please."
"I've recently moved here and I need to set up a bank account."
"Do I need an appointment, or can I do this now?"
"What documents do I need to open an account?"
```

### Tələb olunan sənədlər (adətən):
- Passport
- Proof of address (utility bill, tenancy agreement, bank letter)
- Proof of employment (offer letter or payslip)

```
"Here is my passport."
"This is my tenancy agreement as proof of address."
"I have my employment offer letter — will that work as proof of income?"
"I don't have a utility bill yet — I just arrived. What else can I use?"
```

### Hesab növlərini soruşmaq:

```
"What types of accounts do you offer?"
"What is the difference between a current account and a savings account?"
"Is there a monthly fee?"
"Do you offer online banking and a mobile app?"
"How long does it take to receive the debit card?"
"Can I set up a standing order for rent payments?"
```

### Vacib terminlər

| Term | Məna |
|------|------|
| Current account | Cari hesab (əsas gündəlik hesab) |
| Savings account | Əmanət hesabı |
| Sort code | UK-də bank kodu (6 rəqəm) |
| Account number | Hesab nömrəsi |
| IBAN | Beynəlxalq hesab nömrəsi |
| Standing order | Sabit məbləğin avtomatik ödənişi |
| Direct debit | Dəyişkən məbləğin avtomatik çıxılması (utilities, phone) |
| Overdraft | Mənfi balans icazəsi |
| PIN | Şifrə (bank kartı) |
| Contactless payment | Toxunmadan ödəniş |

## 2. Məktub / Admin

### Post office (poçt):

```
"I'd like to send this letter/parcel to [country], please."
"What's the cheapest option?"
"How long does standard delivery take?"
"I'd like tracked delivery, please."
"Do you have a P.O. box here?"
"I'm expecting a letter — can I redirect my mail?"
```

### Ünvan dəyişikliyi (Address change):

```
"I need to update my address."
"I've recently moved to [new address] and need to update my details."
"Can I do this online or do I need to come in person?"
```

---

## 3. Telefon / İnternet müqaviləsi

```
"I'd like to get a SIM card / phone contract."
"Do you have any plans for new arrivals without a credit history?"
"What's included in this plan — data, calls, texts?"
"Is there a minimum contract period?"
"Can I keep my number and just switch provider?"
"I'd like to cancel my contract — what is the notice period?"
```

### İnternet (Broadband):

```
"I've just moved in and I need to set up broadband."
"What speeds are available at this address?"
"Is there a setup fee?"
"How long does installation take?"
"Is there a contract or can I go month-to-month?"
```

---

## 4. Dövlət qurumları / Registration

### GP (Family Doctor) qeydiyyatı:

```
"I'd like to register as a new patient."
"I've recently moved to the area."
"Do I need to bring any documents?"
"I'm not an NHS patient yet — what do I need to do?"
```

### NI Number (UK — National Insurance):

```
"I need to apply for a National Insurance number."
"My employer needs my NI number for payroll."
"I've started a new job — what documents do I need?"
```

### Council / Local authority:

```
"I need to register for council tax."
"I've moved into a property at [address] on [date]."
"Am I eligible for a single person discount?"
"How do I pay council tax — is direct debit possible?"
```

---

## 5. Telefonda danışmaq (Phone calls)

Ən çətin vəziyyət: telefonda resmi qurumla danışmaq.

### Başlanğıc:
```
"Hello, I'm calling about [topic]."
"My name is [name] and my reference/account number is [X]."
"I need some help with [issue]."
```

### Anlamadıqda:
```
"I'm sorry, could you repeat that please?"
"Could you speak a little more slowly?"
"Could you spell that for me?"
"Let me just confirm — you said [X], is that right?"
```

### Gözləmə:
```
"Of course, I'll hold."
"How long is the wait approximately?"
"Could I call back at a more convenient time?"
```

### Nəticəni həmişə yazılı istə:
```
"Could you send me a confirmation email?"
"Can I get a reference number for this call?"
"Could you confirm that in writing, please?"
```

---

## 6. Kiçik gündəlik admin vəziyyətlər

### Kirayə ödəmək:
```
"I've set up a standing order for the rent — it should arrive on the 1st."
"I transferred the deposit yesterday. Here's the transaction reference."
```

### Sığorta:
```
"I need to arrange contents insurance for my flat."
"What does the policy cover?"
"Is accidental damage included?"
"How do I make a claim?"
```

### Xidmət problemləri:
```
"My internet hasn't been working since yesterday."
"I was charged incorrectly on my last bill."
"I'd like to dispute this charge."
"Can I speak to a manager, please?"
```
