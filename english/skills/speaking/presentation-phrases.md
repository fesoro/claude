# Presentation Phrases (Prezentasiya ifadələri)

---

## 1. Açılış (Opening)

| Expression | Azerbaijani |
|------------|-------------|
| Good morning/afternoon everyone. | Hamıya sabahınız/günortanız xeyrə. |
| Thank you for being here today. | Bu gün burada olduğunuz üçün təşəkkür edirəm. |
| It's a pleasure to be here. | Burada olmaqdan məmnunam. |
| Today, I'd like to talk about... | Bu gün ...barədə danışmaq istəyirəm. |
| The topic of my presentation is... | Prezentasiyamın mövzusu... |
| I'm going to be talking about... | ...haqqında danışacam. |
| My presentation today covers... | Bu günkü prezentasiyam ...ı əhatə edir. |

---

## 2. Strukturu Bildirmək (Outlining the Structure)

| Expression | Azerbaijani |
|------------|-------------|
| I'll start by... then move on to... and finally... | Əvvəl ...dan başlayacam, sonra ...a keçəcəm, nəhayət... |
| My presentation is divided into three parts. | Prezentasiyam üç hissəyə bölünür. |
| First, I'll cover... | Əvvəlcə ...ı əhatə edəcəm. |
| This will take approximately [time]. | Bu, təxminən [vaxt] çəkəcək. |
| Please feel free to ask questions at the end. | Sonunda sual verməkdən çəkinməyin. |
| I'll take questions at the end. | Sualları sonunda alacam. |

---

## 3. Mövzuya Keçmək (Moving to the Next Point)

| Expression | Azerbaijani |
|------------|-------------|
| Let's move on to... | ...a keçək. |
| Now I'd like to turn to... | İndi ...a keçmək istəyirəm. |
| This brings me to my next point. | Bu məni növbəti mövzuma gətirir. |
| Moving on to... | ...a keçərək... |
| That brings us to the next section. | Bu bizi növbəti bölməyə gətirir. |
| Now let's look at... | İndi ...a baxaq. |

---

## 4. Qrafik / Slayd haqqında danışmaq

| Expression | Azerbaijani |
|------------|-------------|
| As you can see from this chart/graph... | Bu qrafik/diaqramdan gördüyünüz kimi... |
| This slide shows/illustrates... | Bu slayd göstərir/təsvir edir... |
| The figures indicate that... | Rəqəmlər göstərir ki... |
| I'd like to draw your attention to... | Diqqətinizi ...a cəlb etmək istəyirəm. |
| This graph represents... | Bu qrafik ...ı əks etdirir. |
| Looking at the data, we can see that... | Məlumatlara baxanda görürük ki... |

---

## 5. Vurğulamaq (Emphasising)

| Expression | Azerbaijani |
|------------|-------------|
| I'd like to emphasise/stress that... | Vurğulamaq istəyirəm ki,... |
| This is particularly important because... | Bu xüsusilə vacibdir çünki... |
| The key point here is... | Burada əsas nöqtə... |
| It is worth noting that... | Qeyd etmək lazımdır ki,... |
| One important thing to remember is... | Yadda saxlamaq lazım olan bir şey... |
| Above all, ... | Hər şeydən əvvəl, ... |

---

## 6. Nümunə vermək

| Expression | Azerbaijani |
|------------|-------------|
| To illustrate this, ... | Bunu izah etmək üçün, ... |
| For example, / For instance, ... | Məsələn, ... |
| A good example of this is... | Bunun yaxşı bir nümunəsi... |
| Let me give you an example. | Sizə bir nümunə verim. |
| This can be seen in the case of... | Bu, ... halında görünür. |

---

## 7. Geri Qayıtmaq / Xatırlatmaq

| Expression | Azerbaijani |
|------------|-------------|
| As I mentioned earlier, ... | Əvvəllər qeyd etdiyim kimi, ... |
| Going back to what I said earlier... | Əvvəl dediyimə qayıtsaq... |
| As I showed before, ... | Əvvəl göstərdiyim kimi, ... |
| As we discussed, ... | Müzakirə etdiyimiz kimi, ... |

---

## 8. Yekunlaşdırmaq (Summarising)

| Expression | Azerbaijani |
|------------|-------------|
| To summarise, ... | Ümumiləşdirsək, ... |
| To sum up, ... | Xülasə etmək üçün, ... |
| In conclusion, ... | Nəticə olaraq, ... |
| To recap the main points... | Əsas məqamları təkrarlasaq... |
| What I've tried to show today is... | Bu gün göstərməyə çalışdığım... |

---

## 9. Bağlayış (Closing)

| Expression | Azerbaijani |
|------------|-------------|
| That brings me to the end of my presentation. | Bu prezentasiyamın sonuna gətirir. |
| Thank you for your attention. | Diqqətiniz üçün təşəkkür edirəm. |
| I hope this has been useful/helpful. | Ümid edirəm bu faydalı oldu. |
| I'm now happy to take any questions. | İndi sualları almaqdan məmnunam. |
| Does anyone have any questions? | Kimin sualı var? |
| Please feel free to ask. | Soruşmaqdan çəkinməyin. |

---

## 10. Sualları Cavablamaq

| Expression | Azerbaijani |
|------------|-------------|
| That's a great question. | Bu əla bir sualdır. |
| I'm glad you asked that. | Bu sualı verdiyinizə sevindim. |
| I'll come to that in a moment. | Bu məsələyə bir azdan toxunacam. |
| I'm not entirely sure, but I think... | Tam əmin deyiləm, amma düşünürəm ki,... |
| That's beyond the scope of today's talk, but... | Bu bu günkü mövzunun kənarındadır, amma... |
| Could you repeat the question, please? | Zəhmət olmasa, sualı təkrar edərdiniz? |
