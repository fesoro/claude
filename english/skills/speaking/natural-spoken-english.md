# Natural Spoken English — Təbii Danışıq İngilisi

## Bu Fayl Haqqında

"I am going to work" və "I'm gonna go to work" — ikisi də eynidir, amma kitab ingiliscəsi və real danışıq ingilisi arasında böyük **ritm** fərqi var.

Bu fayl **reduced forms** (qısaldılmış formalar) və **contractions** (birləşmələr) öyrədir — yəni native speakers-ın həqiqətən necə danışdığını. Bunlar **dinləmək üçün** və **danışmaq üçün** bərabər vacibdir.

**Qayda:** Sən bunları yazmayacaqsan (yazıda formal saxla), amma **eşidəndə başa düşməli** və gündəlik söhbətdə **istifadə edə bilməlisən**.

---

## 1. Contractions (Birləşmələr) — Mütləq Bilinməli

### "Be" feli ilə
| Tam forma | Birləşmə |
|-----------|----------|
| I am | **I'm** |
| You are | **You're** |
| He/She is | **He's / She's** |
| We are | **We're** |
| They are | **They're** |
| That is | **That's** |
| It is | **It's** |
| There is | **There's** |

### "Have" feli ilə
| Tam forma | Birləşmə |
|-----------|----------|
| I have | **I've** |
| You have | **You've** |
| He/She has | **He's / She's** *(be ilə eyni!)* |
| We have | **We've** |
| They have | **They've** |

### Modal verblər
| Tam forma | Birləşmə |
|-----------|----------|
| I will | **I'll** |
| I would | **I'd** |
| I had | **I'd** *(would ilə eyni!)* |
| He/She will | **He'll / She'll** |

### Mənfi
| Tam forma | Birləşmə |
|-----------|----------|
| is not | **isn't** |
| are not | **aren't** |
| was not | **wasn't** |
| were not | **weren't** |
| do not | **don't** |
| does not | **doesn't** |
| did not | **didn't** |
| have not | **haven't** |
| has not | **hasn't** |
| had not | **hadn't** |
| will not | **won't** *(⚠️ willn't deyil!)* |
| would not | **wouldn't** |
| cannot | **can't** *(US: /kænt/; UK: /kɑːnt/)* |
| should not | **shouldn't** |
| must not | **mustn't** *(⚠️ "t" tələffüz olunmur)* |

### Xüsusi:
- **let's** = let us
- **gotta** = got to / have to *(informal)*
- **gonna** = going to *(informal)*
- **wanna** = want to *(informal)*

## 2. Reduced Forms (Qısaldılmış Formalar) — İnformal

### "going to" → "gonna" /ˈɡənə/
- Yalnız **future** mənasında:
  - "I'm **gonna** call her." = "I'm going to call her."
  - "What are you **gonna** do?" = "What are you going to do?"

### ⚠️ "gonna" hərəkət bildirəndə İŞLƏMIR:
- ❌ "I'm gonna the store." 
- ✅ "I'm going to the store." (hərfi hərəkət — gonna olmur)

### "want to" → "wanna" /ˈwɒnə/
- "I **wanna** go home." = "I want to go home."
- "Do you **wanna** eat?" = "Do you want to eat?"

### "got to" / "have to" → "gotta" /ˈɡɒtə/
- "I **gotta** go." = "I have to go."
- "You've **gotta** try this." = "You have got to try this."

### "kind of" → "kinda"
- "It's **kinda** hard." = "It's kind of hard."

### "sort of" → "sorta"
- "I'm **sorta** tired." = "I'm sort of tired."

### "let me" → "lemme"
- "**Lemme** see." = "Let me see."

### "give me" → "gimme"
- "**Gimme** a minute." = "Give me a minute."

### "out of" → "outta"
- "Get **outta** here!" = "Get out of here!"

### "a lot of" → "lotta"
- "**Lotta** people came." = "A lot of people came."

### "don't know" → "dunno"
- "I **dunno**." = "I don't know."

## 3. Auxiliary Verblərin Zəifləməsi

Danışıqda auxiliary verblər "yeyilir" — yəni tez-tez və zəif tələffüz olunur:

| Yazılış | Danışıqda səsi |
|---------|-----------------|
| Did you? | /dɪdʒə/ ("dija") |
| Did he? | /dɪdi/ ("didi") |
| Do you? | /djuː/ or /dʒə/ ("dja") |
| What are you? | /wʌtəjə/ ("wutaya") |
| What did you? | /wʌtɪdʒə/ ("wudija") |
| Are you? | /əjə/ ("aya") |
| Have you? | /həvjə/ or /əvjə/ ("avya") |
| Would you? | /wʊdʒə/ ("wudja") |

### Nümunələr:
- "**What did you** do yesterday?" → "Wudija do yesterday?"
- "**Did you** see it?" → "Dija see it?"
- "**Would you** like a coffee?" → "Wudja like a coffee?"
- "**What are you** doing?" → "Wadəyə doin?"

### Niyə vacibdir?
Azərbaycan danışanlar çox vaxt bu sualları "başa düşmür" — çünki kitablara görə "did you" iki ayrı söz kimi gözləyirlər. Reallıqda bir qarışıq səsdir.

## 4. "T" Səsinin Yox Olması (T-dropping)

### Dropped T (sözün ortasında T itirilir)
Bəzi amerikan aksentlərdə T yumuşayır və ya itir:
- "important" → "impor**'**ant" (T yox)
- "mountain" → "moun**'**ain"
- "winter" → "win**n**er"
- "international" → "interna**'**ional"

### Flap T (T → D kimi səslənir)
Amerikan ingiliscəsində saitlər arasında T → D:
- "water" → "**waa**-der"
- "letter" → "**led**-der"
- "better" → "**bed**-der"
- "butter" → "**bud**-der"

---

## 5. Linking (Birləşmə)

Cümlədəki sözlər **eşidir** ayrı deyil, birlikdə səslənir:

### Samit + Sait
Bir söz samitlə bitir, növbəti saitlə başlayır → birləşir:
- "turn **off**" → "tur**NOFF**"
- "pick **up**" → "pic**KUP**"
- "look **at**" → "loo**KAT**"
- "an **apple**" → "a**NAPPLE**"

### Sait + Sait (bəzən /j/ və ya /w/ əlavə olunur)
- "go on" → "go**W**on"
- "I am" → "ai**Y**em"
- "the end" → "the**Y**end"

---

## 6. "H" Səsi və Zəif Formalar

### Pronoun başında "h" tez-tez yox olur
- "Tell **him**" → "Tell **im**"
- "Give **her** the book" → "Give **er** the book"
- "Did **he** go?" → "Did**i** go?"

### "have / has" cümlənin ortasında zəifləyir
- "I should **have** done it" → "I should**a** done it" → "shoulda"
- "He could **have** known" → "He coulda known" → "coulda"
- "She must **have** left" → "musta left" → "musta"

### Yazıda:
- **shoulda** = should have
- **coulda** = could have
- **woulda** = would have
- **musta** = must have

⚠️ Bunlar **yalnız danışıqda** — yazıda "should have" yaz.

---

## 7. "And" və digər bağlayıcılar

### "and" → /ən/ və ya /n/
- "Fish **and** chips" → "Fish **n** chips"
- "Rock **and** roll" → "Rock **n** roll"
- "You **and** me" → "You **n** me"

### "or" → /ər/
- "Coffee **or** tea?" → "Coffee **r** tea?"

### "of" → /ə/ or /əv/
- "Cup **of** tea" → "Cup **a** tea" → "cuppa tea"
- "Kind **of** weird" → "Kind **a** weird" → "kinda weird"

---

## 8. Tez-tez Eşidəcəyin İfadələr

### "What's up?" = "Sup?"
- Informal salam: "**Sup**, man?"

### "You know what I mean?" = "Know what I mean?" / "Y'know what I mean?" / "Nome sayin?"
- Söhbət bağlayıcı

### "Are you going to?" = "You gonna?"
- "**You gonna** eat that?"

### "Do you want to?" = "Wanna?"
- "**Wanna** get coffee?"

### "I am going to have to" = "I'm gonna hafta"
- "I'm **gonna hafta** leave early."

### "Have you got a minute?" = "Gotta minute?"

---

## 9. Yox Olan "To Be" (Qeyri-rəsmi)

### Danışıqda bəzən "am/is/are" tamamilə unudulur:
- "You ready?" = Are you ready?
- "She coming?" = Is she coming?
- "They here?" = Are they here?

Bu həddindən artıq informaldır — müsahibədə işlətmə. Amma **eşidəndə başa düş**.

## 10. Dialoq Nümunələri — Təbii Formaya Çevir

### Formal → Natural

| Formal | Natural |
|--------|---------|
| I am going to go to the store. | I'm **gonna** go to the store. |
| Do you want to come with me? | **D'ya wanna** come with me? |
| I have to leave now. | I **gotta** leave now. |
| Did you see what happened? | **Didja** see what happened? |
| What are you doing? | **Whaddya doin'**? |
| I don't know where she is. | I **dunno** where she is. |
| Let me think about it. | **Lemme** think about it. |
| Give me a second. | **Gimme** a sec. |
| I should have called earlier. | I **shoulda** called earlier. |
| It is a lot of work. | It's a **lotta** work. |

### Real dialog:
```
A: "Hey, whatcha up to?"                [What are you up to?]
B: "Not much, just chillin'. You?"      [I'm just chilling / relaxing]
A: "I'm gonna grab some food. Wanna come?"   
B: "I'd love to, but I gotta finish this first."
A: "No worries. Lemme know when you're free."
B: "Will do. Thanks, man!"
```

## 11. Nə Vaxt Rəsmi, Nə Vaxt Təbii?

### FORMAL situation (gonna/wanna İŞLƏTMƏ):
- İş müsahibəsi (formal ton saxla)
- Yazılı email
- Prezentasiya
- Müştəri ilə rəsmi görüş

### OK situation (gonna/wanna təbii):
- Kollegia ilə nahar
- Small talk
- Virtual team görüşü
- Dostlarla söhbət

### Müqayisə:
| Formal | OK | Too Casual (qaçın) |
|--------|-----|---------------------|
| "I would like to apply." | "I'd like to apply." | "I wanna apply." ❌ müsahibədə |
| "I am going to discuss..." | "I'm going to discuss..." | "I'm gonna discuss..." ⚠️ |
| "Could you please repeat?" | "Could you repeat that?" | "Say that again?" ❌ müsahibədə |

## 13. Azərbaycan Danışanlarının Ən Çox Çətinlikləri

### "Did you" kimi qarışıq səslər
Azərbaycan dilində hər söz ayrı tələffüz olunur. İngilis dilində söz sərhədləri itir. Bu səbəbdən "Dija" / "Wudija" kimi səsləri ilk başda anlamaq çətindir.

**Həll:** 
- Subtitrli filmlərdə hər dəfə bu ifadələri işarələ
- 2-3 həftə sonra qulağın ayıracaq

### "I am" demək əvəzinə "I'm" deyə bilmirsən?
Tez-tez biz öyrədilmişik "I am tired" demək. Amma native speaker "I'm tired" deyir.

**Həll:**
- Yüksək səslə təkrar: "I'm, I'm, I'm"
- Cümlə qur: "I'm happy. I'm ready. I'm here."

## 14. Bonus: Gündəlik Kiçik Sözlər (Discourse Markers)

| İfadə | Funksiya | Nümunə |
|-------|----------|--------|
| **y'know** | söhbət doldurucusu | "It's, y'know, kinda hard." |
| **like** | fikir pauzası | "He was, like, really mad." |
| **so** | keçid | "**So**, what happened?" |
| **anyway** | mövzu bağlamaq | "**Anyway**, let's go." |
| **right?** | təsdiq axtarış | "It's cold today, right?" |
| **I mean** | dəqiqləşdirmə | "I mean, it's not perfect." |

## 15. Özünü Yoxla

Aşağıdakı rəsmi cümlələri təbii formaya çevir:

1. I am going to help you.
2. Do you want to come with us?
3. I have got to go now.
4. What did you do last weekend?
5. I do not know the answer.
6. Let me see what I can do.
7. I should have told you earlier.
8. Give me a minute, please.

### Cavablar:
1. **I'm gonna** help you.
2. **D'ya wanna** come with us? / "Wanna come with us?"
3. **I gotta** go now.
4. **Whadja do** last weekend? (What did you do...)
5. **I dunno** the answer.
6. **Lemme** see what I can do.
7. **I shoulda** told you earlier.
8. **Gimme** a minute, please.
