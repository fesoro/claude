# Workplace Culture Abroad

Yeni ölkədə iş başlayanda texniki bacarıq bəs deyil — mədəni fərqləri bilmək sən üçün böyük üstünlükdür. Bu fayl UK, US, EU iş mühitinin əsas fərqlərini izah edir.

---

## 1. UK İş Mühiti

### Ünsiyyət tərzi
UK-da insanlar **çox dolayı** danışır. Birbaşa "No" demək kobud sayılır.

| Nə deyirlər | Əsl mənası |
|-------------|-----------|
| "That's quite interesting." | Mənə pis gəlir |
| "With the greatest respect..." | Sən tamamilə yanılırsan |
| "I might be wrong, but..." | Yanılmırsın, sən yanılırsın |
| "Could we perhaps consider..." | Bu olmalıdır |
| "Not bad at all." | Əslində çox yaxşıdır |
| "That's a brave decision." | Bu qərar çox pis fikirdir |
| "I'll bear that in mind." | Bunu tamamilə unutacağam |
| "We should catch up soon." | Bunu etmək fikrim yoxdur |

### Praktik məsləhətlər
- "Please" və "Thank you" hər cümlədə — bu zorakılıq deyil, standartdır
- Şikayəti birbaşa deyil, **dolayı** deyirlər — "It's a bit noisy in here" = "Please be quiet"
- Pünktüallıq vacibdir — 5 dəqiqə gecikmə haqda əvvəlcədən xəbər ver
- Yemək vaxtı stolda iş danışmaq ümumi deyil
- "How are you?" — cavab həmişə qısa: "Fine, thanks. You?"

## 2. US İş Mühiti

### Ünsiyyət tərzi
Amerikalılar **daha birbaşa** danışır, amma həmişə **pozitiv** çərçivədə:

```
UK: "This might have some issues."
US: "Great idea! Let me add a few thoughts on how we can make it even better."
```

### Praktik məsləhətlər
- "How are you?" demə soruşmur — "Great!" kimi qısa cavab ver
- Birinci adla müraciət normaldir — hətta CEO-ya da
- Meeting-lər daha az formal — "Let's jump on a call" çox istifadə olunur
- Feedback birbaşa verilir, amma həmişə "feedback sandwich" strukturunda: yaxşı → düzəltmə → yaxşı
- Promosyon / artım özün istəməlisən — kimsə buna gəlmir

---

## 3. Almanya / Mərkəzi Avropa

### Fərqlər
- Çox **struktur** sevən mədəniyyət — agenda olmadan meeting? Nadir
- Birbaşa ünsiyyət — "No" demək normal
- Pünktüallıq son dərəcə vacibdir (UK-dan da ciddi)
- Qeyri-rəsmi tanışlıq əvvəlcə gəlmir — əvvəlcə formal, sonra "Du" (qeyri-rəsmi)
- Sənəd sevgisi var — hər şeyin yazılı olmasını istəyirlər

```
"Let's document this decision."
"Can you put that in writing?"
"Let's create a process for this."
```

---

## 4. Hər yerdə işləyən universal mədəni qaydalar

### İlk günlər / onboarding:
```
"I'm still getting to know how things work here — could you help me understand..."
"What's the best way to reach out about X in this team?"
"Who should I go to for questions about [topic]?"
"What does a typical day look like for the team?"
```

### Rəy / feedback istəmək:
```
"I'd love your feedback on this approach."
"Am I on the right track?"
"Is this how you'd normally handle it here?"
"Is there anything I should do differently?"
```

### Fərqli fikri ifadə etmək:
```
"That's interesting — I was thinking slightly differently. Would it be okay to share my approach?"
"I see where you're coming from. Could we also consider...?"
"I'm not sure that's the right approach — my concern is X. What do you think?"
```

## 5. Sosial həyat — iş yoldaşları ilə

### After-work / Friday drinks (UK-da çox yaygındır):
```
"Are you coming to the pub tonight?"
"I'd love to join — where are you all heading?"
"I can't make it tonight, but count me in next time."
```

Pub-da içki sırası (rounds):
```
"It's my round — what are you having?"
"I'll get these."
"Same again?"
"Just a soft drink for me, thanks."
```

### Lunch:
```
"Are you getting lunch? Want to grab something together?"
"Where does everyone usually eat around here?"
"I'm going to [place] — anyone want anything?"
```

### Small talk mövzuları (uyğun):
- Hava (UK-da ən populyar) — "Typical British weather, isn't it?"
- Həftə sonu planları — "Any plans for the weekend?"
- Sport — "Did you catch the match last night?"
- TV / film — "Have you seen [series]?"
- İsh həyatı — "How's the project going?"

### Yaxşı olmayan mövzular (yeni iş yoldaşları ilə):
- Siyasət, din, maaş — personal həddini bil
- Şirkət haqqında şikayət (tam oturmamışdan)

---

## 6. Görüntülü zəng / remote team mədəniyyəti

```
"Let me quickly mute myself while you finish."
"Sorry, I think I talked over you — please go ahead."
"Could you just repeat that? I had a connection issue."
"I'll share my screen now."
"I'll drop that in the chat."
```

### Cavab vaxtları:
- Slack/Teams mesajı — adətən eyni gündə cavab gözlənilir
- Email — 1-2 iş günü normaldir
- "@here" / "@channel" — urgent mesajdır, tez cavab ver

---

## 7. Problemli vəziyyətlər

**Çox sürətli danışıldıqda:**
```
"Sorry, could you slow down a bit? I want to make sure I get everything."
"I caught most of that — could you just repeat the last part?"
```

**Aksent anlamadıqda:**
```
"I'm sorry, I didn't quite catch that — could you say it again?"
"Could you type that in the chat? Just to make sure I have it right."
```

**Çox iş verildiqdə:**
```
"I want to make sure I prioritise correctly — which of these is most urgent?"
"I currently have X, Y, Z on my plate. Should I pause one of those for this?"
```
