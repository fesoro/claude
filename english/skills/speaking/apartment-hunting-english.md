# Apartment Hunting English

Yeni ölkədə ev tapmaq — ilk həftələrin ən gərgin vəzifəsi. Bu fayl sənə icarəçi kimi lazım olan bütün danışıq frazalarını verir.

---

## 1. Ev baxışı zamanı (Viewing an apartment)

### Əlaqə saxlamaq / Baxış təyin etmək

```
"Hi, I'm calling about the apartment on [street name]. Is it still available?"
"I saw your listing online. Can I arrange a viewing?"
"Would it be possible to view the property this weekend?"
"What time works best for you?"
```

### Baxış zamanı suallar

**Ümumi:**
```
"How long has the property been available?"
"Why is the current tenant leaving?"
"How many people live in the building?"
"Is parking included?"
"Which floor is it on? Is there a lift/elevator?"
```

**Xərclər:**
```
"What is included in the rent? (bills, internet, water)"
"Are utilities included or separate?"
"How much is the deposit?"
"Is there an agency fee?"
"When is rent due each month?"
```

**Vəziyyət:**
```
"Are there any known issues with the property?"
"When was the last time the boiler/heating was serviced?"
"Is the double glazing throughout?"  
"Does the dishwasher/washing machine come with it?"
"Is there storage space?"
```

**Qonşular / bina:**
```
"What are the neighbours like?"
"Is it noisy at night?"
"Is there a building manager or caretaker?"
"Are pets allowed?"
"Is smoking allowed inside?"
```

## 2. Müzakirə / Danışıq (Negotiating)

```
"The listing says £X — is there any flexibility on the price?"
"We're looking at several properties. Would you consider £X?"
"If we sign a longer lease, would you lower the monthly rent?"
"Could you include broadband in the rent?"
"Would it be possible to move in on [date]?"
```

### Rədd etmək (politely):
```
"Thank you for showing me around. I'll think it over and get back to you."
"It's not quite what we were looking for, but thank you for your time."
"We've decided to go with another property. Thanks anyway."
```

## 3. Müqavilə imzalamaq (Signing the tenancy agreement)

### Vacib terminlər

| İngilis | Məna |
|---------|------|
| Tenancy agreement / lease | İcarə müqaviləsi |
| Security deposit | Zəmanət depoziti (adətən 1-2 aylıq icarə) |
| Holding deposit | Evin tutulması üçün kiçik ödəniş |
| Inventory check | Evdəki əşyaların siyahısı |
| Notice period | Çıxmazdan əvvəl bildiriş müddəti (adətən 1 ay) |
| Break clause | Müqaviləni vaxtından əvvəl bitirmə hüququ |
| Joint tenancy | Birlikdə icarə (hamı məsuliyyət daşıyır) |
| Guarantor | Zamin (icarəçi ödəyə bilməsə ödəyən şəxs) |
| Standing order | Avtomatik aylıq ödəniş |

### Müqavilə haqqında suallar:

```
"Can I read the agreement before signing?"
"What does this clause mean exactly?"
"Is the deposit held in a government-protected scheme?"
"What is the notice period?"
"Is there a break clause after 6 months?"
"Who is responsible for repairs — me or the landlord?"
"Can I make any changes to the apartment (paint, shelves)?"
```

## 4. Sahiblə problemlər (Dealing with issues)

### Təmir tələb etmək:
```
"I'd like to report a maintenance issue."
"The heating isn't working properly."
"There's a leak under the sink."
"The boiler stopped working — can you send someone?"
"I mentioned this before but it hasn't been fixed yet."
```

### Yazılı email formatı:
```
Subject: Maintenance Issue — [your address]

Hi [landlord name],

I'm writing to report an issue at [address]. 
[The boiler / heating / tap / washing machine] is not working correctly.

[Specific description of the problem.]

Could you please arrange for someone to come and fix this at your earliest convenience?

Please let me know what time would suit for an inspection.

Best regards,
[Your name]
```

## 5. Çıxış prosesi (Moving out)

```
"I'd like to give notice that I'll be leaving on [date]."
"I want to make sure I get my full deposit back."
"Can we arrange a check-out inspection together?"
"What is the process for returning the keys?"
"I'd like a written confirmation that the property is in good condition."
```

### Depozit geri almaq:
```
"How long does it take to receive the deposit back?"
"Can you provide an itemised list of any deductions?"
"I disagree with this deduction — the damage was already there when I moved in."
```

## 6. Çatışmayan söz olduqda

```
"Sorry, I didn't catch that — could you repeat it?"
"Could you speak a little more slowly, please?"
"I'm not familiar with that term — what does it mean?"
"Could you write that down for me?"
```
