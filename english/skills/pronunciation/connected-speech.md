# Connected Speech — Birləşmiş Nitq

## Səviyyə
B1-C1

## Niyə Vacibdir?

Ana dilli ingilis danışanlar hər sözü ayrı-ayrı deymirlər. Sözlər birləşir, bəzi səslər yox olur, bəziləri dəyişir. Bu qaydalarsız native speaker-ları anlamaq çətin olur.

---

## 4 Əsas Hadisə

| Hadisə | Azərbaycanca | Nümunə |
|--------|-------------|--------|
| **Linking** | Birləşmə | "turn off" → "tur**n**off" |
| **Elision** | Səs itirilməsi | "next day" → "nex day" |
| **Assimilation** | Səs dəyişməsi | "ten people" → "tem people" |
| **Weak forms** | Zəif formalar | "and" → /ən/ |

---

## 1. LINKING — Sözlər Birləşir

### Samit + Sait (ən çox)
Bir söz samitlə bitir, növbəti saitlə başlayırsa birləşirlər:

| Yazılış | Tələffüz |
|---------|---------|
| turn **off** | tur**n_off** (n-off birlikdə) |
| an **apple** | a**n_apple** |
| pick **it** up | pic**k_it_**up |
| not **at** all | no**t_at_**all |
| come **in** | com**e_in** |

### Sait + Sait — /w/ ya /j/ əlavə olunur
- go **on** → go**/w/**on
- she **is** → she**/j/**is
- do **it** → do**/w/**it
- I **always** → I**/j/**always

---

## 2. ELISION — Səs İtirilir

### /t/ və /d/ aradan çıxır (samitlər arasında)
| Yazılış | Tələffüz |
|---------|---------|
| **next** day | **nex** day |
| **last** year | **las** year |
| **just** go | **jus** go |
| han**ds** | han**s** |
| frien**ds** | frien**s** |

### /h/ zəifləyir (söz ortasında)
| Yazılış | Tələffüz |
|---------|---------|
| tell **him** | tell (**h**)im |
| give **her** | give (**h**)er |
| ask **him** | ask (**h**)im |

---

## 3. ASSIMILATION — Səs Dəyişir

Bir sözdəki səs qonşu sözün təsirinə görə dəyişir:

| Yazılış | Tələffüz | Nə baş verir |
|---------|---------|-------------|
| ten **p**eople | te**m** people | n → m (p qarşısında) |
| in **b**ed | i**m** bed | n → m (b qarşısında) |
| that **g**uy | tha**g** guy | t → g (g qarşısında) |
| good **b**ye | goo**b** bye | d → b (b qarşısında) |
| don't **k**now | don**g** know | t → g (k qarşısında) |

---

## 4. WEAK FORMS — Zəif Formalar

Funksional sözlər (artikl, preposition, yardımçı fel) tez-tez tənbəl tələffüz olunur:

| Söz | Güclü forma | Zəif forma |
|-----|-------------|-----------|
| **and** | /ænd/ | /ən/ |
| **the** | /ðiː/ | /ðə/ |
| **a** | /eɪ/ | /ə/ |
| **of** | /ɒv/ | /əv/ |
| **to** | /tuː/ | /tə/ |
| **for** | /fɔː/ | /fə/ |
| **at** | /æt/ | /ət/ |
| **from** | /frɒm/ | /frəm/ |
| **can** | /kæn/ | /kən/ |
| **have** | /hæv/ | /həv/ |
| **was** | /wɒz/ | /wəz/ |
| **him** | /hɪm/ | /ɪm/ |
| **her** | /hɜː/ | /ə/ |
| **them** | /ðem/ | /ðəm/ |

**Nümunə:**
- "I want to go for a walk" → /aɪ wɒnt **tə** gəʊ **fər ə** wɔːk/

## Çox Sürətli Danışıqda Görülənlər

### Gonna / Wanna / Gotta / Kinda (informal)
| Yazılış | Tələffüz | Mənası |
|---------|---------|--------|
| going to | **gonna** | gətiyrəm |
| want to | **wanna** | istəyirəm |
| got to | **gotta** | lazımdır |
| kind of | **kinda** | bir növ |
| sort of | **sorta** | bir az |
| out of | **outta** | -dan çıxaraq |

> **Diqqət:** Bunlar yalnız informal, gündəlik danışıqda. Yazıda işlətmə!

## Dinləmə Strategiyaları

Sürətli native speech-i anlamaq üçün:

1. **Tam cümləni eşitmək lazım deyil** — əsas sözləri (nouns, verbs, adjectives) tut
2. **Kontekstdən istifadə et** — bildiklərinlə boşluqları doldur
3. **Shadows (kölgə)** — duyduğunu dərhal təkrar et
4. **Transcript ilə dinlə** — BBC, TED, film subtitrı ilə
5. **Tez-tez eşit** — eyni dialoqu 5-10 dəfə dinlə

---

## Məşq: Zəif formaları tapın

Bu cümləni tez oxuyun — hansı sözlər zəifləyir?

"I want to have a cup of tea and some biscuits."

**Zəif olacaqlar:** to /tə/, a /ə/, of /əv/, and /ən/

Tam tələffüz: /aɪ wɒnt **tə** hæv **ə** kʌp **əv** tiː **ən** səm bɪskɪts/
