# Pronunciation of Tech Terms

Müsahibdə texniki terminləri **səhv tələffüz etsəniz** iki problem yaranır: 1) müsahib sizi başa düşməyə bilər, 2) bu, təcrübəsizlik təəssüratı verə bilər. Bu fayl Azərbaycan dilindən ingiliscəyə keçərkən ən çox səhv tələffüz olunan terminləri göstərir.

---

## Oxunuş işarələri

Bu faylda:
- **BÖYÜK hərflərlə** yazılan hissə — **vurğu** (stress) deməkdir
- `/slash/` arasındakı — IPA (beynəlxalq fonetik əlifba)
- Azərbaycanca oxunuş — tələffüzə yaxın, təxmini səs

## 7. Vurğu Nümunələri (Stress Patterns)

### Düzgün: fellərdə son hecada vurğu (bəzən)
- de**VEL**-op (develop)
- im-**PLE**-ment (implement)
- de-**PLOY** (deploy)
- con-**FIG**-ure (configure)
- re-**FAC**-tor (refactor)

### İsimlərdə birinci hecada vurğu (bəzən)
- **DA**-ta-base (database)
- **IN**-put / **OUT**-put
- **AL**-go-rithm
- **CO**-de-base

### Compound words — birinci sözdə vurğu
- **FRONT**-end (NOT front-**END**)
- **BACK**-end
- **HARD**-ware
- **SOFT**-ware
- **OPEN**-source

## 8. Azərbaycandillilər üçün xüsusi məsləhətlər

### 1. "W" səsi
- Azərbaycan dilində yoxdur — "v" ilə qarışdırırıq
- "workflow" — **VORK**-flow deyil, **WORK**-flow (dodaqlar buruq)
- Məşq: "work", "well", "worked", "wait"

### 2. "Th" səsi
- İki variantı var:
  - /θ/ — "three", "path", "thread" (dil diş arasında, hava)
  - /ð/ — "this", "that", "the" (dil diş arasında, səsli)
- Məşq: "thread", "throughput", "the database"

### 3. "R" səsi
- Azərbaycandilli R titrəyir — ingilis R-i titrəmir
- "React", "refactor", "rollback" — dil yuxarı qalxır, amma titrəmir

### 4. Vurğulu və vurğusuz hecalar
- İngiliscədə vurğusuz hecalar çox sakit — "schwa" /ə/ səsi
- "develop" — **VEL** yüksək, əvvəl və son sakit: di-**VEL**-əp
- Bütün hecalarda eyni enerji vermək sizi "yad" eşitdirir

### 5. Son səsi "düzgün bitirmək"
- Azərbaycandillilər tez-tez son "d", "t", "s" səslərini boğurlar
- "deployed" — son "d" eşidilməlidir
- "tests" — son "s" eşidilməlidir

## Məşq 2 — Səhvi tap

Hər cümlədə bir söz səhv tələffüz edilib. Düzəltməyi bacarırsanız qeyd edin:

1. "We store data in **mon-qo** DB." → ?
2. "Let me check the **kash**." → ? (cache)
3. "We use **sixuel** for queries." → ?
4. "Deploy to **kuber-nets**." → ?
5. "Check the **pseudo** code." → ?

### Cavablar
1. **MON**-goh (mongo)
2. Düzdür! "kash" doğrudur
3. **SEE**-kwəl və ya es-kyu-EL (NOT "sixuel")
4. koo-bər-**NEH**-tiz
5. **SOO**-doh (p oxunmur)

## Məşq 3 — Shadowing

Aşağıdakı cümlələri yüksək səslə 5 dəfə təkrar edin (əvvəlcə yavaş, sonra normal sürətlə):

1. "We deployed the microservices to Kubernetes yesterday."
2. "The PostgreSQL query is slow because of a missing index."
3. "Authentication is handled by OAuth, not JWT."
4. "We use Redis for caching and Cassandra for storage."
5. "The CI/CD pipeline runs automated tests on every pull request."
6. "Asynchronous processing keeps the API responsive under load."
7. "The SDK documentation is available on GitHub."

**Məsləhət:** Özünüzü telefonla yazın və dinləyin. Hər dəfə daha təbii səslənməlidir.
