# Söz Vurğusu (Word Stress)

## Səviyyə
A2-B2

## Niyə Vacibdir?

Yanlış vurğu ilə deyilən söz ana dilli danışan tərəfindən anlaşılmaya bilər.
> "reCORD" (fel) vs "REcord" (isim) — eyni yazılış, fərqli mənalar!

---

## Əsas Qayda: Hər Sözdə Bir Əsas Vurğu

- **pho**to (FOtо — birinci heca)
- **beau**tiful (BEAUtiful — birinci heca)
- com**pu**ter (comPUter — ikinci heca)
- un**der**stand (underSTAND — üçüncü heca)

## İsim vs Fel — Eyni Yazılış, Fərqli Vurğu

Bu cütlər çox vacibdir:

| İsim (birinci heca) | Fel (ikinci heca) |
|---------------------|------------------|
| **RE**cord — qeyd | re**CORD** — qeyd etmək |
| **PRE**sent — hədiyyə; indiki | pre**SENT** — təqdim etmək |
| **OB**ject — əşya | ob**JECT** — etiraz etmək |
| **PER**mit — icazə | per**MIT** — icazə vermək |
| **PRO**test — etiraz | pro**TEST** — etiraz etmək |
| **IN**crease — artım | in**CREASE** — artmaq |
| **DE**crease — azalma | de**CREASE** — azalmaq |
| **EX**port — ixracat | ex**PORT** — ixrac etmək |
| **IM**port — idxalat | im**PORT** — idxal etmək |
| **RE**fuse — zibil | re**FUSE** — rədd etmək |
| **CON**tract — müqavilə | con**TRACT** — azalmaq |
| **PER**fect — mükəmməl | per**FECT** — mükəmməlləşdirmək |

## Mürəkkəb Sözlər (Compound Words)

Adətən **birinci söz** vurğulanır:

| Söz | Vurğu |
|-----|-------|
| **BOOK**shop | BOOKshop |
| **AIR**port | AIRport |
| **HAND**bag | HANDbag |
| **NEWS**paper | NEWSpaper |
| **SUN**glasses | SUNglasses |
| **BIRTH**day | BIRTHday |
| **DOOR**bell | DOORbell |

**İstisna:** Bu ifadələrdə ikinci söz vurğulanır:
- ice **CREAM** (dondurma)
- good **NIGHT** (gecəniz xeyr)

## Cümlə Vurğusu (Sentence Stress)

Cümlədə ən vacib məlumat daşıyan söz vurğulanır:

- I **BOUGHT** a new car. (nə etdiyim vacibdir)
- I bought a **NEW** car. (yeni olması vacibdir)
- **I** bought a new car. (başqası deyil, MƏN aldım)
- I bought a new **CAR**. (başqa əşya deyil, MAŞIN)

## Azərbaycanlıların Tez-tez Etdiyi Səhvlər

| Yanlış | Düzgün | İzah |
|--------|--------|-------|
| com-PU-ter → COM-pu-ter | com-**PU**-ter | ikinci heca |
| pho-to-GRAPH | **PHO**-to-graph | birinci heca |
| in-FOR-ma-tion | in-for-**MA**-tion | -tion öncəki heca |
| im-PORT (isim) | **IM**-port (isim) | isim → birinci |
| re-CORD (isim) | **RE**-cord (isim) | isim → birinci |

## Məşq Tapşırıqları

### 1. Vurğulu hecayı müəyyən edin
1. computer
2. information
3. beautiful
4. education
5. photograph
6. economic

**Cavablar:**
1. com**PU**ter
2. infor**MA**tion
3. **BEAU**tiful
4. edu**CA**tion
5. **PHO**tograph
6. e**CO**nomic

### 2. İsim yoxsa Fel? (vurğuya görə)
1. Please **RE**cord the meeting. (isim/fel?)
2. Can I see your **RE**cord? (isim/fel?)
3. They want to **ex**PORT more goods. (isim/fel?)

**Cavablar:** 1-fel (reCORD), 2-isim (REcord), 3-fel (exPORT)
