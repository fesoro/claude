# Linking Words & Transition Phrases

---

## 1. Addition (Əlavə etmək)

| Expression | Example |
|------------|---------|
| Furthermore, | Furthermore, the study revealed unexpected results. |
| Moreover, | Moreover, the cost of living has increased significantly. |
| In addition, | In addition, we need to consider environmental factors. |
| Additionally, | Additionally, there are several other options available. |
| Besides, | Besides, it would take too much time. |
| Also, | Also, the weather was unfavourable. |
| As well as | As well as being expensive, it is impractical. |
| Not only... but also | Not only is it dangerous, but it is also illegal. |

---

## 2. Contrast (Ziddiyyət)

| Expression | Example |
|------------|---------|
| However, | However, there are some disadvantages. |
| Nevertheless, | Nevertheless, she continued her work. |
| Nonetheless, | Nonetheless, the plan was approved. |
| Although / Though | Although it rained, we went out. |
| Even though | Even though he studied, he failed. |
| Despite / In spite of | Despite the difficulties, they succeeded. |
| On the other hand, | On the other hand, some argue that... |
| Whereas | He prefers coffee, whereas she likes tea. |
| While | While some agree, others strongly oppose. |
| Yet | It is expensive, yet very effective. |
| In contrast, | In contrast, rural areas saw little change. |

---

## 3. Cause & Reason (Səbəb)

| Expression | Example |
|------------|---------|
| Because | She left because she was tired. |
| Because of | Because of the rain, the match was cancelled. |
| Since | Since he was ill, he stayed home. |
| As | As it was late, we decided to leave. |
| Due to | Due to budget constraints, the project was delayed. |
| Owing to | Owing to bad weather, the flight was cancelled. |
| As a result of | As a result of the changes, productivity improved. |

---

## 4. Effect & Result (Nəticə)

| Expression | Example |
|------------|---------|
| Therefore, | Therefore, we must act now. |
| Thus, | Thus, the problem remains unsolved. |
| Hence, | Hence, a new approach is needed. |
| As a result, | As a result, sales increased by 20%. |
| Consequently, | Consequently, many jobs were lost. |
| For this reason, | For this reason, we changed our strategy. |
| This means that | This means that we need more funding. |
| This leads to | Stress leads to poor concentration. |

---

## 5. Purpose (Məqsəd)

| Expression | Example |
|------------|---------|
| In order to | In order to improve efficiency, we hired more staff. |
| So as to | She spoke quietly so as not to wake the baby. |
| So that | He saved money so that he could travel. |
| To | She studied hard to pass the exam. |
| With the aim of | With the aim of reducing costs, we cut staff. |

---

## 6. Example (Nümunə)

| Expression | Example |
|------------|---------|
| For example, | For example, many cities now have bike-sharing schemes. |
| For instance, | For instance, London has over 12,000 hire bikes. |
| Such as | Sports such as swimming are excellent for fitness. |
| In particular, | In particular, air pollution is a major concern. |
| Namely, | There are two options, namely A and B. |
| Including | Many factors, including diet and exercise, affect health. |

---

## 7. Sequence & Order (Sıra)

| Expression | Example |
|------------|---------|
| Firstly, / First of all, | Firstly, we need to identify the problem. |
| Secondly, / Then, | Secondly, we should consider the solutions. |
| Thirdly, / Next, | Thirdly, we must evaluate the costs. |
| Finally, / Lastly, | Finally, a decision should be made. |
| Subsequently, | Subsequently, the policy was revised. |
| Previously, | Previously, this method was not available. |
| Meanwhile, | Meanwhile, the other team worked on a backup plan. |

---

## 8. Concession (Güzəşt etmək)

| Expression | Example |
|------------|---------|
| Admittedly, | Admittedly, there are some flaws in this approach. |
| It is true that... | It is true that costs are high, but the benefits are clear. |
| Of course, | Of course, there are exceptions. |
| Granted, | Granted, the idea is creative, but it lacks practicality. |
| While it is true that... | While it is true that change is difficult, it is necessary. |

## 9. Emphasis (Vurğulamaq)

| Expression | Example |
|------------|---------|
| In fact, | In fact, the situation is much more serious. |
| Indeed, | Indeed, this is a significant problem. |
| Clearly, | Clearly, more action is needed. |
| Obviously, | Obviously, this requires careful planning. |
| Above all, | Above all, safety must be the priority. |
| Most importantly, | Most importantly, we must listen to the public. |

## 10. Summary & Conclusion (Yekunlaşdırma)

| Expression | Example |
|------------|---------|
| In conclusion, | In conclusion, the evidence supports this theory. |
| To sum up, | To sum up, there are both advantages and disadvantages. |
| In summary, | In summary, three main points have been discussed. |
| Overall, | Overall, the results are encouraging. |
| To conclude, | To conclude, further research is needed. |
| All in all, | All in all, it was a successful project. |

---

## Punctuation Rules

| Usage | Rule | Example |
|-------|------|---------|
| Cümlə əvvəlində | Vergül qoy | However**,** this is difficult. |
| İki müstəqil cümlə arasında | Nöqtə-vergül + vergül | She studied**;** however**,** she failed. |
| İki hissəni birləşdirən | Vergülsüz | She failed **because** she didn't study. |
