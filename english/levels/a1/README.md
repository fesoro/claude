# A1 — Başlanğıc Səviyyəsi

## Bu Səviyyədə Nə Öyrənəcəksiniz?

A1 — sıfırdan başlayanlar üçündür. Bu səviyyəni bitirəndə:
- Özünüzü tanıda biləcəksiniz
- Sadə suallar verə biləcəksiniz
- Gündəlik ifadələri başa düşəcəksiniz
- Asan formaları doldura biləcəksiniz

**Orta müddət:** 3-4 ay (gündə 1 saat)  
**Lüğət hədəfi:** 500 söz
