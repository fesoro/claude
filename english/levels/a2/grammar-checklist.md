# A2 Qrammatika Yoxlama Siyahısı

A2 səviyyəsini tamamlamaq üçün aşağıdakı qrammatika strukturlarını bilməlisiniz.

---

## ✅ Past Simple
- [ ] Müntəzəm fellər: work → worked
- [ ] Qeyri-müntəzəm fellər: go → went, have → had, see → saw
- [ ] Mənfi: didn't + V1
- [ ] Sual: Did + S + V1?
- [ ] Zaman ifadələri: yesterday, last week, ago, in 2020

## ✅ Present Continuous
- [ ] Quruluş: am/is/are + V-ing
- [ ] İşlənmə: indi baş verir, müvəqqəti vəziyyət
- [ ] Present Simple ilə fərqi

## ✅ Future: Will vs Going To
- [ ] Will: spontan qərar, proqnoz, vəd
- [ ] Going to: əvvəlcədən düşünülmüş plan
- [ ] Dəlilə əsaslanan proqnoz (going to)

## ✅ Comparative & Superlative
- [ ] Qısa sifətlər: -er / -est (hotter, the hottest)
- [ ] Uzun sifətlər: more/most (more beautiful)
- [ ] Qeyri-müntəzəm: good/better/best, bad/worse/worst
- [ ] As...as (bərabərlik)

## ✅ Should / Shouldn't
- [ ] Məsləhət vermək
- [ ] Məsləhətin rədd edilməsi

## ✅ Have to / Don't have to
- [ ] Məcburilik: have to
- [ ] Məcburiyyətsizlik: don't have to

## ✅ Like / Love / Enjoy + -ing
- [ ] -ing forması (gerund) istifadəsi
- [ ] Sıralamanın dərəcəsi: love → like → don't mind → dislike → hate

## ✅ Prepositions of Time
- [ ] at (vaxt, gecə, bayramlar)
- [ ] on (günlər, tarixlər)
- [ ] in (ay, il, mövsüm, günün hissəsi)

## ✅ Countable vs Uncountable
- [ ] Sayıla bilən: a chair, chairs
- [ ] Sayıla bilməyən: water, advice, information
- [ ] Some / any / much / many / a lot of

## ✅ Artikl: A/An/The
- [ ] A/An — qeyri-müəyyən, ilk dəfə
- [ ] The — müəyyən, bilinən
- [ ] Zero article — ümumi isimlərlə
