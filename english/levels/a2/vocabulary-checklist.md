## Mövzu Sahələri — Bilməli Olduqlarınız

### Alış-veriş
- [ ] Mağaza növləri (supermarket, pharmacy, bakery...)
- [ ] Geyim və ölçülər
- [ ] Rəng adları
- [ ] Qiymət ifadələri (How much is...? It costs...)
- [ ] Ödəniş üsulları (cash, card, change)

### Səyahət
- [ ] Nəqliyyat vasitələri
- [ ] Hava limanı/stansiya lüğəti
- [ ] Mehmanxana ifadələri
- [ ] İstiqamət vermək (turn left, go straight...)

### Hava və Mövsümlər
- [ ] Hava sifətləri (sunny, rainy, cold, hot...)
- [ ] Dörd mövsüm
- [ ] Temperatur ifadəsi

### Sağlamlıq
- [ ] Bədən hissələri (30+ söz)
- [ ] Xəstəlik adları (headache, cold, flu...)
- [ ] Tibbi ifadələr

### Hobbilər
- [ ] 15+ hobби adı
- [ ] Like/love/enjoy/hate + -ing ifadələri

### Peşələr
- [ ] 20+ peşə adı
- [ ] İş şəraiti (full-time, part-time, shift work)

### Gələcək
- [ ] Zaman ifadələri (tomorrow, next week, soon...)
- [ ] Plan bildirən ifadələr
