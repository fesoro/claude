# A2 — Elementar Səviyyə

## Bu Səviyyədə Nə Öyrənəcəksiniz?

A2 səviyyəsini bitirəndə:
- Gündəlik mövzularda sadə ünsiyyət qura bilirsiniz
- Alış-veriş, restorana getmək kimi situasiyaları idarə edirsiniz
- Keçmişdən danışa bilirsiniz
- Planlarınızı söyləyə bilirsiniz

**Orta müddət:** 3-4 ay (gündə 1 saat)  
**Lüğət hədəfi:** 1000 söz
