# B1 Unit 03: Şərt Cümlələri (Conditionals)

## Bu Vahiddə Nə Öyrənirsiniz?
- Zero Conditional — həmişə doğru olan həqiqətlər
- First Conditional — real imkan
- Second Conditional — xəyal, qeyri-real vəziyyət
- Unless istifadəsi

---

## Zero Conditional — Həqiqətlər

**Quruluş:** If + present simple, present simple

**Mənası:** Həmişə doğru olan — elmi həqiqət, ümumi qanun

| Nümunə | Azərbaycanca |
|--------|-------------|
| If you heat water, it boils. | Su qızdırsan, qaynayır. |
| If it rains, the ground gets wet. | Yağış yağarsa, yer islanır. |
| If you don't sleep, you feel tired. | Yatmasan, yorğun hiss edirsən. |

---

## First Conditional — Real İmkan

**Quruluş:** If + present simple, will + base verb

**Mənası:** Gələcəkdə mümkün olan şərt

| Nümunə | Azərbaycanca |
|--------|-------------|
| If it rains, I **will** stay home. | Yağış yağarsa, evdə qalacağam. |
| If you study hard, you **will** pass. | Çox oxusan, keçəcəksən. |
| If she calls, I **will** answer. | Zəng etsə, cavab verəcəyəm. |

> **Diqqət:** "If" olan hissədə **will** işlənmir!
> - ✗ If it **will rain**, I'll stay home.
> - ✓ If it **rains**, I'll stay home.

---

## Second Conditional — Qeyri-Real / Xəyal

**Quruluş:** If + past simple, would + base verb

**Mənası:** İndiki vaxtda qeyri-real xəyal, ya da mümkün olmayan vəziyyət

| Nümunə | Azərbaycanca |
|--------|-------------|
| If I had more money, I **would** travel. | Pulum olsaydı, səyahət edərdim. |
| If she **were** a doctor, she would help. | Həkim olsaydı, kömək edərdi. |
| If I **lived** in London, I **would** visit museums. | Londonda yaşasaydım, muzeyləri gəzərdim. |

> **"Were"** — 2-ci conditional-da bütün şəxslər üçün "were" işlənə bilər (formal):
> - If I **were** rich... (formal) / If I **was** rich... (informal)

---

## Unless = If...Not

| Unless | Eyni məna |
|--------|-----------|
| Unless you study, you'll fail. | If you **don't** study, you'll fail. |
| Unless it rains, we'll go. | If it **doesn't** rain, we'll go. |

---

## Müqayisə Cədvəli

| | Şərt | Nəticə | Mənası |
|---|---|---|---|
| **Zero** | If + Pres. Simple | Pres. Simple | Həmişə doğru |
| **First** | If + Pres. Simple | will + base | Real imkan |
| **Second** | If + Past Simple | would + base | Qeyri-real xəyal |

---

## Nümunə Dialoq

**A:** What would you do if you won the lottery?
**B:** If I won the lottery, I would travel around the world!
**A:** Really? I would buy a big house first.
**B:** If you could live anywhere, where would you choose?
**A:** If I had a choice, I'd live by the sea.
**B:** Good choice! If you save some money this year, will you travel?
**A:** Yes! If I save enough, I'll go to Europe next summer.

---

## Məşq Tapşırıqları

### 1. Düzgün formanı seçin
1. If you ___ (heat) ice, it melts. [zero]
2. If she ___ (study), she will pass. [first]
3. If I ___ (be) taller, I would play basketball. [second]
4. If it ___ (not rain), we'll have a picnic. [first]

**Cavablar:** 1-heat, 2-studies, 3-were/was, 4-doesn't rain

### 2. Cümləni tamamlayın
1. If I had more free time, I would ___.
2. If it's sunny tomorrow, I'll ___.
3. If you eat too much sugar, you ___.
