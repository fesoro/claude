# B1 — Orta-aşağı Səviyyə (Threshold)

## Bu Səviyyədə Nə Öyrənəcəksiniz?

B1 səviyyəsini bitirəndə:
- Tanış mövzularda aydın danışa bilirsiniz
- TV-də aydın danışıqda əsas ideyanı anlaya bilirsiniz
- Sadə ardıcıl mətn yaza bilirsiniz
- Müsahibə verə bilərsiniz

**Orta müddət:** 5-6 ay (gündə 1 saat)  
**Lüğət hədəfi:** 2000 söz
