## Formal vs Informal — Alternativ Sözlər

| Informal | Formal/Academic |
|----------|----------------|
| get | obtain / acquire / receive |
| use | utilise / employ / apply |
| show | demonstrate / indicate / illustrate |
| think | believe / consider / argue / contend |
| say | state / claim / assert / suggest |
| start | initiate / commence / begin |
| end | conclude / terminate / cease |
| need | require / necessitate |
| help | facilitate / assist / enable |
| change | alter / modify / transform |
| increase | rise / grow / expand / surge |
| decrease | decline / reduce / fall / diminish |
| very | significantly / considerably / substantially |
| because | due to / owing to / as a result of |
| but | however / nevertheless / nonetheless |

## Akademik Cümlə Şablonları

### Fikir bildirmək
- It is argued that...
- It could be claimed that...
- There is growing evidence that...
- Research suggests / indicates that...

### Ziddiyyət göstərmək
- However, it should be noted that...
- On the contrary, some argue that...
- While this may be true, it overlooks...

### Nəticə çıxarmaq
- In conclusion, it can be said that...
- Overall, the evidence suggests...
- Taking everything into consideration...

### Nümunə vermək
- This is illustrated by...
- A clear example of this is...
- As can be seen from the data...

## Məşq Tapşırıqları

### 1. İnformai → Formal əvəzləyin
1. "We need to **get** more data." → obtain
2. "The study **shows** a link." → demonstrates/indicates
3. "This is **because of** climate change." → due to/owing to
4. "We **think** the policy failed." → believe/argue/contend

### 2. Boşluqları doldurun (AWL sözlərindən)
1. The researchers ___ (identify) three main causes.
2. This phenomenon ___ (occur) more frequently in urban areas.
3. The ___ (evidence) supports the hypothesis.
4. It is important to ___ (evaluate) all options before deciding.

**Cavablar:** 1-identified, 2-occurs, 3-evidence, 4-evaluate
