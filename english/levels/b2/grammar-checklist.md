# B2 Qrammatika Yoxlama Siyahısı

---

## ✅ Advanced Grammar Structures
- [ ] Inversion: Never have I..., Not only...but, Hardly had...
- [ ] Cleft sentences: It was X that..., What I need is...
- [ ] Participle clauses: Having finished..., Written in 1984...

## ✅ Third Conditional
- [ ] Quruluş: If + had + PP, would have + PP
- [ ] Keçmişdəki qeyri-real vəziyyət
- [ ] Mixed Conditional: If + had+PP, would + base

## ✅ Wish / If Only
- [ ] wish + past simple (indiki narazılıq)
- [ ] wish + past perfect (keçmiş narazılıq)
- [ ] wish + would (başqasının davranışından narazılıq)

## ✅ Passive — Bütün Zamanlarda
- [ ] Present/Past Continuous Passive
- [ ] Perfect Passive (has/had been done)
- [ ] Passive Infinitive: is believed to be...
- [ ] Double Passive: was seen to enter...

## ✅ Reported Speech — Mürəkkəb
- [ ] Reporting verbs: suggest/admit/warn/promise/refuse
- [ ] Reporting verbs + -ing: admit doing, deny doing
- [ ] Reporting verbs + to-inf: promise to, refuse to

## ✅ Discourse Markers (Rəsmi)
- [ ] Concession: Admittedly, Granted, While it is true...
- [ ] Emphasis: Indeed, In fact, Above all, It is worth noting...
- [ ] Cause/Effect: Due to, Owing to, Given that, Consequently

## ✅ Hedging Language
- [ ] tend to, appear to, seem to
- [ ] may/might/could + verb
- [ ] It is suggested that, Evidence indicates...
- [ ] It is widely accepted that...

## ✅ Nominalization
- [ ] Fel → isim: decide → decision, fail → failure
- [ ] Akademik yazıda nominalization istifadəsi

## ✅ Complex Sentences
- [ ] Relative clauses (defining + non-defining)
- [ ] Adverbial clauses (zaman, səbəb, güzəşt, şərt)
- [ ] Noun clauses (that-clauses, wh-clauses)
