# B2 — Orta-yuxarı Səviyyə (Vantage)

## Bu Səviyyədə Nə Öyrənəcəksiniz?

B2 səviyyəsini bitirəndə:
- Mürəkkəb mövzularda fikir mübadiləsi edirsiniz
- Ana dilli danışanlarla rahat ünsiyyət qura bilirsiniz
- Akademik məqaləni başa düşürsünüz
- Aydın, ətraflı mətn yaza bilirsiniz

**Orta müddət:** 6-8 ay (gündə 1 saat)  
**Lüğət hədəfi:** 3500 söz  
**Ekvivalent:** IELTS 5.5-6.5
