# Lüğət (Vocabulary)

## Lüğəti Necə Öyrənmək Lazımdır?

### 1. Spaced Repetition (Anki)
Sözləri bu siyahıdan Anki-yə əlavə edin. Anki hər sözü unutmadan əvvəl göstərir.

### 2. Kontekstdə Öyrənin
Yalnız sözü yox, cümləni əzbərləyin. Söz kontekstsiz unudulur.

### 3. Aktiv İstifadə
Öyrəndiyiniz sözü 24 saat ərzində istifadə edin — danışıq, yazı, və ya cümlə qurun.

### 4. Tezliyə Əsaslanan Prioritet
Ən çox işlənən 1000 söz ingilis dilindəki yazılı materialların ~80%-ni əhatə edir. Buradan başlayın.
