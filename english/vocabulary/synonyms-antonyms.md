## IELTS Writing — Avoid Repetition

Yazmada eyni sözü təkrarlamamaq üçün sinonimlərdən istifadə et:

| Original | Alternatives |
|----------|-------------|
| says | argues, suggests, claims, states, contends, asserts, maintains |
| shows | illustrates, demonstrates, indicates, reveals, highlights |
| important | significant, crucial, vital, essential, fundamental |
| many | numerous, a significant number of, a variety of, countless |
| people | individuals, citizens, residents, the public |
| problem | issue, challenge, concern, difficulty |
| because | since, as, due to the fact that, given that |
| but | however, nevertheless, nonetheless, on the other hand |
| also | furthermore, moreover, in addition, additionally |

## Common Collocations with Synonyms

| Instead of "very [adj]" | Use: |
|-------------------------|------|
| very big | enormous, vast, massive, immense |
| very small | tiny, minute, minuscule |
| very happy | delighted, ecstatic, overjoyed |
| very sad | devastated, heartbroken, miserable |
| very tired | exhausted, drained, worn out |
| very angry | furious, outraged, infuriated |
| very surprised | astonished, amazed, astounded |
| very scared | terrified, petrified, horrified |
| very good | excellent, outstanding, exceptional |
| very bad | terrible, dreadful, appalling |
