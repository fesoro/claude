# Tech Meeting Vocabulary

İş mühitində — xüsusilə meeting, standup, sprint planning və retrospektiv zamanı istifadə olunan ifadələr.

---

## 1. Standup / Daily Meeting

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| blocker | maneə, problem | "I have a blocker — I'm waiting for API access." |
| unblock | maneəni aradan qaldırmaq | "Can you unblock me by sharing the credentials?" |
| carry over | növbəti günə/sprintə keçirmək | "I'll carry this task over to tomorrow." |
| in progress | davam edir | "The feature is still in progress." |
| done / completed | bitib | "I'm done with the login page." |
| pick up | işə başlamaq | "I'll pick up the next ticket after lunch." |
| pair on | birlikdə işləmək | "Can we pair on this bug?" |

---

## 2. Sprint Planning & Project Management

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| scope | iş həcmi | "We need to define the scope before we start." |
| out of scope | plandan kənar | "That feature is out of scope for this sprint." |
| scope creep | planın genişlənməsi | "Let's be careful about scope creep." |
| story points | tapşırığın çətinlik vahidi | "I'd estimate this at five story points." |
| backlog | görüləcək işlər siyahısı | "We have twenty items in the backlog." |
| sprint | 1-2 həftəlik iş dövrü | "The sprint ends on Friday." |
| velocity | komandanın iş sürəti | "Our velocity was higher this sprint." |
| stretch goal | əlavə/bonus tapşırıq | "Let's add it as a stretch goal." |
| MVP (Minimum Viable Product) | minimum funksional məhsul | "Let's ship the MVP first." |
| ship / release | buraxmaq, deploy etmək | "We're planning to ship next Monday." |
| rollback | geri qaytarmaq | "We had to rollback the last deployment." |
| hotfix | təcili düzəliş | "We need a hotfix for this critical bug." |

---

## 3. Discussion & Decision-Making

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| circle back | sonra qayıtmaq | "Let's circle back to this after lunch." |
| take it offline | ayrıca müzakirə etmək | "This is getting detailed — let's take it offline." |
| action item | görüləcək tapşırıq | "The action item is for James to update the docs." |
| follow up | davam etmək, yenidən baxmaq | "I'll follow up with the client tomorrow." |
| heads up | xəbərdarlıq | "Just a heads up — the server will be down tonight." |
| on the same page | eyni fikirdə olmaq | "Let's make sure we're all on the same page." |
| loop in | birini qoşmaq | "Can we loop in the design team?" |
| sign off (on) | təsdiq etmək | "Has the manager signed off on this?" |
| push back | etiraz etmək, geri qaytarmaq | "The client pushed back on the deadline." |
| trade-off | güzəşt, kompromis | "There's a trade-off between speed and quality." |
| bottleneck | darboğaz | "Testing is the bottleneck right now." |
| bandwidth | imkan, resurs | "I don't have the bandwidth for that this week." |

## 4. Code Review & Technical Discussion

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| PR (Pull Request) | kod nəzərdən keçirmə sorğusu | "Can you review my PR?" |
| LGTM (Looks Good To Me) | yaxşı görünür | "LGTM — approved!" |
| nit / nitpick | kiçik qeyd | "Just a nit — can you rename this variable?" |
| refactor | kodu yenidən yazmaq | "We should refactor this module." |
| tech debt | texniki borc | "We've accumulated a lot of tech debt." |
| edge case | nadir/uç hal | "What happens in this edge case?" |
| regression | əvvəlki funksiyanın sınması | "This change caused a regression in the search." |
| breaking change | mövcudu pozan dəyişiklik | "This is a breaking change — we need to update the docs." |

## 5. Status & Updates

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| on track | planda | "The project is on track for launch." |
| behind schedule | gecikmədə | "We're a bit behind schedule." |
| ahead of schedule | vaxtından əvvəl | "We're actually ahead of schedule." |
| at risk | risk altında | "This feature is at risk of not making the deadline." |
| ETA (Estimated Time of Arrival) | təxmini vaxt | "What's the ETA on that fix?" |
| blocked on | gözləyir | "I'm blocked on the design specs." |
| wrap up | bitirmək | "Let's wrap up this meeting." |
| touch base | qısa əlaqə saxlamaq | "Let's touch base later today." |

## Məşq

Aşağıdakı cümlələrdəki boşluqları yuxarıdakı ifadələrlə doldurun:

1. "I don't have the __________ to take on another project this week."
2. "Let's __________ to this topic in the next meeting."
3. "Just a __________ — the API will have some downtime tonight."
4. "We should __________ the QA team before we deploy."
5. "This task is __________ for this sprint — we'll do it later."
6. "Can you give me an __________ on when the fix will be ready?"
7. "The __________ for this sprint is to finish the payment feature."
8. "There's a __________ between adding more features and shipping on time."
9. "The manager hasn't __________ on the new design yet."
10. "I'll __________ with the client about the requirements tomorrow."

## 8. Async Remote Communication

Remote teamlərdə video zəng olmadan yazılı şəkildə kommunikasiya:

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| async update | sinxron olmayan yenilik | "I'll post an async update in Slack." |
| EOD / EOW | end of day / end of week | "I'll finish this by EOD." |
| LGTM | looks good to me | "LGTM, approved." |
| FYI | for your information | "FYI, the staging server is down." |
| ASAP | as soon as possible | "We need this fixed ASAP." |
| heads up | xəbərdarlıq | "Just a heads up — the API is changing." |
| loop in | əlavə etmək (söhbətə) | "Please loop in the QA team." |
| circle back | geri dönmək, sonra danışmaq | "Let's circle back on this tomorrow." |
| bandwidth | vaxt/resurs (texniki yox) | "I don't have bandwidth for that this week." |
| over to you | sənin növbən | "I've finished my part — over to you." |
| on my radar | yadımda var | "That's on my radar — I'll get to it." |
| ping | qısa mesaj atmaq | "Ping me when it's ready." |
| thread | söhbət zənciri | "Let's keep this in the thread." |
| bump | yuxarı qaldırmaq (diqqəti çəkmək) | "Bumping this — any update?" |

## 9. Time Zone & Schedule Phrases

| İfadə | Məna (Az) | Misal |
|-------|-----------|-------|
| overlap hours | üst-üstə düşən iş saatları | "Our overlap is 10am-2pm EST." |
| UTC | koordinatlı universal vaxt | "The deploy runs at 03:00 UTC." |
| by [time] your time | sənin saat ilə | "Can you finish this by 5pm your time?" |
| I'm offline after... | ...dan sonra yoxam | "I'm offline after 6pm CET." |
| async-first | əvvəl yazılı/sinxronsuz | "We're an async-first team." |
| I'll pick this up tomorrow | sabah davam edəcəm | "It's late here — I'll pick this up tomorrow." |
