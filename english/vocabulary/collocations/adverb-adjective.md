# Zərf + Sifət Kollokasiyaları (Adverb + Adjective)

## Səviyyə
B1-B2

## Niyə Vacibdir?
Doğru zərf+sifət cütü işlətmək ingilis dilini peşəkar göstərir. Hər zərfi hər sifətlə işlətmək olmaz.

---

## DEEPLY — Dərin şəkildə

Duyğular, münasibətlər ilə:

| Kollokasiya | Azərbaycanca |
|-------------|-------------|
| deeply **concerned** | çox narahat |
| deeply **affected** | dərindən təsirləndı |
| deeply **moved** | dərindən hərəkatlara gəldi |
| deeply **troubled** | çox narahat, sıxıntılı |
| deeply **committed** | tam sadiq, qərarlı |
| deeply **divided** | kəskin ayrılmış |
| deeply **disappointed** | çox məyus |
| deeply **rooted** | dərindən kök salmış |
| deeply **religious** | çox dindar |
| deeply **personal** | çox şəxsi |

## ACADEMICALLY — Akademik Yazıda Ən Çox İşlənənlər

| Kollokasiya | Nümunə cümlə |
|-------------|-------------|
| **widely** accepted | This is **widely accepted** as the best approach. |
| **highly** significant | The results are **highly significant**. |
| **closely** linked/related | Poverty is **closely linked** to crime. |
| **largely** responsible | The government is **largely responsible** for this. |
| **directly** related | This is **directly related** to the main issue. |
| **increasingly** important | Education is **increasingly important** in today's world. |
| **considerably** higher | Costs were **considerably higher** than expected. |
| **significantly** different | The results were **significantly different**. |
| **particularly** relevant | This is **particularly relevant** to our discussion. |
| **primarily** concerned | This essay is **primarily concerned** with... |

## Tez-tez Yanlış Edilənlər

| Yanlış | Düzgün |
|--------|--------|
| very deeply concerned | **deeply** concerned (very olmadan) |
| highly very good | **highly** good (ikiqat intensifier olmaz) |
| completely totally wrong | **completely** wrong (biri seçilir) |
| deeply unlikely | **highly** unlikely (deeply ilə işlənmir) |

---

## Məşq Tapşırıqları

### 1. Düzgün zərfi seçin
1. She is ___ disappointed that she didn't get the job. (deeply/highly)
2. The results were ___ significant. (highly/bitterly)
3. He is ___ recommended for the position. (highly/deeply)
4. It was ___ cold last winter. (bitterly/highly)
5. The new policy is ___ controversial. (highly/perfectly)
6. She was ___ certain about her decision. (absolutely/deeply)

**Cavablar:** 1-deeply, 2-highly, 3-highly, 4-bitterly, 5-highly, 6-absolutely
