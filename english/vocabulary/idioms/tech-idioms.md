# Tech / Tech Industry Idioms — Proqramlaşdırma Slangı

## Səviyyə
B1-B2 (tech interview vacib!)

---

## Niyə Öyrənməliyik?

Tech kontekstdə xüsusi idiomlar var. Bunları bilsən:
- Team söhbətini başa düşərsən
- Daha təbii danışarsan
- Interview-da peşəkar səslənərsən

---

## 1. Code Quality

### Bikeshedding

Xırda detal üzrə çox müzakirə (vacib məsələ göz ardı).
- Stop **bikeshedding** the button color — focus on the architecture.

### Code smell

Şübhəli kod (bug olmasa da).
- That 500-line function is a serious **code smell**.

### Spaghetti code

Qarışıq, izlənilməsi çətin kod.
- This legacy service is all **spaghetti code**.

### Technical debt / Tech debt

Kod borcu (tez yazılıb sonra təmizlənməli).
- We have a lot of **tech debt** to pay off.

### Rubber duck debugging

Kağız ördəyə izah edərək bug tapmaq.
- I solved it by **rubber ducking** with my cat.

### YAGNI

**Y**ou **A**ren't **G**onna **N**eed **I**t — lazım olmayacaq feature yazma.
- Remove that unused code — **YAGNI**.

---

## 2. Performance / Scale

### Low-hanging fruit

Asan qazanc (kiçik iş, böyük effekt).
- Add database indexes first — **low-hanging fruit**.

### Silver bullet

Mucizəvi həll (yoxdur!).
- Microservices aren't a **silver bullet**.

### Premature optimization

Ehtiyac olmadan optimizasiya.
- **Premature optimization** is the root of all evil.

### Move fast and break things

Tez çatdır, səhv ol (Facebook sloqanı).
- We used to **move fast and break things**, now we move fast but test.

### Rock solid

Çox dayanıqlı.
- This service is **rock solid** — 99.99% uptime.

### Hit a wall

Davam edə bilməmək.
- We **hit a wall** with the current architecture.

### Scale up / Scale out

Vertikal / horizontal scale.
- **Scale up** the DB, **scale out** the web servers.

---

## 3. Debugging

### Needle in a haystack

İnə iynə kimi axtarmaq.
- Finding that race condition was like looking for **a needle in a haystack**.

### Heisenbug

Debugger-də gözdən qaçan bug.
- Classic **Heisenbug** — disappears when I attach the debugger.

### Hairy bug

Mürəkkəb bug.
- That's a **hairy bug** — multiple root causes.

### Works on my machine

"Mənim komputerimdə işləyir" (classic joke).
- "Have you tested it?" — "**Works on my machine**!"

### It's a feature, not a bug

"Bug deyil, feature".
- It's not a bug, it's **a feature**. (joke)

### Gremlin

Sirr bug (sanki yaramaz cin var).
- There's a **gremlin** in the code somewhere.

---

## 4. Meetings / Process

### Circle back

Sonra qayıt.
- Let's **circle back** on this next week.

### On the same page

Eyni fikirdə.
- Are we **on the same page** about the design?

### Take it offline

Sonra, ayrıca söhbət.
- Let's **take this offline** and discuss privately.

### Ping me

Zəhmət olmasa xatırlat.
- **Ping me** if you need help.

### Over the wall

Başqa tima ötürmək (mənfi).
- Don't just **throw it over the wall** to QA.

### Boil the ocean

Çox böyük iş görmək istəmək.
- Let's not **boil the ocean** — focus on MVP.

### Rabbit hole

Diqqət itirici yol.
- Don't go down that **rabbit hole** — stick to the task.

---

## 5. Projects / Planning

### MVP — Minimum Viable Product

Minimum işləyən məhsul.
- Ship the **MVP** first, iterate later.

### Happy path

Əsas istifadə ssenarisi.
- Test the **happy path** first, then edge cases.

### Edge case

Nadir hal.
- We forgot the **edge case** of empty inputs.

### Corner case

Çox nadir hal.
- A classic **corner case** — null on a weekend.

### Go-live

Production-a çıxış.
- **Go-live** is scheduled for Monday.

### Dogfooding

Öz məhsulunu istifadə etmək.
- We **dogfood** our own product.

### Eating your own dog food

Dogfooding sinonim.
- Microsoft is big on **eating their own dog food**.

---

## 6. Team / People

### 10x engineer

Çox məhsuldar mühəndis (mövhum).
- They're looking for a **10x engineer** (red flag).

### Imposter syndrome

Özünü layiqsiz hiss (psixoloji).
- Everyone experiences **imposter syndrome**.

### Burned out

İşdən yorulmuş (klinik).
- I'm **burned out** — need a break.

### Rockstar

Super mühəndis.
- She's a **rockstar** developer.

### Ninja / Guru

Ekspert (slang).
- Our JavaScript **ninja**.

### Cross-functional

Çox şöbəli.
- **Cross-functional** team with PM, design, engineering.

### Bus factor

Neçə adam gedərsə sistem dağılır.
- Our **bus factor** is 1 — only Alex knows that service.

---

## 7. Startups / Business

### Unicorn

$1B qiymətli startup.
- Stripe is a **unicorn**.

### Pivot

Biznes istiqamətini dəyişmək.
- We had to **pivot** after the market changed.

### Bootstrap

Öz gücünə başlamaq.
- We **bootstrapped** without VC.

### Lean startup

Minimum xərclə başlat.
- Follow the **lean startup** methodology.

### Runway

Qalan pul müddəti.
- 6 months of **runway**.

### Exit

Startup satmaq / IPO.
- Looking for a successful **exit**.

### Down round

Aşağı qiymətli funding.
- A **down round** after last year's hype.

---

## 8. Cloud / DevOps

### Serverless

Serversiz (idiom: əslində var, amma gizlidir).
- Going **serverless** with Lambda.

### Lift and shift

Dəyişmədən migrasiya.
- Simple **lift and shift** to AWS.

### Vendor lock-in

Satıcı bağlılığı.
- Careful of **vendor lock-in** with AWS.

### Cloud native

Bulud üçün dizayn.
- **Cloud native** architecture.

### 99.9% (Three nines)

İl boyu çox az downtime.
- We target **three nines** uptime.

### Five nines

99.999% (çox yüksək standart).
- **Five nines** is for mission-critical systems.

---

## 9. Code Review / Collaboration

### LGTM

**L**ooks **G**ood **T**o **M**e.
- **LGTM**, shipping it!

### Nit / Nitpick

Xırda iradlar.
- Just a few **nits** — approved.

### Rubber stamp

Mexaniki təsdiq.
- Don't just **rubber stamp** the PR — actually review.

### Drive-by

Tez iştirak (peşəkar deyil).
- Avoid **drive-by** code reviews.

### Yak shaving

Əsas işdən uzaqlaşan yan işlər.
- Went down **yak shaving** trying to fix the build.

---

## 10. Classic Programmer Jokes

### There are only 10 types of people...

İkili sistemdə: 0 və 1.
- "There are only 10 types of people — those who understand binary and those who don't."

### Naming things is hard

Adlandırmaq çətindir.
- "There are only two hard things in CS: cache invalidation and **naming things**."

### It works!

Möcüzə.
- "I don't know why, but **it works**." (never trust this)

### Stack Overflow-driven development

Copy-paste development.
- Classic **Stack Overflow-driven development**.

---

## 11. Speed / Shipping

### Ship it

Çatdır! Release.
- Tests pass, **ship it**!

### Rubber stamp approval

Mexaniki təsdiq (neqativ).

### Done is better than perfect

Bitirmək perfekt olmaqdan yaxşıdır.
- **Done is better than perfect** — let's ship.

### MVP mindset

Minimum viable product zehniyyəti.
- Start with an **MVP mindset**.

---

## 12. Anti-patterns

### God object

Hər şeyi edən class (pis).
- That User class is a **god object**.

### Copy-paste programming

Kopyala-yapışdır proqramlaşdırma.
- Avoid **copy-paste programming**.

### Cargo cult

Anlamadan təqlid.
- Don't **cargo cult** architectures — understand why.

### Magic numbers / strings

İzahsız sabitlər.
- Extract **magic numbers** into named constants.
