# Emosiyalar (Emotions & Feelings)

## Səviyyə: A2-B2

---

## Əsas Emosiyalar

| İngilis | Azərbaycanca | Nümunə |
|---------|--------------|--------|
| happy | xoşbəxt | I'm so **happy** today! |
| sad | kədərli | She felt **sad** after the news. |
| angry / mad | qəzəbli | He was **angry** about the delay. |
| scared / afraid | qorxmuş | I'm **scared** of spiders. |
| surprised | təəccüblü | I was **surprised** to see her. |
| excited | həyəcanlı | I'm **excited** about the trip! |
| bored | cansıxmış | I'm **bored** — let's do something. |
| tired | yorğun | I'm **tired** — I need rest. |
| nervous | əsəbi | I feel **nervous** before exams. |
| embarrassed | utandırılmış | I was so **embarrassed**! |
| proud | qürurlu | She's **proud** of her son. |
| disappointed | məyus | I was **disappointed** with the result. |
| worried | narahat | I'm **worried** about the exam. |
| confused | çaşmış | I'm **confused** by this question. |
| relieved | arxayın | I felt **relieved** after the test. |
| jealous | qısqanc | He was **jealous** of his friend's success. |
| grateful | minnətdar | I'm **grateful** for your help. |
| lonely | tənha | She felt **lonely** in the new city. |
| disgusted | iyrənmiş | I was **disgusted** by the smell. |
| ashamed | utancaqlıq hiss edən | He was **ashamed** of his behavior. |

## Emosiyaları İfadə Etmək

| İngilis | Azərbaycanca |
|---------|--------------|
| I feel... | ...hiss edirəm. |
| I'm feeling a bit... | Bir az ...hiss edirəm. |
| I can't help feeling... | ...hiss etməkdən özümü saxlaya bilmirəm. |
| It makes me feel... | Bu məni ...hiss etdirir. |
| I'm over the moon! | Çox xoşbəxtəm! (idiom) |
| I'm down in the dumps. | Mən çox kədərli hiss edirəm. (idiom) |
| I'm on edge. | Əsəbiyəm. (idiom) |
| I'm in seventh heaven. | Vəcdə gəlmişəm. (idiom) |
