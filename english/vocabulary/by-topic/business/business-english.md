# Business English

---

## 1. Meetings (Görüşlər)

### Açılış:
| Expression | Azerbaijani |
|------------|-------------|
| Let's get started. | Başlayaq. |
| Shall we begin? | Başlayaq mı? |
| The purpose of today's meeting is to... | Bu günkü görüşün məqsədi... |
| We're here today to discuss... | Bu gün ...müzakirə etmək üçün buradayıq. |
| Let me quickly run through the agenda. | Gündəliyi qısa keçim. |

### Fikir bildirmək:
| Expression | Azerbaijani |
|------------|-------------|
| I'd like to suggest that... | Təklif etmək istəyirəm ki,... |
| In my view, we should... | Mənim baxışımca, etməliyik... |
| Have we considered...? | ...nəzərdən keçirdikmi? |
| What if we tried...? | ...cəhd etsək necə? |
| I think we need to look at... | Düşünürəm ki, ...a baxmalıyıq. |

### Qərar qəbul etmək:
| Expression | Azerbaijani |
|------------|-------------|
| We've decided to... | Qərara aldıq ki,... |
| The next step is to... | Növbəti addım... |
| Who will be responsible for...? | ...kim cavabdeh olacaq? |
| The deadline is... | Son tarix... |
| Let's summarise what we've agreed. | Razılaşdığımızı ümumiləşdirək. |

## 2. Negotiations (Danışıqlar)

| Expression | Azerbaijani |
|------------|-------------|
| We'd like to propose... | Təklif etmək istəyirik... |
| Our initial offer is... | İlkin təklifimiz... |
| We're prepared to... | Hazırıq ki,... |
| That's not quite what we had in mind. | Bu tam düşündüyümüz deyil. |
| Could we find a middle ground? | Orta yol tapa bilərik? |
| We'd be willing to compromise on... | ...da kompromisə getməyə hazırıq. |
| I think we can work with that. | Düşünürəm bununla işləyə bilərik. |
| We have a deal. | Razılaşdıq. |

## 6. Common Business Idioms

| Idiom | Meaning | Example |
|-------|---------|---------|
| get the ball rolling | işə başlamaq | Let's **get the ball rolling** on this project. |
| on the same page | eyni fikirdə olmaq | Are we all **on the same page**? |
| think outside the box | yaradıcı düşünmək | We need to **think outside the box**. |
| touch base | qısaca əlaqə saxlamaq | I'll **touch base** with you tomorrow. |
| the bottom line | əsas nöqtə, son nəticə | **The bottom line** is we need more sales. |
| in the loop | məlumatlı olmaq | Keep me **in the loop**. |
| ballpark figure | təxmini rəqəm | Can you give me a **ballpark figure**? |
| cut corners | keyfiyyətə ziyan vurmaq | We can't **cut corners** on safety. |
| go the extra mile | əlavə çalışmaq | She always **goes the extra mile**. |
| hit the ground running | dərhal fəal olmaq | We need someone who can **hit the ground running**. |
