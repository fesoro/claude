## Mağazada Dialoglar

### Satıcı ilə
```
Can I help you? — Sizə kömək edə bilərəmmi?
I'm just looking, thanks. — Sadəcə baxıram, sağ olun.
Do you have this in a different size/colour? — Bunun başqa ölçüsü/rəngi varmı?
How much is this? — Bu nə qədərdir?
Is this on sale? — Bu endirimdədir?
```

### Ödəniş
```
How would you like to pay? — Necə ödəmək istərdiniz?
Cash or card? — Nağd yoxsa kartla?
Do you have a receipt? — Çekiniz varmı?
Can I have a bag, please? — Çanta verərsəniz?
Keep the change. — Üstü sizin olsun.
```

### Geri qaytarma
```
I'd like to return this. — Bunu geri qaytarmaq istəyirəm.
It doesn't fit. — Ölçüsü olmur.
It's faulty / broken. — Qüsurludur / sınıqdır.
Can I exchange it? — Dəyişdirə bilərəmmi?
```

## Məşq

**Bir alış-veriş dialoqu yazın (8-10 cümlə):**
Siz bir geyim mağazasındasınız. Bir köynək tapmaq istəyirsiniz. Satıcıya suallar verin, ölçüsünü soruşun, qiyməti öyrənin, ödəyin.
