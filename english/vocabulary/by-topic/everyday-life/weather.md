---

## Qrammatika: "It" ilə hava ifadəsi

Hava haqqında danışanda həmişə **it** subyekti işlənir:

| İngilis | Azərbaycanca |
|---------|--------------|
| It is sunny. | Hava günəşlidir. |
| It is raining. | Yağış yağır. |
| It is snowing. | Qar yağır. |
| It was cold yesterday. | Dünən soyuq idi. |
| It will be warm tomorrow. | Sabah isti olacaq. |
| It is 25 degrees. | 25 dərəcədir. |

## Faydalı İfadələr

| İngilis | Azərbaycanca |
|---------|--------------|
| What's the weather like? | Hava necədir? |
| What's the forecast for tomorrow? | Sabahkı hava proqnozu necədir? |
| It looks like rain. | Yağış yağacaq kimi görünür. |
| It's getting colder / warmer. | Hava soyuyur / istiləşir. |
| The weather is terrible / awful. | Hava dəhşətlidir. |
| What lovely weather! | Nə gözəl havadır! |
| I'm soaking wet! | Tamamilə islanmışam! |
| It's freezing cold! | Dondurucu soyuqdur! |
| Take an umbrella — it might rain. | Çəтir götür — yağış yağa bilər. |

---

## Faydalı Kollokasiyalar

| Fel | + Hava | Cümlə |
|-----|--------|-------|
| it **is** | sunny, cold, hot | It **is** sunny today. |
| it **rains / is raining** | — | It **rains** a lot in autumn. |
| it **snows / is snowing** | — | It **was snowing** last night. |
| the sun **shines** | — | The sun **shines** brightly. |
| the wind **blows** | — | A cold wind **is blowing**. |
| temperatures **rise / fall** | — | Temperatures **fall** in winter. |

---

## Məşq

**A1:** Bu gün hava necədir? 3 cümlə yazın.
**B1:** Sevdiyiniz fəsli izah edin. Niyə sevdiyinizi 5-6 cümlə ilə yazın.
