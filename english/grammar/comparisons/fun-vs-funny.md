# Fun vs Funny — Əyləncəli vs Gülməli

## Səviyyə
A2-B1 (Azərbaycanlılar üçün klassik səhv!)

---

## Əsas Fərq

| Söz | Məna | Test |
|-----|------|------|
| **fun** | əyləncəli, xoş | Xoş vaxt keçir? → fun |
| **funny** | gülməli, qəribə | Güldürür? → funny |

> **Qısa qayda:**
> - **fun** = xoş, əyləncəli (enjoyable)
> - **funny** = gülməli (makes you laugh) **VƏ YA** qəribə (strange)

---

## 1. Fun — Əyləncəli / Xoş

Adətən **noun** (isim) kimi, bəzən sifət.

### İsim kimi

- We had **fun**.
- It was **fun**.
- **Fun** at the party.
- **Fun** and games.

### Sifət kimi

- A **fun** game.
- A **fun** person.
- **Fun** activity.
- A **fun** trip.

### Nümunələr

- Learning Python is **fun**.
- The team is really **fun** to work with.
- Code reviews can be **fun**.

---

## 2. Funny — Gülməli və ya Qəribə

Həmişə sifət.

### A. Gülməli (makes you laugh)

- A **funny** joke.
- That's **funny**!
- She's so **funny**.
- **Funny** video.

### B. Qəribə / anlaşılmaz

- That's **funny** — my code worked yesterday.
- A **funny** smell.
- The server is acting **funny**.
- **Funny** behavior in the logs.

---

## 3. Kritik Fərq

### Cümlə 1

- "The party was **fun**." = Partiya əyləncəli idi. (xoş vaxt)
- "The party was **funny**." = Partiya gülməli idi. (güldürdü — bəlkə də qərib)

### Cümlə 2

- "He's a **fun** person." = O əyləncəli insandır. (xoş, onunla vaxt keçirmək xoşdur)
- "He's a **funny** person." = O gülməli adamdır. (güldürür — stand-up kimi)

---

## Test

Fun / Funny?

1. Learning new frameworks is ______ . (əyləncəli)
2. That joke was really ______ . (gülməli)
3. We had so much ______ at the team outing. (xoş)
4. The bug is acting ______ — inconsistent behavior. (qəribə)
5. She's a ______ colleague. (xoş, onunla işləmək rahatdır)
6. She's a ______ colleague. (güldürür)

**Cavablar:**
1. fun
2. funny
3. fun
4. funny (qəribə mənada)
5. fun
6. funny

---

## İnterview / Tech Kontekstində

- "It was a **fun** project." ✓ (xoş, enjoyable)
- "The bug is acting **funny**." ✓ (qərib)
- "I enjoyed it — it was **fun**." ✓
- "That's a **funny** story about debugging." ✓ (gülməli)

---

## Azərbaycanlı Səhvləri

- ✗ Python is funny. (language-ə "gülməli" demisən!)
- ✓ Python is **fun**. (əyləncəli)

- ✗ We had funny at the party. (yox — funny isim deyil, adjective)
- ✓ We had **fun** at the party.

- ✗ He's a very fun comedian. (komik → funny)
- ✓ He's a very **funny** comedian.

---

## Fun + Noun kombinasiyaları

- **fun** fact (maraqlı fakt)
- **fun** run
- **fun**-loving
- just for **fun**
- make **fun of** = lağ etmək (mənfi!)

**Diqqət:** "make fun of someone" = lağ etmək. **Negativ**!

---

## Funny bone / Funny story

- **funny bone** = qolun alt tərəfindəki yer (çırpılanda ağrıyır)
- **funny story** = maraqlı/gülməli hekayə

---

## Xatırlatma

| Sual | Söz |
|------|-----|
| Xoş, maraqlı? | **fun** |
| Güldürür? | **funny** |
| Qəribə / anlaşılmaz? | **funny** |

**Test metodu:** "Güldü?" → funny. "Xoş vaxt keçirdi?" → fun.
