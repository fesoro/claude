# Prevent / Avoid / Prohibit — Qarşısını Almaq

## Səviyyə
B1

---

## Əsas Fərq

| Söz | Kim edir? | Necə? |
|-----|-----------|-------|
| **prevent** | aktiv addım atmaq | qarşısını almaq |
| **avoid** | yan keçmək | daxil olmamaq |
| **prohibit** | qadağan etmək (rəsmi) | qayda ilə |

> **Qısa qayda:**
> - **prevent** = qarşısını al (aktiv)
> - **avoid** = yan keç (passiv / şəxsi seçim)
> - **prohibit** = rəsmi qadağa

---

## 1. Prevent — Qarşısını Almaq (Aktiv)

Bir şey baş verməsin deyə tədbir görürsən.

### Struktur

- **prevent** + N
- **prevent** + N + from + V-ing
- **prevent** + V-ing (nadir)

### Nümunələr

- **Prevent** bugs.
- **Prevent** fires.
- This **prevents** data loss.
- The firewall **prevents** attacks.
- Tests **prevent** regressions.
- We **prevented** the outage.

### Prevent from

- This **prevents** users **from** losing data.
- Guards **prevent** access.

---

## 2. Avoid — Yan Keçmək (Passiv)

Zəruri deyildir, sadəcə onunla qarşılaşma.

### Struktur

- **avoid** + N
- **avoid** + V-ing

**Diqqət:** "avoid" + **V-ing** (no "to"!)

### Nümunələr

- **Avoid** traffic.
- **Avoid** mistakes.
- **Avoid** conflict.
- I'm trying to **avoid** coffee.
- **Avoid** making the same mistake.
- He **avoided** answering.

### ✗ Yanlış struktur

- ✗ **Avoid to be late.**
- ✓ **Avoid being late.**
- ✓ **Avoid** lateness. (no "to")

### Prevent vs Avoid

- **prevent** = qarşı aktiv hərəkət
- **avoid** = ondan uzaq qaç

- "Tests **prevent** bugs." (tədbir — bug-lar ortaya çıxmır)
- "I **avoid** buggy code." (ondan uzaq qalıram)

Fərqi:
- **Prevent** = bir şeyin baş verməsinə MANEÇİSƏN (aktiv)
- **Avoid** = bir şeydən UZAQSAN (passiv)

---

## 3. Prohibit — Rəsmi Qadağa

Qanun, qayda, siyasət. Rəsmi.

### Struktur

- **prohibit** + N
- **prohibit** + V-ing
- **be prohibited from** + V-ing

### Nümunələr

- Smoking is **prohibited**.
- The law **prohibits** this.
- **Prohibited** items.
- You are **prohibited from** entering.

### Prevent vs Prohibit

- **prevent** = fiziki / sistematik manea
- **prohibit** = qayda ilə qadağa

- The lock **prevents** entry. (fiziki)
- The rule **prohibits** entry. (qayda)

### Gündəlik alternative

- **forbidden** = qadağan
- **not allowed**

---

## Kritik Fərq Test

### Prevent

"I **prevent** X from happening."
→ Mən tədbir görürəm → X olmur.

### Avoid

"I **avoid** X."
→ Mən X-dən uzaq qalıram (X ola bilər, amma mən yoxam).

### Prohibit

"X is **prohibited**."
→ X qadağandır (qayda).

## Tech / İş Kontekstində

### Prevent

- **Prevent** XSS attacks. ✓
- **Prevent** data leaks. ✓
- **Prevent** race conditions. ✓
- Tests **prevent** regressions. ✓
- **Prevent** timeouts. ✓

### Avoid

- **Avoid** tight coupling. ✓
- **Avoid** premature optimization. ✓
- **Avoid** making direct DB calls. ✓
- **Avoid** using deprecated APIs. ✓

### Prohibit

- Direct commits to main are **prohibited**. ✓
- Force pushes are **prohibited**. ✓

## Azərbaycanlı Səhvləri

- ✗ Avoid to repeat the same mistake. (to yox — V-ing!)
- ✓ **Avoid repeating** the same mistake.

- ✗ I prevent him to enter. (struktur → from + V-ing)
- ✓ I **prevented him from entering**.

- ✗ Smoking is avoided here. (rəsmi → prohibited / forbidden)
- ✓ Smoking is **prohibited** / **forbidden** here.

## Related Sözlər

### Stop

Sadə durdurmaq.
- **Stop** the deployment.

### Block

Fiziki manea.
- **Block** the request.

### Dodge

İnformal yan keçmək.
- **Dodge** the question.

### Forbid

Qadağa — prohibit-a yaxın.
- It's **forbidden**.

### Ban

Güclü qadağa.
- The site is **banned**.

### Prohibit-in formal spektri

```
discourage < avoid < prevent < forbid < prohibit < ban
```
