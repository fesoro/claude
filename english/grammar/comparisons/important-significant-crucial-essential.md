# Important / Significant / Crucial / Essential / Critical — Əhəmiyyət Sifətləri

## Səviyyə
B1

---

## Əsas Cədvəl (İntensivlik)

| Söz | Dərəcə | Kontekst |
|-----|--------|----------|
| **important** | vacib | ümumi |
| **significant** | əhəmiyyətli, böyük | rəsmi |
| **essential** | əsas (must-have) | güclü |
| **crucial** | həlledici | güclü |
| **critical** | kritik (life-death) | çox güclü |
| **vital** | həyati | çox güclü |

> **Qısa qayda:**
> - **important** = vacib (ümumi)
> - **significant** = böyük təsirli
> - **essential** = olmazsa-olmaz
> - **crucial / critical** = həlledici / kritik
> - **vital** = həyati

---

## İntensivlik Skalası

```
important < significant < essential < crucial ≈ critical ≈ vital
     normal              must-have       life-death
```

---

## 1. Important — Vacib (Ümumi)

Ən çox işlənir. Ümumi vaciblik.

### Nümunələr

- This is an **important** decision.
- **Important** meeting.
- **Important** information.
- Very **important**.

### Struktur

- **important** to + V
- **important** for + N
- **important** that + clause

- It's **important** to test.
- **Important** for the team.
- **Important** that we finish on time.

---

## 2. Significant — Əhəmiyyətli / Böyük

Rəsmi. Ölçüləbilən təsiri var.

### Nümunələr

- **Significant** improvement.
- **Significant** impact.
- **Significant** change.
- **Significant** difference.
- **Significantly** better.

### Important vs Significant

- **important** = vacib
- **significant** = böyük fərqi var (measurable)

- "An **important** bug." (vacibdir)
- "A **significant** bug." (təsir böyükdür)

### Tech / statistics

- **Statistically significant**.
- **Significant** performance gain.
- **Significant** code change.

---

## 3. Essential — Əsas (Must-have)

Olmazsa, olmaz. Zəruri.

### Nümunələr

- **Essential** feature.
- **Essential** skill.
- Absolutely **essential**.
- **Essentially**, it works like...

### Important vs Essential

- **important** = vacib (amma olmadan da olar)
- **essential** = olmazsa, işləmir

- "Tests are **important**." (vacib)
- "Tests are **essential**." (olmazsa olmaz)

### Synonym

- must-have
- vital
- indispensable

---

## 4. Crucial — Həlledici

Önəmli qərar / məqam. Strong.

### Nümunələr

- **Crucial** moment.
- **Crucial** decision.
- **Crucial** for success.
- Absolutely **crucial**.

### Important vs Crucial

- **important** = vacib
- **crucial** = çox vacib (həlledici)

- "Documentation is **important**." (vacib)
- "Documentation is **crucial** for onboarding." (həlledici)

### Tech

- **Crucial** feature for launch.
- **Crucial** bug fix.
- **Crucial** review before deploy.

---

## 5. Critical — Kritik

Ən güclü. Sistem / həyat səviyyəsində.

### Nümunələr

- **Critical** bug.
- **Critical** error.
- **Critical** infrastructure.
- **Critical** path.
- In **critical** condition.

### Tech kontekst

- **Critical** severity bug (SEV-1).
- **Critical** production issue.
- **Critical** patch.
- **Mission-critical** system.

### Crucial vs Critical

- **crucial** = həlledici
- **critical** = kritik (failure riski)

- "**Crucial** decision" (vacib qərar)
- "**Critical** bug" (kritik bug — sistemi bozur)

Tech incident kontekstində **critical** ən güclü.

---

## 6. Vital — Həyati

**Critical**-a yaxın. Rəsmi.

### Nümunələr

- **Vital** organs (bədən).
- **Vital** information.
- **Vital** to the mission.
- Of **vital** importance.

### Vital vs Critical

Demək olar eyni. **Vital** daha rəsmi / ədəbi.

Tech-də **critical** üstün.

---

## Test

Hansı söz uyğundur?

1. This bug is ______ — it brings down the whole system. (kritik)
2. Regular backups are ______ . (olmazsa olmaz)
3. The meeting is ______ to our quarterly planning. (ümumi)
4. We made a ______ improvement in speed. (ölçülə bilən böyük)
5. This decision is ______ for our future. (həlledici)
6. The engine is ______ to the car. (həyati)

**Cavablar:**
1. critical
2. essential / vital
3. important
4. significant
5. crucial
6. vital / essential

## Tech / İş Kontekstində

### Important (general)

- "It's **important** to test." ✓

### Significant (measurable)

- "**Significant** performance boost." ✓
- "**Significant** code refactor." ✓

### Essential (must-have)

- "Tests are **essential**." ✓
- "Monitoring is **essential**." ✓

### Crucial (decisive)

- "Timing is **crucial**." ✓
- "**Crucial** feature for launch." ✓

### Critical (failure risk)

- "**Critical** security vulnerability." ✓
- "**Critical** incident response." ✓
- "SEV-1 **critical** issue." ✓

### Vital (formal)

- "**Vital** business function." ✓

## Azərbaycanlı Səhvləri

- ✗ Critical meeting tomorrow. (sadəcə vacib → important)
- ✓ **Important** meeting tomorrow.

(Amma həqiqətən kritik, qərar tələb edən olsa → critical OK)

- ✗ Essential to finish, but not urgent. (olmazsa olmaz → crucial)
- ✓ **Crucial** to finish, but not urgent.

## Related Sözlər

### Key

Ən vacib.
- **Key** stakeholder.
- **Key** feature.

### Major

Böyük əhəmiyyətli.
- **Major** update.

### Paramount

Çox rəsmi "ən vacib".
- Of **paramount** importance.

### Indispensable

Əvəz edilməz.
- **Indispensable** team member.

### Pivotal

Həlledici, dönüş nöqtəsi.
- **Pivotal** moment.
