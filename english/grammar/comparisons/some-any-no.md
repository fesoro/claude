# Some / Any / No — Kəmiyyət Göstəriciləri

## Səviyyə
A1-A2

---

## Əsas Qayda — Tez Baxış

| Forma | İşlənmə | Nümunə |
|-------|---------|--------|
| **some** | müsbət (affirmative) | I have **some** money. |
| **any** | sual, inkar | Do you have **any** money? / I don't have **any**. |
| **no** | güclü inkar (fel müsbət) | I have **no** money. |

> Ümumi qayda: müsbət cümlədə `some`, sual/inkarda `any`, güclü inkarda `no` (fel isə affirmative qalır).

---

## 1. Some — müsbət cümlələr

Sayıla bilən cəm və ya sayılmayan isimlərlə:

- I have **some** friends in London. (cəm)
- There is **some** water in the glass. (sayılmayan)
- She bought **some** bread.
- I need **some** help.
- We met **some** interesting people.

### Some istisnaları — TƏKLİF və XAHİŞdə

"Cavab bəli gözləyirəm" → sualda belə **some** işlənir:

- Would you like **some** coffee? (təklif)
- Can I have **some** water, please? (nəzakətli xahiş)
- Could you give me **some** advice? (xahiş)

> **Qayda:** Təklif və müsbət cavab gözlənilən suallarda `some`.

---

## 2. Any — suallar və inkarlar

### Suallarda

- Do you have **any** siblings?
- Is there **any** milk left?
- Are there **any** questions?
- Have you got **any** ideas?

### İnkarda (not + any)

- I don't have **any** money.
- There isn't **any** sugar.
- She didn't make **any** mistakes.
- We haven't got **any** bread.

### "Hər hansı, istənilən" (müsbət cümlədə də)

- You can call me at **any** time. (istənilən vaxt)
- Take **any** book you want. (istənilən)
- **Any** help would be great.

> Bu halda `any` = "fərqi yoxdur, hansı olsa olar".

---

## 3. No — güclü inkar

`no` istifadə edəndə fel **müsbət** qalır (ikiqat inkar yoxdur):

- I have **no** money. (= I don't have any money.)
- There is **no** time.
- She has **no** friends here.
- We had **no** choice.

| İki ifadə eynidir: |
|--------------------|
| I don't have **any** time. = I have **no** time. |
| There isn't **any** bread. = There is **no** bread. |

> **Diqqət:** `no` daha güclü, daha emfatikdir:
> - "I don't have any money." (neytral)
> - "I have no money." (daha güclü)

---

## 4. Compounds — Birləşmiş formalar

| some- | any- | no- |
|-------|------|-----|
| **someone** / **somebody** | **anyone** / **anybody** | **no one** / **nobody** |
| **something** | **anything** | **nothing** |
| **somewhere** | **anywhere** | **nowhere** |

### Nümunələr

- **Someone** is knocking at the door.
- Is there **anyone** here?
- I saw **nobody** in the office.
- I need **something** to eat.
- Did you buy **anything**?
- There is **nothing** in the fridge.
- Let's go **somewhere** nice.
- I can't find my keys **anywhere**.
- He is **nowhere** to be seen.

### Eyni qayda tətbiq olunur

- Müsbət: something, someone, somewhere
- Sual/inkar: anything, anyone, anywhere
- Güclü inkar: nothing, nobody, nowhere

---

## 5. Not any vs No — Fərq

| Vəziyyət | any / no |
|----------|----------|
| Cümlənin əvvəlində | **No** (Not any işlənmir) |
| Fel "to be" ilə | **There is no / There isn't any** |
| Emfatik təsir | **no** (daha güclü) |
| Neytral | **not any** |

- ✓ **No** students came to class.
- ✗ Not any students came. (belə başlanmır)
- ✓ There were **no** students. / There weren't **any** students.

---

## 6. Some/Any + of + the

Müəyyən qrupdan bir hissə:

- **Some of** the students are here.
- **Any of** these books is fine.
- **None of** the answers are correct.

---

## Tez-tez Edilən Səhvlər

| Səhv | Düzgün | İzah |
|------|--------|------|
| Do you have some money? | Do you have **any** money? | Sualda adətən any |
| I don't have some friends. | I don't have **any** friends. | İnkarda any |
| I have not no money. | I have **no** money. / I **don't have any** money. | İkiqat inkar yoxdur |
| Would you like any coffee? | Would you like **some** coffee? | Təklifdə some |
| There isn't nothing here. | There is **nothing** here. / There isn't **anything**. | İkiqat inkar yoxdur |
| I need something informations. | I need **some** information. | information — uncountable, compound yox |

## Məşq Tapşırıqları

### 1. some / any / no seçin

1. There is ___ milk in the fridge, sorry.
2. I need ___ help with this task.
3. Did you buy ___ bread?
4. Would you like ___ tea?
5. I have ___ idea what to do. (heç bir ideyam yoxdur)
6. Are there ___ good restaurants near here?
7. Take ___ book you like.

**Cavablar:** 1-no, 2-some, 3-any, 4-some, 5-no, 6-any, 7-any

### 2. somebody / anybody / nobody seçin

1. Is ___ home?
2. ___ called while you were out. (naməlum şəxs)
3. I saw ___ — the office was empty.
4. Does ___ know the answer?

**Cavablar:** 1-anybody, 2-Somebody, 3-nobody, 4-anybody

### 3. İki formada yazın

Nümunə: There isn't any water. → There is **no** water.

1. I don't have any time. → ___
2. She doesn't have any friends. → ___
3. There aren't any tickets left. → ___

**Cavablar:** 1-I have no time. 2-She has no friends. 3-There are no tickets left.
