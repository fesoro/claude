# Cheap / Inexpensive / Affordable / Expensive / Costly / Pricey — Qiymət Sifətləri

B1

## Əsas Cədvəl

### Ucuz / Sərfəli

| Söz | Tonu |
|-----|------|
| **cheap** | ucuz (bəzən neqativ!) |
| **inexpensive** | ucuz (neytral) |
| **affordable** | sərfəli (pozitiv) |
| **budget** | büdcəyə uyğun |
| **low-cost** | ucuz xərcli |

### Bahalı

| Söz | Tonu |
|-----|------|
| **expensive** | bahalı (neytral) |
| **costly** | bahalı (rəsmi) |
| **pricey** | bahalı (informal) |
| **high-priced** | yüksək qiymətli |
| **overpriced** | həddən bahalı (neqativ!) |

> **Qısa qayda:**
> - **cheap** = ucuz (bəzən keyfiyyətsiz kimi səslənir)
> - **affordable** = sərfəli (pozitiv!)
> - **expensive** = bahalı (neytral)
> - **overpriced** = həddən bahalı (pis)

---

## 1. Cheap — Ucuz (Ehtiyatlı!)

**İki məna:**
- **A. Ucuz** (qiymət az) — neytral
- **B. Keyfiyyətsiz** — NEGATİV!

### Nümunələr

- A **cheap** flight.
- **Cheap** hotel.
- Too **cheap** to be good. (şübhəli)
- **Cheap** materials. (aşağı keyfiyyət)

### ⚠ Tələ

"Cheap" bəzən "pinti" mənasında da:
- He's so **cheap**. (pinti — pul xərcləmir)

### Cheap vs Affordable

- **cheap** = ucuz (bəzən keyfiyyətsiz)
- **affordable** = sərfəli (pozitiv)

- A **cheap** car. (aşağı keyfiyyət təəssüratı)
- An **affordable** car. (qiyməti münasib)

**Biznesdə "affordable" daha yaxşıdır!**

---

## 2. Inexpensive — Ucuz (Neytral)

"Cheap"-in neytral versiyası. Keyfiyyət neqativ deyil.

### Nümunələr

- An **inexpensive** meal.
- **Inexpensive** option.
- Relatively **inexpensive**.

### Cheap vs Inexpensive

- **cheap** = pulsuz / aşağı keyfiyyət hissi
- **inexpensive** = sadəcə az qiymət

Yazıda və rəsmi kontekstdə **inexpensive** üstün.

---

## 3. Affordable — Sərfəli (Pozitiv!)

Büdcəyə uyğun. **Həmişə pozitiv**.

### Nümunələr

- **Affordable** housing.
- **Affordable** price.
- **Affordable** for students.
- **Affordability** index.

### Marketing favorite

Reklam / biznesdə:
- "**Affordable** luxury"
- "**Affordable** quality"

Müştəri üçün pozitiv səslənir.

---

## 4. Budget — Büdcə Dostu

Budget-friendly — büdcə imkanlarında.

### Nümunələr

- **Budget** phone.
- **Budget** laptop.
- **Budget** travel.
- **Budget-friendly** options.

### Compound

- **low-budget** = az büdcəli (neqativ çalar)
- **budget-friendly** = büdcəyə uyğun (pozitiv)
- **tight budget** = sıx büdcə

---

## 5. Expensive — Bahalı (Neytral)

Ən çox istifadə. Neytral.

### Nümunələr

- **Expensive** car.
- **Expensive** restaurant.
- Too **expensive**.
- **Expensive** tastes.

### Struktur

- **expensive** for + person
- This is **too expensive for me**.

---

## 6. Costly — Bahalı (Rəsmi)

Expensive-dən rəsmi. Həm də "təsirlidən".

### Nümunələr

- **Costly** mistake.
- **Costly** repair.
- **Costly** delays.

### Expensive vs Costly

- **expensive** = gündəlik
- **costly** = rəsmi, bahalı

### Abstract istifadə

- A **costly** mistake = baha bir səhv (yalnız maddi deyil)
- A **costly** decision = təsirli qərar

Tech-də:
- A **costly** operation (CPU / memory).
- **Costly** downtime.

---

## 7. Pricey — Bahalı (İnformal)

İnformal "expensive". Gündəlik söhbət.

### Nümunələr

- That's **pricey**!
- A bit **pricey**.
- **Pricey** restaurant.

### Expensive vs Pricey

- **expensive** = rəsmi / neytral
- **pricey** = informal, "bir az çox bahadır"

Gündəlik danışıqda **pricey** təbii.

---

## 8. Overpriced — Həddən Bahalı (NEGATİV!)

"Lazım olandan çox baha". Dəyərindən baha.

### Nümunələr

- That's **overpriced**.
- **Overpriced** restaurant.
- Completely **overpriced**.

### Kontekst

- Müştəri şikayəti
- Rəy / tənqid

**Neqativ söz — ehtiyatlı işlət!**

---

## Bonus Sözlər

### Reasonable

Ağıllı qiymət.
- A **reasonable** price.

### Fair

Ədalətli.
- A **fair** price.

### Steep

Şıxışıcı (nadir, informal).
- The price is **steep**.

### A rip-off (isim)

Aldatma qiyməti.
- That's a **rip-off**!

### A bargain

Qazanılan alış.
- What a **bargain**!

### A steal

Ucuz alış.
- It's **a steal**!

## Test

Hansı söz uyğundur?

1. We offer ______ prices for students. (pozitiv, sərfəli)
2. Those headphones are too ______ — I can't justify it. (neqativ, həddən)
3. This restaurant is a bit ______ . (informal bahalı)
4. ______ materials break easily. (neqativ keyfiyyət)
5. A ______ mistake that caused the outage. (rəsmi, təsirli)
6. ______ hotel for a weekend trip. (neytral ucuz)

**Cavablar:**
1. affordable
2. overpriced
3. pricey
4. Cheap
5. costly
6. inexpensive / budget

## İnterview / Tech Kontekstində

### Tech-də

- **Costly** operation = CPU-heavy ✓
- **Costly** mistake ✓
- **Expensive** query = slow SQL ✓
- **Cheap** compute = inexpensive resources ✓

### İş

- "We need **affordable** infrastructure." ✓ (pozitiv)
- "The outage was **costly**." ✓ (təsirli)
- "An **expensive** license." ✓

## Azərbaycanlı Səhvləri

- ✗ Cheap quality. (oxymoron deyil, amma neqativ)
- ✓ **Low quality** / **poor quality**.

- ✗ He's cheap. (→ bu "pinti")
- Meaning: pinti → He's **stingy**. / He's **frugal** (milder).

- ✗ Very cheap restaurant for students. (reklamda → affordable)
- ✓ Very **affordable** restaurant for students.
