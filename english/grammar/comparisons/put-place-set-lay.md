# Put / Place / Set / Lay — Qoymaq Fellər

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Tonu | Xüsusi |
|-----|------|--------|
| **put** | ümumi | ən çox işlənən |
| **place** | rəsmi | diqqətli qoyma |
| **set** | qurmaq / təyin et | çoxmənalı |
| **lay** | düzgün qoy (flat) | müəyyən səth |

> **Qısa qayda:**
> - **put** = qoy (ümumi)
> - **place** = rəsmi qoy (diqqətlə)
> - **set** = qoyub quraşdır
> - **lay** = düz qoy (flat / horizontal)

---

## 1. Put — Qoymaq (Ümumi)

Ən çox istifadə. Hər kontekstdə.

### Forma

- put → put → put (heç dəyişmir!)

### Nümunələr

- **Put** the book on the table.
- **Put** it here.
- **Put** your bag down.
- **Put** the keys in the drawer.

### İstifadə

- Hər şey üçün (fiziki / abstract).

### Abstract

- **Put** pressure on.
- **Put** the blame on.
- **Put** an effort.
- **Put** in time.

### Phrasal (çox!)

- **put on** = geyin
- **put off** = təxirə sal
- **put up** = qurmaq
- **put down** = aşağı qoy
- **put away** = yığışdır
- **put together** = birləşdir

---

## 2. Place — Rəsmi Qoy

Diqqətlə, dəqiq yerə qoy. Rəsmi.

### Nümunələr

- **Place** the vase here.
- **Place** your order.
- **Place** your bet.
- **Place** carefully.

### Put vs Place

- **put** = sadə qoy
- **place** = diqqətlə qoy

- "**Put** the book." (sadə)
- "**Place** the book carefully." (diqqətlə)

### Rəsmi kontekstlər

- **Place** your order. (sifariş ver)
- **Place** a call. (zəng et)
- **Place** value on.

### İsim

- A nice **place**.
- My **place**.

---

## 3. Set — Qurmaq / Təyin Et

Çoxmənalı. "Put" + "arrange".

### Forma

- set → set → set (dəyişmir!)

### A. Qurmaq / Yerləşdir

- **Set** the table.
- **Set** a date.
- **Set** the clock.
- **Set** the temperature.

### B. Təyin etmək

- **Set** a goal.
- **Set** rules.
- **Set** the standard.
- **Set** a deadline.

### C. Get ready

- **Set** up.
- Get **set**.
- **Set** and ready.

### D. Günəş / gün

- The sun **sets** at 6.

### Put vs Set

- **put** = sadə qoy
- **set** = qoyub arrange / configure

- **Put** the plate. (sadə qoy)
- **Set** the table. (bütün hazırlıq)

### Tech kontekst

- **Set** a variable.
- **Set** the config.
- **Set** up the environment.

---

## 4. Lay — Düz Qoy (Flat)

Horizontal olaraq, səthə qoymaq.

### ⚠ Diqqət: Lay vs Lie (tələ!)

Ayrıca fayl: [lay-vs-lie.md](lay-vs-lie.md)

- **lay** = qoymaq (transitive — need object!)
- **lie** = uzanmaq (intransitive — no object!)

### Forma (lay = transitive)

- lay → laid → laid

### Nümunələr

- **Lay** the blanket on the bed.
- **Lay** the cards on the table.
- **Lay** the foundation.
- **Lay** a brick.

### Put vs Lay

- **put** = qoy (hər yerə)
- **lay** = düz, səthə qoy

- **Put** the book. (sadə)
- **Lay** the book flat. (düz, açıq)

## Nümunə Müqayisəsi

### Masa hazırlamaq

- **Put** the plates. (sadə qoy)
- **Place** the plates. (diqqətlə qoy)
- **Set** the table. (bütün masa hazırla — preferred!)
- **Lay** the plates. (düz qoy — rare)

En təbii: **Set the table**.

### Kitab

- **Put** the book on the desk. ✓ (ümumi)
- **Place** the book carefully. ✓ (diqqət)
- **Lay** the book flat. ✓ (açıq, horizontal)
- ~~Set the book~~ ✗ (set ümumi kitab üçün uyğun deyil)

## Tech Kontekstində

### Put

- **Put** data in the cache. ✓
- **Put** items in a list. ✓
- HTTP **PUT** method (REST API). ✓

### Place

- Rarely used (except "place an order").

### Set

- **Set** environment variables. ✓
- **Set** a variable = x = 5. ✓
- **Set** the config. ✓
- **Set** up a new service. ✓
- **Setup** / **setup guide**. ✓

### Lay

- Rare in tech.
- **Lay** the groundwork / foundation. ✓ (metaphoric)

## Phrasal Verbs (Put ilə)

Put ən çox phrasal fellərin əsasıdır:

- **put off** = təxirə sal
- **put on** = geyin / aç
- **put up** = qurmaq
- **put down** = qoy (aşağı)
- **put away** = təmizlə / saxla
- **put together** = birləşdir
- **put forward** = təklif et
- **put out** = söndür / çap et
- **put through** = bağla (telefonda)
- **put up with** = dözmək

### Set phrasal

- **set up** = qur
- **set out** = başla / yola düş
- **set off** = başla / partla (alarm)
- **set back** = gecikdir
- **set aside** = ayır

---

## Azərbaycanlı Səhvləri

- ✗ Set the book here. (→ put)
- ✓ **Put** the book here.

- ✗ Put the table for dinner. (→ set)
- ✓ **Set** the table for dinner.

- ✗ Lay the book down. (transitive — amma dialect-ə görə)
- ✓ **Lay** the book down. ✓ (actually OK)
- But: **Lie** down. (you yourself — intransitive)

---

## Common Mistakes — Lay vs Lie

Bu ən çox səhv salınır:

- ✓ **Lay** the book down. (transitive — book is object)
- ✓ I need to **lie** down. (intransitive — no object)

- ✗ I'm laying on the couch.
- ✓ I'm **lying** on the couch.

→ Ətraflı: [lay-vs-lie.md](lay-vs-lie.md)
