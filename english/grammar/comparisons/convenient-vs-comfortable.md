# Convenient vs Comfortable — Əlverişli vs Rahat

## Səviyyə
A2-B1 (Azərbaycanlılar üçün false friend!)

---

## Niyə Tələ?

Azərbaycanca "rahat" = həm fiziki, həm də vəziyyətcə. İngilis dilində iki fərqli söz:
- **comfortable** = fiziki rahat (bədən, hiss)
- **convenient** = əlverişli, uyğun (vaxt, məkan, praktik)

---

## Əsas Fərq

| Söz | Məna |
|-----|------|
| **comfortable** | fiziki rahat (soft, cozy) |
| **convenient** | əlverişli, uyğun (practical, suitable) |

---

## 1. Comfortable — Fiziki Rahat

Bədən hiss edir. Yumşaq, isti, rahat hiss.

### Nümunələr

- A **comfortable** chair.
- **Comfortable** clothes.
- Make yourself **comfortable**.
- **Comfortable** temperature.
- I feel **comfortable** here.
- A **comfortable** bed.

### Emosional (bəzi hallarda)

- **Comfortable** with the idea.
- **Comfortable** speaking English.
- I'm not **comfortable** with this decision.

### Tech kontekst

- **Comfortable** with the codebase. ✓
- **Comfortable** with Python. ✓

---

## 2. Convenient — Əlverişli / Uyğun

Vaxt, məkan, proses praktikdir. Asan.

### Nümunələr

- A **convenient** time.
- A **convenient** location.
- **Convenient** for everyone.
- Very **convenient** app.
- **Convenience** store (24/7 shop).
- What time is **convenient** for you?

### Sinonim

- suitable
- handy
- practical
- easy

---

## 3. Kritik Fərq

### Cümlə 1 — Meeting time

- "2 PM is **comfortable**." ✗ (məna vermir — ofis rahat olmaz)
- "2 PM is **convenient**." ✓ (məna: uyğundur)

### Cümlə 2 — Chair

- "This chair is **comfortable**." ✓ (fiziki rahat)
- "This chair is **convenient**." ✗ (qərib səslənir)

### Cümlə 3 — Location

- "The office is **comfortable**." → binanın içi rahat (fiziki)
- "The office is **convenient**." → məkanın yeri əlverişli (metroya yaxın)

---

## Test

Convenient / Comfortable?

1. Is 10 AM ______ for you? (vaxt uyğun)
2. This sofa is so ______ ! (fiziki rahat)
3. The app is ______ — saves time. (praktik)
4. I'm ______ talking to clients now. (emosional/rahat hiss)
5. A ______ location near the metro. (əlverişli yer)
6. These shoes aren't ______ . (fiziki rahat deyil)

**Cavablar:**
1. convenient
2. comfortable
3. convenient
4. comfortable
5. convenient
6. comfortable

## Tech / İş Kontekstində

### Convenient

- "When is **convenient** for our sync?" ✓
- A **convenient** API design. ✓ (istifadə asandır)
- **Convenient** for debugging. ✓
- Docker is **convenient** for dev setups. ✓

### Comfortable

- "I'm **comfortable** with microservices." ✓ (bilirəm, rahat hiss edirəm)
- "Are you **comfortable** presenting?" ✓
- "Get **comfortable** with Git first." ✓

---

## İnterview Kontekstində

### "Are you comfortable with X?"

Çox istifadə olunur — "Tanışsan? / Rahatsan?"

- "Are you **comfortable** with Python?" = Pythonu yaxşı bilirsən?
- "I'm **comfortable** with distributed systems." ✓
- "I'm **comfortable** speaking to stakeholders." ✓

### Meeting request

- "Let me know a **convenient** time." ✓
- "When is most **convenient**?" ✓

---

## Azərbaycanlı Səhvləri

- ✗ What time is comfortable for you? (vaxt → convenient)
- ✓ What time is **convenient** for you?

- ✗ This chair is convenient. (fiziki → comfortable)
- ✓ This chair is **comfortable**.

- ✗ I'm convenient with this decision. (emosional → comfortable)
- ✓ I'm **comfortable** with this decision.

---

## Related Sözlər

### Convenience (isim)

- At your **convenience** = sənə uyğun olduqda
- For your **convenience** = rahatlığın üçün

### Convenience store

- 24/7 kiçik mağaza (süpermarketdən fərqli)

### Comfort (isim)

- **Comfort** zone = rahat zona
- Step out of your **comfort zone**.

### Comfortably (zərf)

- **Comfortably** seated.

### Conveniently (zərf)

- **Conveniently** located.

---

## Comfortable vs Cozy

- **comfortable** = rahat (ümumi)
- **cozy** = isti, kiçik və rahat (specific — cafe, room)

---

## Xülasə

| Kontekst | Söz |
|----------|-----|
| Vaxt uyğunluğu | **convenient** |
| Məkan əlverişliliyi | **convenient** |
| İstifadə asanlığı | **convenient** |
| Fiziki (chair, bed) | **comfortable** |
| Paltar | **comfortable** |
| Emosional rahatlıq | **comfortable** |
| "Are you ___ with X?" | **comfortable** |

---

## Xatırlatma

- **comfortable** = rahat (hiss)
- **convenient** = əlverişli (praktik)

**Test:** Vaxt / məkan? → convenient. Chair / paltar / hiss? → comfortable.
