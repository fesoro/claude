# Anybody / Someone / Nobody / Everybody — "-body" və "-one" Sözləri

## Səviyyə
A2-B1

---

## Əsas Cədvəl

| -one forma | -body forma | Məna |
|------------|-------------|------|
| **someone** | **somebody** | kimsə (təsdiqdə) |
| **anyone** | **anybody** | hər kim / heç kim (sualda, inkarda) |
| **no one** | **nobody** | heç kim |
| **everyone** | **everybody** | hər kəs |

Eyni: yer üçün (somewhere, anywhere, nowhere, everywhere) və şey üçün (something, anything, nothing, everything).

---

## 1. -one vs -body

Demək olar **eyni mənada**!

- **someone** = **somebody** ✓
- **anyone** = **anybody** ✓
- **no one** = **nobody** ✓
- **everyone** = **everybody** ✓

### Fərqlər (çox kiçik)

- **-one** = daha rəsmi (yazılı)
- **-body** = daha informal (gündəlik)

**Tech yazı / email**: **-one** daha təbii.
**Gündəlik chat**: **-body** normal.

---

## 2. Some + one/body — Təsdiqdə

Pozitiv cümlələrdə. "Kimsə".

### Nümunələr

- **Someone** called.
- **Somebody** knocked.
- I met **someone** at the event.
- **Somebody's** laptop is here.

### Struktur

- Təsdiq cümləsi.
- Təkliflər (polite request).

### Təkliflərdə

- Would **someone** help me?
- Can **someone** explain?

---

## 3. Any + one/body — Sualda, İnkarda

Suallar və inkar. "Hər kim" / "heç kim".

### Suallar

- Is **anyone** here?
- Did **anybody** call?
- Does **anyone** know?

### İnkar

- I don't see **anyone**.
- There's not **anybody** home.

### Təsdiqdə — "hər kim"

- **Anyone** can learn to code. (pozitiv, ama məna: hər kim)
- **Anybody** is welcome.

### Some vs Any

- **some** = specific (təsdiqdə)
- **any** = istənilən (sualda / inkarda)

- "**Someone** called." (bir spesifik adam)
- "Did **anyone** call?" (hər kim)

---

## 4. No + one/body — İnkar

"Heç kim". Cümlə özü inkar olmur.

### Nümunələr

- **No one** came.
- **Nobody** knows.
- **No one** is perfect.

### Struktur — cümlə **pozitiv**, söz **inkar**!

- ✗ Nobody doesn't know. (ikili inkar!)
- ✓ **Nobody knows**. (pozitiv fel + nobody)

### Fel forma

**No one / nobody** həmişə **singular** fel alır:
- **No one** **is** here.
- **Nobody knows**.
- ~~No one are~~ ✗

---

## 5. Every + one/body — Hər Kəs

Bütün insanlar.

### Nümunələr

- **Everyone** is invited.
- **Everybody** agrees.
- **Everyone** has a laptop.

### Fel forma

**Everyone / everybody** həmişə **singular** fel!
- **Everyone is** here.
- **Everybody knows**.
- ~~Everyone are~~ ✗

### "Every + person"

- **Every** person = everyone.

---

## Pronoun Agreement (Kritik!)

### Eskik qayda: "he"

- Everyone brought **his** laptop. (köhnə)

### Modern qayda: "they" (singular they!)

- Everyone brought **their** laptop. ✓ (modern, inclusive)

Modern English-də "they" singular istifadəsi qəbul edilib:
- Someone left **their** phone.
- If anyone wants to help, **they** can.

### Someone / Somebody

- "**Someone** broke the build." ✓
- "Can **someone** review this?" ✓

### Anyone / Anybody

- "Does **anyone** have access?" ✓
- "Is **anybody** on-call?" ✓
- "**Anyone** can contribute." ✓

### No one / Nobody

- "**No one** has done this yet." ✓
- "**Nobody** responded to the PR." ✓

### Everyone / Everybody

- "**Everyone** approved the design." ✓
- "**Everybody** is on the call." ✓

### Polite request

- "Can **someone** help with...?" ✓
- "Does **anyone** have context on...?" ✓

### Announcements

- "**Everyone**, the deploy is complete." ✓
- "Attention **everyone**!" ✓

### Negative announcement

- "**No one** should push to main." ✓

---

## Azərbaycanlı Səhvləri

- ✗ Somebody don't know. (ikili inkar)
- ✓ **Nobody knows**. / **Somebody doesn't know**. (amma bu qəribədir)

- ✗ Everyone are happy. (singular fel!)
- ✓ **Everyone is** happy.

- ✗ Nobody don't came. (ikili inkar)
- ✓ **Nobody came**.

- ✗ Someone brought his or her laptop. (modern → their)
- ✓ **Someone** brought **their** laptop.

---

## "Anyone" Güclü İstifadə

Təsdiq cümləsində "anyone" = "hər kim, istisna yoxdur".

### Nümunələr

- **Anyone** can do it. (hər kim, hər kəs)
- **Anybody** is welcome.
- Call me **anytime**.

### Vurğu

- You can ask **anyone**. (istənilən)
- **Anyone** who works hard succeeds.

---

## Phrasal / İfadələr

### -one ilə

- someone special
- anyone but you
- everyone's a critic

### -body ilə

- nobody's business
- somebody famous
- everybody's talking

### Modern Singular "They"

- "Somebody forgot **their** keys." ✓ (modern)
- "Somebody forgot **his or her** keys." (köhnə, awkward)

---

## Fel Uyğunluğu (Agreement)

| Pronoun | Fel |
|---------|-----|
| someone | is / was / has |
| anyone | is / was / has |
| no one | is / was / has |
| everyone | is / was / has |
| everybody | is / was / has |

**Hamısı singular!**

Nümunə:
- Everybody **is** excited. ✓
- Somebody **has** called. ✓

## Xatırlatma

| Kontekst | İnsan | Şey | Yer |
|----------|-------|-----|-----|
| Təsdiq | someone/body | something | somewhere |
| Sual / İnkar | anyone/body | anything | anywhere |
| Yalnız inkar | no one/nobody | nothing | nowhere |
| Hamı | everyone/body | everything | everywhere |

**Qızıl qayda:**
- Bütün pronoun-lar → **singular fel**.
- Modern → **their** istifadə (singular they).
- -body = informal, -one = formal (çox kiçik fərq).
