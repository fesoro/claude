# Sensible vs Sensitive — Ağıllı vs Həssas

## Səviyyə
B1 (Azərbaycanlılar üçün TOP FALSE FRIEND!)

---

## Niyə Tələ?

Azərbaycanca "sensitiv" = həssas. Amma İngilis dilində:
- **sensitive** = həssas (Azərbaycanca kimi)
- **sensible** = **ağıllı / məntiqli** ("sensibl" = "sensitive" deyil!)

---

## Əsas Fərq

| Söz | Məna |
|-----|------|
| **sensible** | ağıllı, məntiqli, praktik |
| **sensitive** | həssas (emosional / fiziki) |

> **Qısa qayda:**
> - **sensible** = "sens" (məna, ağıl) → ağıllı
> - **sensitive** = "sense" (hiss) → həssas

---

## 1. Sensible — Ağıllı / Praktik

Məntiqli qərar verən, praktik.

### Nümunələr

- That's a **sensible** decision.
- Be **sensible**!
- A **sensible** approach.
- **Sensible** advice.
- **Sensible** shoes. (rahat, məntiqli)
- She's a very **sensible** person.

### Kontekst

- qərar vermə
- davranış
- geyim (rahat, uyğun)
- məsləhət

### Sinonim

- reasonable
- practical
- wise

---

## 2. Sensitive — Həssas

İki əsas məna:

### A. Emosional həssas

- He's very **sensitive** — be careful.
- **Sensitive** topic.
- Don't be too **sensitive**.
- **Sensitive** feelings.

### B. Fiziki həssas

- **Sensitive** skin.
- **Sensitive** to light.
- **Sensitive** sensor.
- **Sensitive** data!

### Tech kontekst — ÇOX VACİB!

- **Sensitive** data = həssas məlumat (password, credit card)
- **Sensitive** information
- GDPR / data protection

~~Sensible data~~ səhvdir! **Sensitive data** ✓

---

## 3. Kritik Fərq

### Cümlə 1

- "She's very **sensible**."
  → Ağıllı, məntiqli qərar verir.

- "She's very **sensitive**."
  → Həssasdır, asan incikir.

Tam fərqli!

### Cümlə 2

- "A **sensible** choice."
  → Məntiqli seçim.

- "A **sensitive** choice."
  → (çox nadir) Həssas / ehtiyatlı seçim.

## Tech / İş Kontekstində

### Sensible (ağıllı)

- A **sensible** architectural decision. ✓
- **Sensible** defaults in the config. ✓
- **Sensible** approach to testing. ✓

### Sensitive (həssas)

- **Sensitive** customer data. ✓ (GDPR!)
- **Sensitive** information. ✓
- **Sensitive** to performance issues. ✓

## Azərbaycanlı Səhvləri (TOP!)

- ✗ He's a sensitive engineer. (demək istəyir "ağıllı mühəndis")
- ✓ He's a **sensible** engineer.

- ✗ Sensible data must be encrypted.
- ✓ **Sensitive** data must be encrypted.

- ✗ Be sensitive with money. (ağıllı → sensible)
- ✓ Be **sensible** with money.

## Related Sözlər

### Sensibility

İsim — ağıllılıq / həssaslıq (bu söz iki mənada!).

### Sensibly (zərf)

- Act **sensibly**! = Ağıllı davran!

### Sensitively (zərf)

- Handle it **sensitively**. = Həssaslıqla davran.

### Nonsense

- **Nonsense** = mənasızlıq (sens-siz)
