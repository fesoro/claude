## Əsas Cədvəl

| Söz | Kontekst |
|-----|----------|
| **tool** | sadə alət (fiziki + abstract) |
| **instrument** | dəqiq alət (music, science) |
| **equipment** | avadanlıq (uncountable!) |
| **device** | cihaz (elektronik) |

> **Qısa qayda:**
> - **tool** = alət (generic)
> - **instrument** = dəqiq alət (science, music)
> - **equipment** = bütöv avadanlıq (uncountable)
> - **device** = elektronik cihaz
> - **gadget** = kiçik elektronik (informal)
> - **appliance** = məişət (yemək bişirən)

## 1. Tool — Alət (Generic)

Ən çox istifadə. Fiziki + abstract.

### Nümunələr

- A **tool** for the job.
- Gardening **tools**.
- Right **tool**.
- **Tool** box.

### Abstract tools

- Communication **tools**.
- Analysis **tools**.
- Programming **tools**.

### Tech kontekst — ÇOX!

- Development **tools**.
- Debugging **tools**.
- DevOps **tools**.
- AI **tools**.

### İdiom

- **the right tool for the job**
- **tool of the trade** = peşə aləti

---

## 2. Instrument — Dəqiq Alət

Musiqi və elm. Dəqiq / detallı.

### Music

- **Musical instrument**.
- String **instrument**.
- **Instrument**al music.

### Science / tıbbi

- Scientific **instrument**.
- Medical **instrument**.
- Measuring **instrument**.

### Tool vs Instrument

- **tool** = ümumi
- **instrument** = dəqiq / elmi / musiqi

- "A **tool** for measuring." (ümumi)
- "An **instrument** for measuring." (dəqiq elmi)

**Tech-də "instrument" az:** "Observability instrumentation" — kod içi metric.

---

## 3. Equipment — Avadanlıq (Uncountable!)

Bütöv set. **Həmişə uncountable**.

### Nümunələr

- Office **equipment**.
- Sports **equipment**.
- Medical **equipment**.
- IT **equipment**.

### Uncountable!

- ✗ an equipment
- ✗ equipments
- ✓ **equipment** (singular)
- ✓ **a piece of equipment**
- ✓ **some equipment**

### Tool vs Equipment

- **tool** = tək alət (can count)
- **equipment** = bütöv set (uncountable)

- "A **tool**." ✓
- "3 **tools**." ✓
- "Some **equipment**." ✓
- ~~3 equipments~~ ✗

### Tech

- IT **equipment**.
- Network **equipment**.
- Lab **equipment**.
- Server **equipment**.

---

## 4. Device — Elektronik Cihaz

Elektronik / texnoloji. Adətən smart.

### Nümunələr

- Mobile **device**.
- Smart **device**.
- Medical **device**.
- Input **device**.

### Tech kontekst

- **Device** drivers.
- IoT **devices**.
- Wearable **devices**.
- **Device** management.

### Tool vs Device

- **tool** = istənilən alət
- **device** = elektronik (adətən)

- A hammer = **tool**.
- A phone = **device**.

---

## 5. Gadget — Kiçik Elektronik (Informal)

İnformal. Kiçik, smart, yeni.

### Nümunələr

- Cool **gadgets**.
- Latest tech **gadgets**.
- **Gadget** lover.

### Device vs Gadget

- **device** = rəsmi
- **gadget** = informal, kiçik

- "Medical **device**." (rəsmi)
- "Cool **gadget**." (informal, yeni)

### Kontekst

- Consumer electronics
- Geeky stuff
- Smartwatches, earbuds

---

## 6. Appliance — Məişət Cihazı

Mətbəx, evdə.

### Nümunələr

- Kitchen **appliances**.
- Home **appliances**.
- Electrical **appliances**.

### Common appliances

- Fridge, stove, microwave
- Washing machine
- Dishwasher
- Vacuum cleaner
- Air conditioner

### Device vs Appliance

- **device** = elektronik (phone, tablet)
- **appliance** = məişət (fridge, oven)

---

## 7. Machine — Maşın

Böyük / kompleks. Mexaniki.

### Nümunələr

- Washing **machine**.
- Coffee **machine**.
- Sewing **machine**.
- Vending **machine**.

### Tech

- Virtual **machine** (VM).
- State **machine**.
- **Machine** learning (ML).

### Machine vs Device

- **machine** = böyük, mexaniki
- **device** = kiçik, elektronik

- A car = **machine**.
- A phone = **device**.

Overlap var amma ümumi qayda.

6. Violin is a string ______ . (musiqi)

4. gadgets
