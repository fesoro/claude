# Since / Because / As / For — Səbəb Bağlayıcıları

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Tonu | Güc |
|-----|------|-----|
| **because** | gündəlik, neytral | güclü səbəb |
| **since** | rəsmi + zaman mənası | zəif səbəb |
| **as** | rəsmi | zəif səbəb |
| **for** | rəsmi / ədəbi | zəif |

> **Qısa qayda:**
> - **because** = güclü səbəb (main reason)
> - **since** = ikinci dərəcəli + zaman (həm də)
> - **as** = rəsmi arka plan səbəb
> - **for** = ədəbi, nadir

---

## 1. Because — Güclü Səbəb (Main Reason)

Ən çox istifadə olunur. Əsl səbəb.

### Struktur

- clause, **because** + clause
- **Because** + clause, main clause

### Nümunələr

- I'm tired **because** I worked late.
- **Because** it's late, I'll go home.
- The test failed **because** of a bug.

### Because vs Because of

- **because** + clause (subject + verb)
- **because of** + noun/N

- I'm late **because** the bus broke down.
- I'm late **because of** the bus.

### Sual: Niyə?

**Because** ən birbaşa cavabdır:
- Why? **Because** it was broken.

---

## 2. Since — İkinci Dərəcəli Səbəb + Zaman

İki məna:

### A. Səbəb (zəif)

- **Since** you're here, let's start.
- **Since** it's late, we'll stop.
- **Since** tests pass, we can deploy.

### B. Zaman (from + time)

- I've worked here **since** 2020.
- **Since** yesterday.
- **Since** then.

→ Zaman mənası: [for-since-ago.md](for-since-ago.md)

### Because vs Since

- **because** = main səbəb (cavab fokusda)
- **since** = məlum səbəb (background, daha rəsmi)

- "I'm late **because** of traffic." (əsas səbəb)
- "**Since** you're already here, let's start." (məlum fakt)

**Diqqət:** "Since" cümlənin əvvəlində daha təbii.

---

## 3. As — Rəsmi Background Səbəb

"Since"-ə çox yaxın. Background information.

### Nümunələr

- **As** it's raining, we'll stay inside.
- **As** you know, the deadline is tight.
- **As** per the requirement...

### Because vs As

- **because** = güclü səbəb
- **as** = zəif, məlum

- "I went home **because** I was tired." (əsas)
- "**As** I was tired, I went home." (background)

### Tech yazıda

- **As** the system scales, performance matters.
- **As** we discussed, let's proceed.

---

## 4. For — Ədəbi / Rəsmi

Çox rəsmi, nadir. Ədəbi yazı.

### Nümunələr

- He went home, **for** he was tired.
- I can't come, **for** I'm busy.

### Struktur

- clause, **for** + clause (comma vacib!)

### İstifadə

- Müasir danışıqda çox nadir
- Köhnə ingilis / ədəbi
- **Because** daha təbii

---

## Kritik Fərq

### Cümlə 1

- "I'm happy **because** I got a promotion." (fokus: səbəb)
- "**Since** I got a promotion, I'm happy." (fokus: nəticə)
- "**As** I got a promotion, I'm happy." (fakt + nəticə)

### Cümlə 2 — Since'un iki mənası

- "**Since** you're here, let's talk." (because you're here — səbəb)
- "I've known you **since** 2010." (from 2010 — zaman)

Kontekst göstərir!

## Test

Hansı uyğundur?

1. I left early ______ of the meeting. (preposition)
2. ______ you've finished, let's review. (arka plan)
3. I failed ______ I didn't study. (əsas səbəb)
4. He's been sick ______ Monday. (zaman!)
5. ______ the logs show errors, we need to investigate. (rəsmi background)
6. I'll drive ______ I have the car keys. (daily)

**Cavablar:**
1. because of
2. Since
3. because
4. since (time)
5. As / Since
6. because

---

## Tech / İş Kontekstində

### Because (strong reason)

- "We rolled back **because** tests failed." ✓
- "**Because of** the outage, we lost users." ✓

### Since (background)

- "**Since** the DB is slow, we added caching." ✓
- "**Since** you're the owner, please review." ✓

### As (formal)

- "**As** we discussed, let's prioritize." ✓
- "**As** the documentation states..." ✓

### For (ədəbi, az)

- Rare in tech. Avoid.

---

## Interview Kontekstində

### Explaining

- "I chose this role **because** it aligns with my goals." ✓
- "**Since** I have 5 years of experience, I'm confident." ✓

### Background

- "**As** a senior engineer, I'm expected to mentor." ✓
- "**Since** we're scaling fast, we need better tools." ✓

---

## Punctuation

### Cümlə ortasında

- "I left **because** it was late." (no comma before because — usually)
- "I left, **since** it was late." (comma optional)

### Cümlə başında

- "**Because** it was late, I left."
- "**Since** it was late, I left." ✓ (more common)
- "**As** it was late, I left." ✓

---

## Azərbaycanlı Səhvləri

- ✗ Because of I'm tired. (of + N, because + clause)
- ✓ **Because** I'm tired.
- ✓ **Because of** my tiredness.

- ✗ Since 2 years I'm working. (for + müddət)
- ✓ I've been working here **for** 2 years.
- ✓ I've been working here **since** 2020.

- ✗ Because it's raining, so I'll stay. (because + so birgə səhv!)
- ✓ **Because** it's raining, I'll stay.
- ✓ It's raining, **so** I'll stay.
