# Confused / Puzzled / Perplexed / Baffled — Anlamamaq Sifətləri

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Dərəcə | Kontekst |
|-----|--------|----------|
| **confused** | anlamıram | daily |
| **puzzled** | qəribə / anlaşılmır | detektiv |
| **perplexed** | rəsmi anlamamaq | formal |
| **baffled** | tamamilə anlamaz | güclü |
| **stumped** | tapmaq olmur (informal) | quiz |
| **bewildered** | hiyrət / anlamamaq | ədəbi |

> **Qısa qayda:**
> - **confused** = anlamıram (daily)
> - **puzzled** = qəribə (məsələ)
> - **perplexed** = rəsmi
> - **baffled** = tamamilə anlamaz
> - **stumped** = cavab yoxdur (informal)

---

## 1. Confused — Anlamıram (Daily)

Ən çox istifadə. Basic.

### Nümunələr

- I'm **confused**.
- This is **confusing**.
- Don't be **confused**.
- **Confusing** explanation.

### Struktur

- **confused** about + N
- **confused** by + N

- **Confused about** the feature.
- **Confused by** the error.

### Tech kontekst

- "The docs are **confusing**." ✓
- "I'm **confused** about the API." ✓

---

## 2. Puzzled — Qəribə / Məntiqli Deyil

Detektiv-tərzli. "Mənasız görünür."

### Nümunələr

- I'm **puzzled** by this result.
- **Puzzling** outcome.
- His answer **puzzles** me.

### Confused vs Puzzled

- **confused** = anlamıram
- **puzzled** = qəribə hiss (puzzle = tapmaca)

- "I'm **confused** about the docs." (oxudum, başa düşmədim)
- "I'm **puzzled** by this bug." (davamlı qərib)

### Tech

- "**Puzzling** error." ✓ (nə baş verir?)
- "**Puzzled** by the logs." ✓

---

## 3. Perplexed — Rəsmi Anlamamaq

Rəsmi / ədəbi. Güclü confusion.

### Nümunələr

- I'm **perplexed** by the decision.
- **Perplexed** look.
- Totally **perplexed**.

### Confused vs Perplexed

- **confused** = daily
- **perplexed** = rəsmi, daha güclü

- "I'm **confused**." (normal)
- "I'm **perplexed**." (rəsmi + güclü)

Gündəlikdə perplexed nadir.

---

## 4. Baffled — Tamamilə Anlamaz

Çox güclü. Heç anlamıram.

### Nümunələr

- Scientists are **baffled**.
- Completely **baffled**.
- **Baffling** mystery.

### Confused vs Baffled

- **confused** = anlamıram
- **baffled** = heç başa düşə bilmirəm

- "**Confused** about X." (bəlkə həll var)
- "**Baffled** by Y." (heç yox)

### Tech kontekst

- "The team was **baffled** by the intermittent issue." ✓
- "**Baffling** performance problem." ✓

---

## 5. Stumped — Cavab Yoxdur (Informal)

Quiz / sual kontekstdə. İnformal.

### Nümunələr

- I'm **stumped**.
- **Stumped** by the question.
- That **stumped** me.

### Struktur

- be **stumped** by + N
- You've got me **stumped**.

### Stumped vs Baffled

- **baffled** = daily + formal
- **stumped** = informal, specific

- Interview question → **stumped**.
- Ongoing bug → **baffled**.

---

## 6. Bewildered — Hiyrət + Anlamamaq

Ədəbi. Emotional.

### Nümunələr

- **Bewildered** expression.
- I was **bewildered**.
- **Bewildering** maze.

Gündəlikdə nadir. Kitablar / film.

---

## İntensivlik Skalası

```
curious < confused < puzzled < perplexed < baffled < stumped ≈ mystified
  sorğu      daily     qərib    formal     güclü      nothing
```

## Fel Formaları

| Sifət | Fel |
|-------|-----|
| confused | to confuse |
| puzzled | to puzzle |
| baffled | to baffle |
| perplexed | to perplex |

### Nümunələr

- "The docs **confuse** me." (məni qarışdırır)
- "This bug **puzzles** the team." (qarışdırır)
- "The issue **baffled** everyone." (heç kim başa düşmədi)

---

## Azərbaycanlı Səhvləri

- ✗ I'm confusing. (Fərqli məna! = "mən qarışdıran insanam")
- ✓ I'm **confused**. (mən qarışıqam)
- ✓ The situation is **confusing**. (situasiya qarışıqdır)

**Qayda:** -ed / -ing fərqi!
- **-ed** = how you feel (I'm confused)
- **-ing** = the cause (it's confusing)

→ [ed-vs-ing-adjectives.md](ed-vs-ing-adjectives.md)
