# Famous / Well-known / Popular / Renowned — Tanınma Sifətləri

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Kimin tərəfindən? | Tonu |
|-----|--------------------|------|
| **famous** | çox-çox insan bilir | ümumi tanınma |
| **well-known** | müəyyən dairədə | peşəkar / rəsmi |
| **popular** | sevilir, istifadə olunur | pozitiv |
| **renowned** | hörmətli, dəyərli | rəsmi, tərif |
| **notorious** | pis üçün tanınmış | neqativ! |
| **celebrated** | bəyənilən, qeyd olunan | ədəbi |

> **Qısa qayda:**
> - **famous** = milyonlar bilir (actor, singer)
> - **well-known** = sahəsində tanınır
> - **popular** = sevilir, çox istifadə olunur
> - **renowned** = hörmətli (pozitiv!)
> - **notorious** = pis üçün tanınır (NEGATIV!)

---

## 1. Famous — Milyonlar Bilir

Geniş tanınma. Dünya / ölkə səviyyəsi.

### Nümunələr

- A **famous** actor.
- The **famous** pyramid.
- **Famous** for his speeches.
- World-**famous** brand.

### Struktur

- **famous** for + N/V-ing
- **famous** as + N

- He's **famous for** his books.
- She's **famous as** a painter.

---

## 2. Well-known — Sahəsində Tanınır

Müəyyən dairədə / community-də tanınır. "Famous"-dan kiçik.

### Nümunələr

- A **well-known** expert.
- **Well-known** developer in the community.
- **Well-known** framework.
- **Well-known** issue.

### Famous vs Well-known

- **famous** = ümumi kütlə bilir
- **well-known** = sahədə tanınır

- Tom Cruise → **famous**
- Linus Torvalds → **well-known** (tech community-də famous)

### Tech kontekst

- A **well-known** bug. ✓
- A **well-known** pattern. ✓
- A **well-known** library. ✓

---

## 3. Popular — Sevilir / Çox İstifadə Olunur

Çox insan sevir və ya istifadə edir.

### Nümunələr

- A **popular** movie.
- **Popular** framework.
- React is **popular**.
- **Popular** among developers.
- **Popular** vote.

### Famous vs Popular

- **famous** = çoxu bilir (bilməsi vacibdir, sevir olmaya bilər)
- **popular** = çoxu sevir / istifadə edir

- Hitler is **famous**, NOT **popular**.
- Beyoncé is both **famous** and **popular**.

### Tech

- **Popular** among startups.
- Most **popular** language.
- **Popular** library (npm downloads çox).

---

## 4. Renowned — Hörmətli / Tərif

Rəsmi, ədəbi. Həmişə pozitiv.

### Nümunələr

- World-**renowned** scientist.
- **Renowned** architect.
- **Renowned** for excellence.
- Internationally **renowned**.

### Struktur

- **renowned** for + N

- She's **renowned** for her research.

### Well-known vs Renowned

- **well-known** = tanınır (neytral)
- **renowned** = tərifli tanınır (pozitiv)

---

## 5. Notorious — Pis Üçün Tanınır! (NEGATIV)

**Diqqət:** "famous"-ın pis versiyası. Asla kompliment kimi istifadə etmə!

### Nümunələr

- **Notorious** criminal.
- **Notorious** for bugs.
- The **notorious** bad code.
- **Notorious** boss (çətin rəis).

### Struktur

- **notorious** for + N/V-ing

- This library is **notorious for** memory leaks.

### Tech kontekst

- **Notorious** for breaking things. ✓ (pis)
- A **notorious** anti-pattern. ✓ (pis)
- **Notorious** race condition. ✓ (çətin bug)

---

## 6. Celebrated — Bəyənilən / Qeyd Olunan

Ədəbi, rəsmi. "Renowned" kimi pozitiv.

### Nümunələr

- **Celebrated** author.
- **Celebrated** for her work.
- A **celebrated** musician.

**B1-də nadir işlənir — renowned-a üstünlük ver.**

## Test

Hansı söz uyğundur?

1. Tom Cruise is a ______ actor. (dünya)
2. Linus Torvalds is ______ in the tech community. (dairə)
3. React is a ______ framework. (sevilir / istifadə)
4. This library is ______ for memory leaks. (pis)
5. A world-______ scientist. (hörmətli)
6. The ______ N+1 query problem. (sahədə tanınır)

**Cavablar:**
1. famous
2. well-known
3. popular
4. notorious
5. renowned
6. well-known

## Tech / İş Kontekstində

### Famous (rarely used in tech)

- The **famous** "It works on my machine" quote. ✓

### Well-known

- A **well-known** design pattern. ✓
- **Well-known** port (HTTP 80, HTTPS 443). ✓
- **Well-known** issue. ✓

### Popular

- **Popular** open-source project. ✓
- **Popular** among developers. ✓

### Renowned

- **Renowned** architect. ✓
- A **renowned** CTO. ✓

### Notorious

- **Notorious** for performance issues. ✓
- The **notorious** leftpad incident. ✓

---

## Interview-da Kompliment

- ✓ "He's a **well-known** engineer in the community."
- ✓ "She's **renowned** for her architectural skills."
- ✓ "A **popular** framework we used was..."
- ✗ "The **notorious** Senior Engineer." (təhqir!)

---

## Azərbaycanlı Səhvləri

- ✗ React is famous. (sevilir → popular)
- ✓ React is **popular**.

- ✗ This senior is notorious for his skills. (pis → əksi!)
- ✓ This senior is **renowned** / **well-known** for his skills.

---

## Related Sözlər

### Infamous

- **infamous** = notorious kimi (pis üçün tanınır)
- The **infamous** bug.

### Legendary

- **legendary** = əfsanəvi (pozitiv)
- A **legendary** engineer.

### Prominent

- **prominent** = görkəmli
- A **prominent** figure.

### Eminent

- **eminent** = seçkin (elmi / rəsmi)
- **Eminent** scholar.
