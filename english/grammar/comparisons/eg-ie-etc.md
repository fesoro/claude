# e.g. / i.e. / etc. / viz. — Latın Qısaltmaları

## Səviyyə
B1 (yazıda çox vacib!)

---

## Əsas Cədvəl

| Qısaltma | Açılışı (Latın) | Məna (Azərbaycanca) |
|----------|------------------|----------------------|
| **e.g.** | *exempli gratia* | məsələn (for example) |
| **i.e.** | *id est* | yəni (that is / in other words) |
| **etc.** | *et cetera* | və s. (and so on) |
| **viz.** | *videlicet* | yəni / namely (rəsmi) |

> **Qısa qayda:**
> - **e.g.** = **e**xample
> - **i.e.** = **i**n other words
> - **etc.** = and so on

---

## 1. e.g. — Məsələn (Nümunə)

"For example" əvəzinə. Nümunə verərkən.

### Nümunələr

- Use modern frameworks, **e.g.** React, Vue, Angular.
- Backend languages, **e.g.** Python, Go, Java.
- Databases, **e.g.** Postgres, MySQL.
- Include tests, **e.g.** unit tests, integration tests.

### Xatırlatma trick

**e.g.** = **e**xample given.

---

## 2. i.e. — Yəni (İzah)

"In other words" / "that is" əvəzinə. İzah / dəqiqləşdirmə.

### Nümunələr

- Use a REST API, **i.e.** HTTP-based stateless service.
- Use ORMs, **i.e.** object-relational mappers like Prisma.
- The deploy target, **i.e.** production, is ready.
- He's a polyglot, **i.e.** he speaks many languages.

### Xatırlatma trick

**i.e.** = **i**n other words, **i**t means.

---

## 3. Kritik Fərq: e.g. vs i.e.

### e.g. — nümunə (siyahı tam deyil)

- "Bring snacks, **e.g.** chips, cookies, fruits."
  → Bir neçə nümunə; başqa snack-lər də ola bilər.

### i.e. — izah / dəqiqləşmə

- "Bring snacks, **i.e.** chips and cookies."
  → Tam olaraq bunları bring (dəqiq siyahı).

### Test metodu

- "for example" yerinə qoy → məna verir → **e.g.**
- "in other words" yerinə qoy → məna verir → **i.e.**

---

## 4. etc. — və s. (Davamı Var)

Siyahının davamı olduğunu göstərir.

### Nümunələr

- Apples, oranges, bananas, **etc.**
- Git commands: push, pull, commit, **etc.**
- Languages: Python, Go, Rust, **etc.**

### Diqqət — etc. istifadə qaydaları

- ✓ Birkaç element göstər, sonra **etc.**
- ✗ **etc.** tək istifadə etmə.
- ✗ "e.g." ilə "etc." birgə istifadə etmə:
  - ✗ "e.g. Python, Go, etc."
  - ✓ "e.g. Python, Go" ya da "Python, Go, etc."
  ("e.g." zatən "nümunə" deməkdir)

---

## 5. Yazı Qaydaları

### Nöqtələr

- **e.g.** (hər hərfdən sonra nöqtə)
- **i.e.** (hər hərfdən sonra nöqtə)
- **etc.** (sonunda nöqtə)

### Vergül

American style:
- "Languages, **e.g.,** Python, Go."
- "A language, **i.e.,** Python."

British style:
- "Languages, **e.g.** Python, Go."
- "A language, **i.e.** Python."

### Cümlənin əvvəlində

**Yox!** Cümlə başlanğıcında qısaltma istifadə etmə — tam söz yaz:
- ✗ "e.g. Python..."
- ✓ "For example, Python..."

### Mötərizədə

- "Use a language (**e.g.**, Python) for scripting."
- "Use a language (**i.e.**, Python) only."

---

## 6. viz. — Yəni / Namely (Rəsmi)

"Namely" əvəzinə. Çox rəsmi, nadir.

### Nümunələr

- Three cities, **viz.** London, Paris, Rome.

Modern yazıda nadir. **B1-də bilsən kifayətdir.**

---

## Test

e.g. / i.e. / etc.?

1. Bring pets, ______ cats, dogs, hamsters. (siyahı davam edə bilər)
2. The CEO, ______ the head of the company, decides.
3. I use common languages, ______ Python, Go, Java. (nümunə)
4. Coffee, tea, juice, ______ . (siyahı davam)
5. I'll use a JVM language, ______ Kotlin. (dəqiq izah)
6. Bring tools, ______ a screwdriver, a hammer. (nümunə)

**Cavablar:**
1. e.g.
2. i.e.
3. e.g.
4. etc.
5. i.e.
6. e.g.

## Tech / İş Kontekstində

### e.g.

- "Use a cache, **e.g.**, Redis or Memcached." ✓
- "Unit tests frameworks, **e.g.**, Jest, Mocha." ✓

### i.e.

- "Use a NoSQL database, **i.e.**, MongoDB." ✓ (tam dəqiq)
- "Deploy to prod, **i.e.**, the live environment." ✓

### etc.

- "Frontend: React, Vue, Angular, **etc.**" ✓
- "CI/CD tools: Jenkins, GitLab CI, GitHub Actions, **etc.**" ✓

## Slack / Email Etiquette

### Professional

- "Can you add the missing deps, **e.g.**, axios?" ✓
- "The main service, **i.e.**, auth, is down." ✓

### Informal

Gündəlik Slack-də tam formanı seç:
- "For example, we need..." ✓
- "In other words, this means..." ✓

---

## Tələffüz (Meeting-də)

Qısaltmalar oxunarkən:

- **e.g.** → "for example" oxu (latın adını demə)
- **i.e.** → "that is" oxu
- **etc.** → "et cetera" oxu

- "We need snacks, **for example**, chips." (e.g. yazılır, "for example" oxunur)

---

## Azərbaycanlı Səhvləri

- ✗ I like languages e.g. Python (no comma, no period)
- ✓ I like languages, **e.g.**, Python.

- ✗ Python, e.g. Django, etc. (e.g. + etc. birgə)
- ✓ Python, **e.g.**, Django.
- ✓ Python, Django, **etc.**

- ✗ "e.g." cümlə başında
- ✓ "For example, ..."

---

## Bonus: cf. / et al.

### cf. — compare

- "See Chapter 2, **cf.** p. 42."

### et al. — and others

- "Smith **et al.** (2020)" (akademik)

Tech yazıda az, amma bilsən faydalıdır.

## Xatırlatma

| Qısaltma | Məna | Hansı vaxt? |
|----------|------|-------------|
| **e.g.** | for example | nümunə (çox ola bilər) |
| **i.e.** | that is / namely | dəqiq izah |
| **etc.** | and so on | siyahı davam |
| **viz.** | namely (rəsmi) | nadir |

**Test:** "Nümunə?" → e.g. "Yəni?" → i.e. "Və s.?" → etc.
