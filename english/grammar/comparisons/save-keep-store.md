# Save / Keep / Store / Preserve — Saxlamaq Fellər

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Kontekst | Tonu |
|-----|----------|------|
| **save** | yaddaş / qənaət / qurtarmaq | ümumi |
| **keep** | əldə saxlamaq | daily |
| **store** | yerdə / bazada saxlamaq | physical / tech |
| **preserve** | pozulmamaq üçün saxlamaq | rəsmi |

> **Qısa qayda:**
> - **save** = yadda saxla / qənaət et / qurtar
> - **keep** = özündə saxla (davamlı)
> - **store** = yer / rəfdə / bazada qoy
> - **preserve** = keyfiyyəti qoru

---

## 1. Save — Çoxsaylı Məna

### A. Yadda saxlamaq (tech)

- **Save** the file.
- **Save** your work.
- **Save** to the cloud.
- Ctrl+S = **save**.

### B. Qənaət etmək

- **Save** money.
- **Save** time.
- **Save** energy.

### C. Qurtarmaq (xilas etmək)

- **Save** the day.
- **Save** lives.
- **Save** someone from danger.

### D. Kenara qoymaq

- **Save** some cake for me.
- **Save** a seat.

### Tech kontekst

- **Save** the state.
- **Save** to database.
- Auto-**save** feature.
- **Save** changes.

---

## 2. Keep — Özündə Saxlamaq (Davamlı)

### Nümunələr

- **Keep** the receipt.
- **Keep** it secret.
- **Keep** going!
- **Keep** the door closed.
- I'll **keep** this in mind.

### Struktur

- **keep** + N
- **keep** + adj (state)
- **keep** + V-ing (davam)

- **Keep** the file.
- **Keep** quiet.
- **Keep** trying.

### Phrasal

- **keep up** = qalmaq / davam et
- **keep on** = davam et
- **keep off** = uzaq qal
- **keep out** = içəri girmə

### Save vs Keep

- **save** = qəsdən saxla (action)
- **keep** = davamlı saxla (state)

- "**Save** the PDF to your drive." (action)
- "**Keep** the PDF on your drive." (davamlı)

---

## 3. Store — Yerdə / Bazada Saxlamaq

Konkret yerə qoymaq. Physical və ya digital.

### Nümunələr

- **Store** the data in the database.
- **Store** files in S3.
- **Store** the boxes in the garage.
- **Store** passwords securely.
- How do you **store** user sessions?

### Tech kontekst — ÇOX İŞLƏNİR!

- **Store** data.
- **Storage** (isim) = yaddaş.
- Cloud **storage**.
- Key-value **store**.
- Object **store**.

### İsim

- A **store** = mağaza (retail) / yaddaş

### Save vs Store

- **save** = əməliyyat (bu an)
- **store** = uzunmüddətli saxlama yeri

- "**Save** your changes." (Ctrl+S action)
- "We **store** data in PostgreSQL." (yer)

---

## 4. Preserve — Pozulmamaq Üçün Saxlamaq

Rəsmi / keyfiyyət saxlama. Qida, sənəd, mədəniyyət.

### Nümunələr

- **Preserve** food.
- **Preserve** culture.
- **Preserve** history.
- **Preserve** data integrity.

### Tech kontekst

- **Preserve** data integrity.
- **Preserve** formatting.
- **Preserve** order.
- Maintain and **preserve** state.

### Keep vs Preserve

- **keep** = ümumi, gündəlik
- **preserve** = keyfiyyət qoruma

- "**Keep** the files." (saxla)
- "**Preserve** the original quality." (pozma)

## Tech Kontekstində Hansı Hansı?

### Save

- **Save** a file (immediate action).
- Auto-**save** enabled.
- **Save** draft.

### Store

- **Store** in database.
- Key-value **store**.
- Persistent **storage**.

### Keep

- **Keep** a backup.
- **Keep** the logs for 30 days.

### Preserve

- **Preserve** order in the list.
- **Preserve** data integrity.
- **Preserve** whitespace.

## İfadələr

### Save

- **save the day** = günü xilas et
- **save for later** = sonra üçün saxla
- **save face** = hörmətini qoru

### Keep

- **keep in touch** = əlaqə saxla
- **keep a secret** = sir saxla
- **keep an eye on** = göz qoy
- **keep up with** = addım uyğunlaşdır
- **keep it up!** = davam et!

### Store

- **store up** = toplamaq
- **store away** = saxlamaq (kənar qoymaq)

## Azərbaycanlı Səhvləri

- ✗ I saved the file in the database. (baza → store)
- ✓ I **stored** the file in the database.

- ✗ Keep your work frequently. (tech save → save)
- ✓ **Save** your work frequently.

- ✗ Please save the original. (keyfiyyət → preserve)
- ✓ Please **preserve** the original.
