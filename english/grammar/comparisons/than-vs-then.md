# Than vs Then — Bir Hərf, Böyük Fərq

## Səviyyə
A2-B1 (yazıda ən çox səhv)

---

## Əsas Fərq

| Söz | Məna | Nə zaman? |
|-----|------|-----------|
| **than** | -dən, nisbətən | müqayisə |
| **then** | sonra, o zaman | zaman / ardıcıllıq |

> **Qısa qayda:**
> - **than** = compAr**a**tive → müqay**i**sə
> - **then** = thEn → tim**E** (zaman)

---

## 1. Than — Müqayisə

Həmişə **comparative sifət / zərf** ilə gəlir.

### Nümunələr

- She is taller **than** me.
- Python is easier **than** Java.
- This code is faster **than** the old one.
- I earn more **than** last year.
- He speaks better **than** her.
- Redis is faster **than** MySQL for caching.

### Comparative formalar

- bigger **than**
- more difficult **than**
- less expensive **than**
- better **than**
- worse **than**

### Rather than / Other than / Less than

- I'd rather walk **than** drive.
- Other **than** coffee, I don't drink much.
- Less **than** 10 users reported this.

---

## 2. Then — Sonra / O Zaman

### A. Zaman ardıcıllığı (sonra)

- First, clone the repo. **Then**, run npm install.
- I finished work, **then** I went home.
- We tested locally, **then** deployed to staging.

### B. O zaman / həmin vaxtda

- Back **then**, we used jQuery.
- I was a junior **then**.
- See you **then**!

### C. Bu halda (if...then)

- If it fails, **then** check the logs.
- If you're free, **then** let's meet.

### D. Ayrıca (by then, since then, from then on)

- By **then**, it was too late.
- Since **then**, performance improved.
- From **then** on, we used CI/CD.

---

## Test (than / then)

1. She's smarter _____ him.
2. First write tests, _____ refactor.
3. Back _____, JavaScript was simple.
4. This is bigger _____ that.
5. If it breaks, _____ roll back.
6. I'd rather code _____ attend meetings.

**Cavablar:** 1. than, 2. then, 3. then, 4. than, 5. then, 6. than

---

## Azərbaycanlı Səhvi

- ✗ He is taller **then** me.
- ✓ He is taller **than** me.

- ✗ First test, **than** deploy.
- ✓ First test, **then** deploy.

---

## Tez Yoxlama

Cümlədə "sonra / o zaman" varsa → **then**
"-dən, nisbətən" varsa → **than**

---

## Xatırlatma

**th**An → comp**A**re
**th**En → tim**E**
