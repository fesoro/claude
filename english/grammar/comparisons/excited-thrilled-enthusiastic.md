# Excited / Thrilled / Enthusiastic / Eager — Həyəcan və Marağın

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Dərəcə | Kontekst |
|-----|--------|----------|
| **interested** | maraqlıdır | ümumi |
| **keen** | maraqlı (British) | informal |
| **eager** | istəkli | güclü |
| **enthusiastic** | həvəsli | rəsmi + davamlı |
| **excited** | həyəcanlı | ani, pozitiv |
| **thrilled** | çox həyəcanlı | güclü |
| **ecstatic** | dəli olmuş qədər şad | ən güclü |
| **passionate** | tutqun / tamamilə istəkli | rəsmi |

> **Qısa qayda:**
> - **excited** = həyəcanlı (ani)
> - **thrilled** = çox həyəcanlı
> - **enthusiastic** = həvəsli (davamlı)
> - **eager** = istəkli
> - **passionate** = tutqun (peşə haqqında)

---

## İntensivlik

```
interested < keen < eager < enthusiastic < excited < thrilled < ecstatic
```

---

## 1. Excited — Həyəcanlı (Ani)

Gələcək hadisədən həyəcanlı. Pozitiv.

### Nümunələr

- I'm **excited** about the trip.
- **Excited** for the release.
- She's so **excited**.
- Really **excited** to start.

### Struktur

- **excited** about + N
- **excited** for + N
- **excited** to + V
- **excited** that + clause

- **Excited about** the project.
- **Excited for** the launch.
- **Excited to** join.
- **Excited that** we won.

### About vs For

- **excited about** = bir şey haqqında (daha ümumi)
- **excited for** = hadisə gözləyəndə (AmE, "someone's happy for X")

Hər ikisi mümkün.

---

## 2. Thrilled — Çox Həyəcanlı

**Excited**-dan güclü. Böyük sevinc.

### Nümunələr

- I'm **thrilled** to get the job!
- **Thrilled** with the results.
- Absolutely **thrilled**.

### Excited vs Thrilled

- **excited** = pozitiv həyəcan
- **thrilled** = güclü sevinc (surprise + joy)

- "I'm **excited** about the interview." (gözləyirəm)
- "I'm **thrilled** I got the offer!" (nəticə)

---

## 3. Enthusiastic — Həvəsli (Davamlı)

Davamlı həvəs / enerji. Rəsmi.

### Nümunələr

- An **enthusiastic** learner.
- **Enthusiastic** about Python.
- **Enthusiastically** received.
- Very **enthusiastic** response.

### Excited vs Enthusiastic

- **excited** = hər hansı hadisəyə (müvəqqəti)
- **enthusiastic** = xarakter / davamlı həvəs

- "**Excited** about today's meeting." (bu an)
- "**Enthusiastic** about learning." (həmişə)

### Interview kontekstində

- "I'm an **enthusiastic** learner." ✓ (xarakter)
- "I'm **excited** about this role." ✓ (bu an)

---

## 4. Eager — İstəkli / Səbirsiz

"Çox istəyirəm, tezliklə başlamaq istəyirəm."

### Nümunələr

- **Eager** to start.
- **Eager** to learn.
- An **eager** student.
- **Eager** for change.

### Struktur

- **eager** to + V
- **eager** for + N

### Eager vs Enthusiastic

- **enthusiastic** = ümumi həvəs
- **eager** = tezliklə istəyən (action-ə yönəlmiş)

- "**Enthusiastic** about this tech." (həvəs)
- "**Eager** to try it." (tez başla!)

### Idiom

- **eager beaver** = çalışqan adam (bəzən neqativ)

---

## 5. Keen — Maraqlı (British)

British English. "Eager"-ə yaxın.

### Nümunələr

- **Keen** to learn.
- A **keen** interest.
- **Keen** observer.

### Struktur

- **keen** to + V
- **keen** on + N
- **keen** for + N

- **Keen** to start.
- **Keen on** Python.

### American vs British

- AmE: "**eager** to learn" ✓
- BrE: "**keen** to learn" ✓

Hər ikisi anlaşılır, lakin region ilə fərqli.

---

## 6. Passionate — Tutqun (Tam İstəkli)

Peşə / sahə haqqında ciddi maraq.

### Nümunələr

- **Passionate** about coding.
- **Passionate** leader.
- **Passion** for design.
- Deep **passion** for ML.

### Enthusiastic vs Passionate

- **enthusiastic** = həvəsli
- **passionate** = dərin, tutqun (uzun müddət)

- "**Enthusiastic** about the project." (bu layihə)
- "**Passionate** about software." (bütün sahə)

### Interview qızıl söz!

- "I'm **passionate** about clean code." ✓
- "I'm **passionate** about mentorship." ✓

---

## 7. Interested — Maraqlı (Neytral)

Ən sadə. Maraq göstərir.

### Struktur

- **interested** in + N/V-ing

- **Interested in** AI.
- **Interested in** learning Rust.

### ⚠ Diqqət

- **interested** = mən marağım var
- **interesting** = bu şey maraqlıdır

→ [ed-vs-ing-adjectives.md](ed-vs-ing-adjectives.md)

## Test

Hansı söz uyğundur?

1. I'm ______ about this new opportunity. (ani həyəcan)
2. I'm absolutely ______ to join your team! (güclü, ani)
3. She's ______ about teaching — been doing it 20 years. (dərin, davamlı)
4. I'm ______ to start on Monday. (tezliklə istəyirəm)
5. An ______ team with high energy. (rəsmi davamlı)
6. I'm ______ in machine learning. (neytral maraq)

**Cavablar:**
1. excited
2. thrilled
3. passionate
4. eager
5. enthusiastic
6. interested

## Interview Nümunələri

### Accepting offer

- "I'm **thrilled** to accept this offer." ✓ (strong)
- "I'm **excited** to join the team." ✓
- "I'm **eager** to start contributing." ✓

### Describing yourself

- "I'm **passionate** about backend development." ✓ (ən yaxşı!)
- "I'm an **enthusiastic** learner." ✓
- "I'm **keen on** new technologies." ✓ (British-friendly)

### About role

- "I'm **excited** about the challenges." ✓
- "I'm **eager** to contribute." ✓

## Azərbaycanlı Səhvləri

- ✗ I'm excited at the job. (at → about / for)
- ✓ I'm **excited about** the job.

- ✗ Passionated about coding. (passionated yox!)
- ✓ **Passionate** about coding.

- ✗ I'm interesting in AI. (interested → mən)
- ✓ I'm **interested** in AI.

## Xatırlatma

- **excited** = həyəcanlıyam (indi)
- **thrilled** = çox həyəcanlı (böyük xəbər)
- **enthusiastic** = həvəsli (davamlı)
- **eager** = tez başlamaq istəyirəm
- **passionate** = tutqun, peşə
- **interested** = maraqlanıram
