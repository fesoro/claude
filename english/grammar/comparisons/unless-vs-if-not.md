# Unless vs If Not — Əksər Hallarda Eyni, Amma...

## Səviyyə
B1

---

## Əsas Fərq

| Forma | Məna |
|-------|------|
| **unless** | ...-madıqca, ...olmasa |
| **if ... not** | əgər ... deyilsə |

> **Qısa qayda:**
> - **unless** = if ... not (əksər hallarda eyni)
> - Amma: hipotetik situasiyalarda fərq var!

---

## 1. Unless — ...Olmasa

"Unless" = "if ... not".

### Nümunələr

- **Unless** you study, you'll fail.
- I won't come **unless** you invite me.
- **Unless** it rains, we'll play.
- Don't call **unless** it's urgent.

### Struktur

- **unless** + present tense (şərt kimi)
- **unless** + subject + V

### If not ilə çevir

- **Unless** you study = If you **don't** study.
- **Unless** it rains = If it **doesn't** rain.

Eyni məna.

---

## 2. If Not — Əgər ... Deyilsə

"Unless"-in açıq forması.

### Nümunələr

- If you **don't** study, you'll fail.
- I won't come if you **don't** invite me.
- If it **doesn't** rain, we'll play.

### Kritik Fərq

Əksər hallarda eyni mənaya gəlir.

### Struktur

- **if** + subject + **don't / doesn't / won't** + V
- **if not** (kısa forma — nadir)

---

## 3. HOLA FƏRQİ (Hipotetik!)

**Unless hipotetik situasiyalarda işləmir!**

### Real situation — hər ikisi işləyir

- **Unless** you hurry, you'll be late. ✓
- If you **don't hurry**, you'll be late. ✓

### Hipotetik (unreal) — yalnız "if not"!

- ✗ **Unless** I were tired, I'd come. ✗
- ✓ If I **weren't** tired, I'd come. ✓

- ✗ **Unless** I had time, I would travel.
- ✓ If I **didn't have** time, I wouldn't travel.

**Qayda:** 2nd / 3rd conditional-də **unless** istifadə etmə!

---

## 4. Nə Vaxt Unless Daha Təbii?

### Güclü şərt ifadə

- You'll fail **unless** you study.
  (mütləq oxu, yoxsa fail)

### Exception mənası

- No entry **unless** authorized.
- Don't open **unless** necessary.

### Warning / threat

- **Unless** you pay, we'll stop service.
- You can't drive **unless** you have a license.

---

## Test

Hansı forma uyğundur?

1. ______ you're careful, you'll break it. (şərt — hər ikisi)
2. If I ______ tired, I would come. (hipotetik — if not)
3. We'll go ______ it rains. (şərt)
4. ______ I had money, I'd travel. (hipotetik — if not)
5. Don't email ______ it's urgent. (şərt)

**Cavablar:**
1. Unless / If not
2. weren't (if I weren't tired)
3. unless
4. If (If I had money)
5. unless

---

## Tech / İş Kontekstində

### Unless (real)

- "**Unless** tests pass, we don't deploy." ✓
- "Don't roll back **unless** it's critical." ✓
- "Merge **unless** there are blocking issues." ✓

### If not

- "If tests **don't pass**, we don't deploy." ✓ (eyni)

### Rəsmi polisy

- "**Unless** authorized, access is denied." ✓

---

## Documentation

API docs-də çox istifadə olunur:
- "Returns 200 **unless** the request is invalid."
- "**Unless** specified, default values are used."

---

## Azərbaycanlı Səhvləri

- ✗ Unless I were you, I wouldn't do it. (hipotetik — unless YOX!)
- ✓ If I **weren't** you, I wouldn't do it. (but actually this sentence is: If I were you...)

Müqayisə:
- ✓ **If I were you, I would...** (2nd conditional — if istifadə et)
- ✗ Unless I were you... (səhv)

- ✗ Unless you would help, I failed. (hipotetik)
- ✓ If you **hadn't helped**, I would have failed. (3rd conditional — if!)

---

## Double Negative Tələsi

**Unless** zatən negative elementi daşıyır. Ona görə cümlənin qalan hissəsində **negative yox!**

### ✗ Double negative (səhv)

- ~~Unless you don't study...~~ (if you study → mənasız)

### ✓ Düzgün

- **Unless** you study, you'll fail. (if you don't study)

---

## Unless vs Except When / Except If

### Unless

- **Unless** it rains, we'll go.

### Except when

- We go out **except when** it rains.

### Except if

- We go **except if** it rains.

Hər üçü oxşar, amma:
- **unless** = ən təbii
- **except when** = "həmişə... amma bu hallarda yox"
- **except if** = "əgər... deyilsə, edirik"

---

## "Unless" ilə Tenses

### Present / future

- **Unless** you hurry, you'll be late. ✓

### Past (real)

- **Unless** I was sick, I came to work. ✓
- I came **unless** I was sick.

### Past hipotetik? ✗

- ✗ Unless I was sick, I would have come.
- ✓ If I **hadn't been** sick, I would have come.

---

## "Unless" Cümlə Yerində

### Başda

- **Unless** you study, you'll fail.

### Ortada

- You'll fail **unless** you study.

Hər ikisi düzdür.

---

## Related İfadələr

### Only if

Məhdudlaşdıran şərt.
- **Only if** you agree.

### Provided that

Rəsmi. "Unless"-in əksi.
- **Provided that** you agree, we'll proceed.

### As long as

- **As long as** it works, we're fine.

### In case

Ehtiyat üçün.
- Take an umbrella **in case** it rains.

---

## Xatırlatma

| Forma | İstifadə |
|-------|----------|
| **unless** | real şərt + "if not" |
| **if ... not** | hər yerdə + hipotetik |

**Qızıl qayda:** Hipotetik (if I were / had) → **if not** (unless yox!)

**Test:** Real şərt? → **unless** OK. Hipotetik (would / had)? → **if not**.
