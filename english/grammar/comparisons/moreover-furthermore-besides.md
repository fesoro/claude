# Moreover / Furthermore / Besides / In Addition — Əlavə Etmə

## Səviyyə
B1 (rəsmi yazıda çox işlənir)

---

## Əsas Cədvəl

| Söz | Tonu | İstifadə yeri |
|-----|------|---------------|
| **moreover** | rəsmi | yazı, rəsmi çıxış |
| **furthermore** | rəsmi | akademik, rəsmi |
| **besides** | informal | gündəlik |
| **in addition** | neytral | ümumi |
| **also** | daily | danışıqda |

> **Qısa qayda:**
> - **moreover, furthermore** = rəsmi "bundan əlavə"
> - **besides** = ümumi "əlavə olaraq"
> - **in addition** = neytral
> - **also** = sadə (daily)

---

## 1. Moreover — Rəsmi "Bundan Əlavə"

Akademik / rəsmi yazı. Güclü əlavə arqument.

### Nümunələr

- We need more tests. **Moreover**, we need documentation.
- This approach is fast. **Moreover**, it's simple.
- He's experienced. **Moreover**, he leads well.

### Struktur

- Cümlənin əvvəlində + vergül
- **Moreover**, ...

### Kontekst

- essay / rəsmi yazı
- business report
- güclü arqument

---

## 2. Furthermore — Rəsmi "Əlavə Olaraq"

Moreover-a çox yaxın. Akademik.

### Nümunələr

- The system is efficient. **Furthermore**, it's scalable.
- Performance improved. **Furthermore**, costs decreased.
- We found a bug. **Furthermore**, there are security issues.

### Moreover vs Furthermore

Praktik eyni:
- **furthermore** = bir az daha rəsmi
- **moreover** = orta rəsmi

Hər ikisi eyni yerlərdə istifadə olunur.

---

## 3. Besides — İnformal / Ümumi

Gündəlik yazı. "Həm də, onsuz da".

### İki məna

#### A. Əlavə olaraq (in addition)

- I'm tired. **Besides**, it's late.
- She speaks Python. **Besides**, she knows Go.

#### B. Preposition — istisna / əlavə

- **Besides** Python, I know Go.
- Is anyone else here **besides** you?

### Besides vs Beside

- **besides** = əlavə olaraq
- **beside** = yanında (location)

Diqqət! → [beside-vs-besides.md](beside-vs-besides.md)

### Moreover vs Besides

- **moreover** = rəsmi əlavə arqument
- **besides** = informal, "onsuz da"

- "The code is buggy. **Moreover**, it's slow." (rəsmi)
- "I'm tired. **Besides**, it's late." (informal)

---

## 4. In Addition — Neytral "Əlavə Olaraq"

Ümumi. Hər yerdə istifadə olunur.

### Forma

- **In addition**, ... (cümlə başı)
- **In addition to** + N (preposition)

### Nümunələr

- He's smart. **In addition**, he's kind.
- **In addition to** Python, I know Java.
- **In addition to** speed, we need quality.

### In addition vs Moreover

- **in addition** = neytral
- **moreover** = güclü rəsmi

Bir-birini əvəz edə bilər. **In addition** daha çox work-də.

---

## 5. Also — Sadə / Gündəlik

Ən çox istifadə olunur. Neytral-informal.

### Forma

- middle of sentence: I **also** speak French.
- cümlə başı: **Also**, we need tests.

### Nümunələr

- I speak Python. I **also** know Go.
- We need speed. **Also**, reliability matters.
- The team is skilled. **Also**, they're motivated.

### Also vs Moreover

- **also** = sadə (her konteks)
- **moreover** = rəsmi, güclü

---

## 6. Əlavə Sözlər

### As well (cümlə sonunda)

- I know Python **as well**.

### Too (cümlə sonunda)

- I want coffee, **too**.

### Plus (informal)

- He's smart. **Plus**, he's funny.

### What's more (informal)

- He's skilled. **What's more**, he's fast.

### On top of that

- The code is slow. **On top of that**, it's buggy.

### Not only ... but also

- **Not only** is it fast, **but also** scalable.

---

## Spektrum: Formal → Informal

```
Furthermore → Moreover → In addition → Also → Besides → What's more → Plus
    ←── RƏSMİ ──              ──→           ──→ İNFORMAL ──→
```

## Yazı Stili

### Rəsmi məqalə / essay

- **Moreover**, the system demonstrates scalability.
- **Furthermore**, the authors provide detailed analysis.

### Business email

- **In addition**, please review the attached docs.
- We'll meet at 3. **Also**, bring your laptop.

### Informal Slack

- Busy today. **Plus**, I have a meeting.
- We need to fix this. **Besides**, it's been broken for weeks.

---

## Tech / İş Kontekstində

### Design doc

- The API is RESTful. **Moreover**, it supports GraphQL.
- We use Redis for caching. **Furthermore**, we plan to add CDN.

### Code review

- This is cleaner. **In addition**, it's more testable.
- The fix works. **Also**, consider the edge case.

### Standup

- I fixed the bug. **Plus**, I wrote tests.

---

## Azərbaycanlı Səhvləri

- ✗ Moreover of the bug, there are errors. (moreover cümlənin başı)
- ✓ There's a bug. **Moreover**, there are errors.

- ✗ In addition the features, we need tests. (to lazım!)
- ✓ **In addition to** the features, we need tests.

- ✗ "Too" cümlənin başında. (adətən sonda olur!)
- ✓ I want coffee, **too**. / **Also**, I want coffee.
