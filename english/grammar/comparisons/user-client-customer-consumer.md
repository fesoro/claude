# User / Client / Customer / Consumer — Kim Kimdir?

## Səviyyə
B1 (tech və business kontekst)

---

## Əsas Cədvəl

| Söz | Kontekst | Nüans |
|-----|----------|-------|
| **user** | məhsulu istifadə edən | tech / app |
| **customer** | pul ödəyən | biznes / satış |
| **client** | xidmət alan | B2B / rəsmi |
| **consumer** | son istehlakçı | marketinq / iqtisadi |
| **end user** | texniki zəncirdə son istifadəçi | technical |

> **Qısa qayda:**
> - **user** = istifadə edir (pul verə bilər və ya verməz)
> - **customer** = pul verir
> - **client** = xidmət alır (uzun müddət)
> - **consumer** = iqtisadi/market terminologiyası

---

## 1. User — İstifadəçi

Tech / product konteksti. Pul verib-verməməsi önəmli deyil.

### Nümunələr

- The app has 1 million **users**.
- **Users** can log in via Google.
- This feature helps **users** save time.
- **User** experience (UX).
- **User** interface (UI).
- **End users** don't care about the backend.

### Compound sözlər

- **user** account
- **user** profile
- **user** data
- **user** feedback
- **user** journey
- **user** story (Agile)
- power **user** (advanced)

---

## 2. Customer — Müştəri (Pul Verən)

Pul ödəyir və ya məhsul alır. Retail / e-commerce.

### Nümunələr

- Our **customers** love the product.
- **Customer** support is critical.
- The **customer** is always right.
- 80% of **customers** renewed.
- **Customer** satisfaction.

### Compound

- **customer** service
- **customer** support
- **customer** feedback
- **customer** journey
- **customer** retention
- **customer** churn
- paying **customer**

---

## 3. Client — Klient / Müştəri (Xidmət)

B2B, agentlik, konsultant iş. Uzunmüddətli əlaqə.

### Nümunələr

- We have 5 major **clients**.
- The **client** wants a redesign.
- I'm meeting the **client** tomorrow.
- **Client** requirements.
- Enterprise **clients**.

### Tech dual mənası

Tech-də "client" həm də yarpaqdır (client-server):
- **Client** app / **client**-side code.
- The **client** sends a request to the server.
- **Client** library / SDK.

### Client vs Customer fərqi

- **client** → xidmət (consulting, law, agency)
- **customer** → məhsul / mal (retail, SaaS)

- A lawyer has **clients**.
- McDonald's has **customers**.

---

## 4. Consumer — İstehlakçı

Marketinq, iqtisadi, ümumi istehlak.

### Nümunələr

- **Consumer** behavior.
- **Consumer** electronics (iPhone, TV...).
- Average **consumer**.
- **Consumer** demand is rising.
- B2C = Business-to-**Consumer**.

### Customer vs Consumer

- **Customer** = bu mağazadan alan insan
- **Consumer** = ümumi istehlakçılar kütləsi

- Our **customers** love us. (bizim müştərilər)
- **Consumer** tastes are changing. (bazarda istehlakçılar)

---

## 5. End User — Son İstifadəçi

Texniki zəncirdə son insan. API konsumer, developer deyil.

### Nümunələr

- The API is used by developers, but **end users** see only the UI.
- Design for the **end user**.
- **End-user** experience.
- **End-user** documentation.

---

## Hansını Nə Vaxt?

### Software product (SaaS)

- If free → **users**
- If paid → **customers** / **users** (hər ikisi)
- If B2B → **clients**

### E-commerce

- Adətən **customers**.

### Consulting / Agency

- **Clients** (uzunmüddətli iş).

### API / dev tool

- **Users** (developerlər).

### Market research

- **Consumers** (ümumi kütlə).

## Azərbaycanlı Səhvləri

- ✗ Our application has 1000 customers. (app free isə → users)
- ✓ Our application has 1000 **users**.

- ✗ The law firm's users.
- ✓ The law firm's **clients**.
