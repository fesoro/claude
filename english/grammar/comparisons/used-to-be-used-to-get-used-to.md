# Used to / Be Used to / Get Used to — Üç Fərqli Struktur

## Səviyyə
B1 (ən çox qarışdırılan struktur)

---

## Əsas Cədvəl

| Struktur | Məna | Sonra gələn |
|----------|------|-------------|
| **used to + V1** | keçmişdəki adət / vəziyyət (indi yoxdur) | base verb |
| **be used to + -ing / noun** | öyrəşmişəm (artıq adət) | -ing / isim |
| **get used to + -ing / noun** | öyrəşirəm / öyrəşəcəyəm (proses) | -ing / isim |

> **Qısa qayda:**
> - **used to + V1** = keçmişdə edirdim, indi yox (past habit)
> - **be used to + -ing** = artıq öyrəşmişəm (current familiarity)
> - **get used to + -ing** = öyrəşirəm (process of becoming familiar)

---

## 1. Used to + V1 — Keçmiş Adət

Keçmişdə müntəzəm edirdim, amma indi etmirəm.

### Struktur: subject + used to + base verb

- I **used to smoke**. (əvvəllər siqaret çəkirdim, indi yox)
- She **used to live** in London. (əvvəllər yaşayırdı)
- We **used to play** football every Sunday.
- He **used to be** shy.
- They **used to go** to the cinema often.

### İnkar və sual

| İnkar | Sual |
|-------|------|
| I **didn't use to** smoke. | **Did** you **use to** smoke? |
| She **didn't use to** live here. | **Did** she **use to** live here? |

> **Diqqət:** İnkar/sualda "use to" — "d" itir (did+use to)!
> - ✓ Did you **use to** play football?
> - ✗ Did you used to play football?

### Nümunələr (interview kontekstində)

- I **used to work** at a small startup before joining Zipmend.
- We **used to release** once a month. Now we release daily.
- I **used to be** afraid of public speaking.

---

## 2. Be Used to + -ing / noun — Artıq Öyrəşmiş

Indi artıq adətdir, qəbul etmişəm, tanışam.

### Struktur: subject + be + used to + -ing / noun

- I **am used to working** late. (gec işləmək mənim üçün normaldır)
- She **is used to** the cold weather.
- We **are used to** long meetings.
- He **is used to getting up** early.
- They **are used to** stress.

### İnkar

- I **am not used to** this food.
- She **isn't used to working** alone.

### Zamanlar

`be` feili dəyişə bilər:

| Zaman | Nümunə |
|-------|--------|
| Present | I **am used to** this. |
| Past | I **was used to** it. |
| Future | You **will be used to** it soon. |
| Perfect | I **have been used to** it for years. |

### Nümunələr

- I **'m used to working** remotely. (uzaqdan işləməyə öyrəşmişəm)
- I **'m used to deadlines**. (son tarixlərə alışmışam)
- Developers **are used to debugging**.

---

## 3. Get Used to + -ing / noun — Öyrəşmə Prosesi

Öyrəşmə prosesindəsən — hələ tam yox, amma gedir.

### Struktur: subject + get used to + -ing / noun

- I **am getting used to** the new job. (öyrəşirəm)
- It takes time to **get used to** living abroad.
- You'll **get used to it**. (öyrəşəcəksən)
- She **got used to** waking up early.
- I can't **get used to** this keyboard.

### Zamanlar — yaygın istifadə

| Forma | Nümunə |
|-------|--------|
| am/is/are getting used to | I'**m getting used to** the schedule. (prosesdə) |
| will get used to | You'**ll get used to** it. (gələcəkdə) |
| can/can't get used to | I **can't get used to** this. (bacarmıram) |
| got used to | He **got used to** the cold eventually. (öyrəşdi) |

### Nümunələr (interview və iş)

- I **'m getting used to** the new codebase.
- New developers need time to **get used to** our process.
- After a week, I **got used to** the timezone.

## Müqayisə — Üçü Yan-yana

| Struktur | Vaxt | Məna |
|----------|------|------|
| I **used to smoke**. | keçmiş | Əvvəllər çəkirdim, indi yox. |
| I **am used to smoking**. | indi | Siqaret çəkməyə alışmışam (adətdir). |
| I **am getting used to smoking**. | proses | Siqaret çəkməyə öyrəşirəm. (qeyri-adi cümlə, amma qaydaya uyğun) |

### Real nümunə

- Əvvəllər 9-5 işləyirdim: I **used to work** 9-5.
- İndi gec işləməyə alışmışam: I **'m used to working** late now.
- Yeni gecə növbəsinə öyrəşirəm: I **'m getting used to** the night shift.

---

## Əsas Fərq — Grammatik Baxımdan

### Used to — fel kimi (+ V1)

`used to` burada **yarı-modal** kimi davranır. Sonrası **base verb**.

- ✓ I used to **play**.
- ✗ I used to playing.

### Be used to / Get used to — ifadə (+ -ing / noun)

`used to` burada **sifət ifadəsi** kimi. `to` = preposition. Sonrası **gerund (-ing)** və ya **isim**.

- ✓ I am used to **playing**.
- ✓ I am used to **the game**.
- ✗ I am used to play. (yanlış!)

> **Praktik qayda:** `be / get` varsa → -ing. `be / get` yoxdur → base verb.

---

## "Used to" və ya "Would" — keçmiş adət

Keçmişdə adəti bildirmək üçün "would" da işlənir, amma fərq var:

| used to | would |
|---------|-------|
| Keçmiş adət **+ vəziyyət** | Yalnız keçmiş **adət** (vəziyyət yox) |
| I **used to live** in Baku. ✓ (vəziyyət) | I **would live** in Baku. ✗ (yanlış) |
| I **used to play** football. ✓ | I **would play** football. ✓ (hadisə) |
| She **used to be** shy. ✓ | She **would be** shy. ✗ |

> **Qayda:** Vəziyyət / stative (be, have, know, love) → **used to** işlənir, **would** yox.

Ətraflı: [used-to-vs-would.md](used-to-vs-would.md)

## Tez-tez Edilən Səhvlər

| Səhv | Düzgün | İzah |
|------|--------|------|
| I used to working late. | I'**m used to working** late. | be used to + -ing |
| I am used to work late. | I'**m used to working** late. | be used to + -ing (to = preposition) |
| Did you used to smoke? | **Did you use to** smoke? | sual/inkar: use to |
| I didn't used to drink. | I **didn't use to** drink. | -d itir |
| I'm used to play football. (adətə öyrəşmişəm) | I'**m used to playing** football. | -ing lazımdır |
| I get used to it now. | I'**m getting used to** it now. (proses) / I'**m used to** it now. (tam öyrəşmişəm) | fərq: proses vs state |
| She uses to live here. | She **used to live** here. | past → used to |
| I am used to the cold for many years. | I've **been used to** the cold for many years. | present perfect |

## Müsahibə Kontekstində İstifadə

### Keçmiş təcrübə

- I **used to work** with PHP, but now I mainly use Go.
- We **used to** deploy once a month — now we deploy every day.
- I **used to be** afraid of code reviews, but now I enjoy them.

### Öyrəşmiş vəziyyət

- I'**m used to working** under pressure.
- I'**m used to** remote collaboration.
- My team **is used to** rapid iteration cycles.
- I **was used to** long hours at my previous job.

### Öyrəşmə prosesi

- I'**m still getting used to** the new framework.
- It took me two weeks to **get used to** the codebase.
- Junior devs usually **get used to** our tools in a month.

## Məşq Tapşırıqları

### 1. Düzgün formanı seçin

1. I ___ (used to / am used to) live in Ganja when I was young.
2. I ___ (used to / am used to) working with Docker.
3. She ___ (is getting used to / used to) the new office.
4. Did you ___ (use to / used to) smoke?
5. I can't ___ (get used to / used to) this keyboard layout.
6. We ___ (used to / get used to) play football every Sunday.
7. He ___ (is used to / used to) getting up early — it's no problem for him.
8. I ___ (used to / am used to) hate coffee, but now I love it.

**Cavablar:** 1-used to, 2-am used to, 3-is getting used to, 4-use to, 5-get used to, 6-used to, 7-is used to, 8-used to

### 2. Cümləni tamamlayın

1. I used to ___ (live) in a small town.
2. I am used to ___ (work) long hours.
3. She is getting used to ___ (wake) up early.
4. He didn't use to ___ (like) coffee.
5. You'll get used to ___ (drive) on the left.

**Cavablar:** 1-live, 2-working, 3-waking, 4-like, 5-driving

### 3. Tərcümə edin

1. Əvvəllər kitab oxuyurdum, indi yox. → I **used to read** books, but not anymore.
2. Gecə növbəsinə öyrəşmişəm. → I'm **used to** the night shift.
3. Yeni komandaya öyrəşirəm. → I'm **getting used to** the new team.
4. Qatarla işə getməyə alışmışam. → I'm **used to** taking the train to work.
5. Əvvəllər ingilis dilindən qorxurdum. → I **used to be** afraid of English.
