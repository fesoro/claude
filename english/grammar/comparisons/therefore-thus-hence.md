# Therefore / Thus / Hence / So / Consequently — Nəticə Bağlayıcıları

## Səviyyə
B1 (rəsmi yazı)

---

## Əsas Cədvəl

| Söz | Tonu | İstifadə yeri |
|-----|------|---------------|
| **therefore** | rəsmi | yazı, rəsmi |
| **thus** | çox rəsmi | akademik |
| **hence** | rəsmi, nəticə | rəsmi, yazılı |
| **consequently** | rəsmi, nəticə | biznes, rəsmi |
| **so** | daily | gündəlik |
| **as a result** | neytral | ümumi |

> **Qısa qayda:**
> - **therefore** = rəsmi "beləliklə"
> - **thus** = akademik
> - **hence** = "buna görə"
> - **so** = gündəlik
> - **as a result** = ən təbii

---

## 1. Therefore — Rəsmi "Beləliklə"

Ən çox rəsmi yazıda istifadə olunur.

### Nümunələr

- The tests passed. **Therefore**, we can deploy.
- He's the best candidate. **Therefore**, we hired him.
- The server is slow. **Therefore**, we need caching.

### Struktur

- Cümlənin əvvəlində + vergül
- **Therefore**, ...

### İnformal əvəz

- "**So**, we can deploy."
- "**That's why** we need caching."

---

## 2. Thus — Çox Rəsmi (Akademik)

Akademik / bilimi. Nadir gündəlikdə.

### Nümunələr

- The test failed. **Thus**, we reverted.
- The API returned 500. **Thus**, the request failed.

### Therefore vs Thus

- **therefore** = rəsmi, ama gündəlik iş yazısında OK
- **thus** = akademik, elm

Rəsmi yazıda məktəbdə / universitetdə **thus** daha çox.

---

## 3. Hence — "Buna görə"

Rəsmi. İki forma:

### A. Normal bağlayıcı

- The code is slow. **Hence**, we need optimization.
- He studied hard. **Hence**, he passed.

### B. Uzaq ifadə ("indidən")

- Two weeks **hence** = iki həftə sonra (rəsmi)

**Tech kontekst ikinci az istifadə olunur.**

### Hence vs Therefore

Çox yaxındır. **Therefore** daha çox, **hence** daha rəsmi.

---

## 4. Consequently — Nəticə Olaraq

Xüsusilə nəticə vurğulamaq üçün.

### Nümunələr

- Tests failed. **Consequently**, the deploy was blocked.
- Traffic spiked. **Consequently**, the server crashed.
- **Consequently**, we added monitoring.

### Güclü nəticə hissi

"Consequently" ağır nəticə ifadə edir.

---

## 5. So — Gündəlik "Beləliklə"

Ən çox istifadə. Sadə.

### Struktur

- Cümlənin ortasında (bağlayıcı)
- Cümlənin başında

### Nümunələr

- It's raining, **so** I'll stay home.
- I was tired. **So**, I went to bed.
- The build failed, **so** we rolled back.

### So vs Therefore

- **so** = daily, informal
- **therefore** = rəsmi, yazılı

- "It's late, **so** I'm leaving." (informal)
- "The meeting ended. **Therefore**, I left." (rəsmi)

### Diqqət: "So" cümlə başında?

Modern yazıda OK. Köhnə qaydada qəbul olunmurdu.

- "**So** what do you think?" ✓ (informal)
- "The project ended. **So**, we celebrated." ✓ (daily)

---

## 6. As a Result — Neytral "Nəticədə"

Ən təbii. Hər yerdə işləyir.

### Nümunələr

- Traffic grew. **As a result**, we scaled up.
- **As a result** of the changes, performance improved.
- We refactored. **As a result**, tests are faster.

### Struktur

- **As a result**, + clause (bağlayıcı)
- **As a result of** + N (preposition)

---

## Spektrum: Formal → Informal

```
Thus → Hence → Therefore → Consequently → As a result → So
 ←── AKADEMİK ──         ──→        NEYTRAL   ──→    İNFORMAL ──→
```

2. It was late. ______ I went home. (informal)
3. We identified the bug. ______ , we fixed it. (akademik)
4. Traffic increased 10x. ______ , the DB crashed. (güclü nəticə)
5. ______ of the outage, customers left. (preposition)

**Cavablar:**
1. Therefore
2. So
3. Thus
4. Consequently
5. As a result

## Tech / İş Kontekstində

### Design doc / rəsmi

- "Performance testing showed bottlenecks. **Therefore**, we added indexes."
- "The logs indicate memory issues. **Hence**, we recommend increasing RAM."
- "All tests pass. **Thus**, the feature is ready."

### Standup / informal

- "Bug was critical, **so** I fixed it."
- "Deploy failed, **so** I reverted."

### Post-mortem

- "The config was wrong. **Consequently**, services went down."
- "**As a result**, users couldn't log in."

---

## Email Templates

### Formal

- "Thank you for your feedback. **Therefore**, we're adjusting the timeline."
- "The scope has increased. **Consequently**, we need more resources."

### Informal

- "We got the feedback, **so** we'll adjust."

---

## Azərbaycanlı Səhvləri

- ✗ So we can deploy. (cümlə başı, yazıda → therefore)
- ✓ **Therefore**, we can deploy. (rəsmi)
- ✓ The tests passed, **so** we can deploy. (informal)

- ✗ As result of changes... (article lazım)
- ✓ **As a result of** the changes...

---

## Double-check: Səbəb vs Nəticə

### Səbəb bağlayıcıları (→ ətraflı: [because-due-to.md](because-due-to.md))

- because, since, as, due to
- → "niyə?"

### Nəticə bağlayıcıları (bu səhifə)

- therefore, thus, hence, so, consequently, as a result
- → "nə oldu?"

### Eyni cümlə, iki cür

- "We failed **because** the tests crashed." (səbəb)
- "The tests crashed, **so** we failed." (nəticə)

---

## Punctuation Qaydaları

### Cümlə başında

- **Therefore,** we... (vergül vacibdir!)
- **Thus,** we... (vergül)
- **Hence,** we... (vergül)
- **As a result,** we... (vergül)
- **Consequently,** we... (vergül)

### Cümlə ortasında (bağlayıcı)

- ..., **so** we... ✓
- ...; **therefore,** we... ✓ (semicolon)
