# Test / Check / Verify / Validate / Confirm — Yoxlama Fellər

## Səviyyə
B1-B2

---

## Əsas Cədvəl

| Söz | Nə edirsən? | Kontekst |
|-----|-------------|----------|
| **test** | yoxlama keçir | ümumi, software testing |
| **check** | sürətli baxış | gündəlik |
| **verify** | düzgün olduğunu təsdiq et | rəsmi, security |
| **validate** | qaydalara uyğunluğu yoxla | input, data |
| **confirm** | təsdiq et | rəsmi razılıq |

> **Qısa qayda:**
> - **test** = detallı yoxlama
> - **check** = sadə baxış
> - **verify** = düzdür? (təsdiq)
> - **validate** = qaydalara uyğun?
> - **confirm** = bəli, düzdür

---

## 1. Test — Yoxlamaq / Sınamaq

Ən geniş istifadə. Software testing, eksperiment.

### Nümunələr

- Did you **test** this in production?
- **Test** the new feature.
- Unit **tests** pass.
- A/B **testing**.
- Load **testing**.
- Smoke **test**.

### Combinasiyalar

- **test** it out
- **run** the tests
- **write** tests
- **pass** / **fail** the test
- **test coverage**

---

## 2. Check — Sürətli Baxış

Sadə, cəld. Təcrübəli yoxlama deyil.

### Nümunələr

- **Check** the logs.
- Can you **check** if it's working?
- **Check** the docs first.
- Quick **check** before deploy.
- **Check** your email.

### Check vs Test

- **check** = bir dəfəlik baxış
- **test** = sistemli yoxlama

- "**Check** if the server is up." (ping)
- "**Test** the new endpoint." (unit / integration)

### Phrasal

- **check out** = yoxla / nəzərdən keçir
- **check in** = qeydiyyatdan keç
- **double-check** = iki dəfə yoxla

---

## 3. Verify — Təsdiq Etmək (Düzgündür?)

Bir şeyin **doğru / həqiqi** olduğunu təsdiq. Rəsmi səslənir.

### Nümunələr

- Please **verify** your email.
- **Verify** the user identity.
- Can you **verify** the data?
- Two-factor **verification**.
- **Verify** the claim.

### Tech kontekst

- **Verify** the signature (cryptography)
- **Verify** the build artifact
- **Verify** tests ran in CI

### Verify vs Check

- **check** = sadə baxış
- **verify** = təsdiq (daha güclü, rəsmi)

- "**Check** if the tests pass." (sadə baxış)
- "**Verify** the deployment succeeded." (təsdiq)

---

## 4. Validate — Qaydalara Uyğunluğu Yoxlamaq

Input, form, data — **qaydalara uyğundurmu?**

### Nümunələr

- **Validate** user input.
- Form **validation**.
- **Validate** the JSON schema.
- **Validate** the config file.
- **Validate** the design with users.

### Tech

- Input **validation** = security-in əsası
- Email **validation**
- Schema **validation**
- Data **validation**

### Validate vs Verify

- **validate** = **qaydalara** uyğun?
- **verify** = **doğru / həqiqi**?

- Email **validation** = format düzdürmü? (e.g., has @)
- Email **verification** = bu email həqiqətən sənindir?

---

## 5. Confirm — Təsdiq Etmək (Razılıq)

Rəsmi razılıq, anlaşma, meeting.

### Nümunələr

- Please **confirm** your attendance.
- **Confirm** the order.
- Can you **confirm** the meeting time?
- **Confirmation** email.
- **Confirmed** — I'll be there.

### Confirm vs Verify

- **confirm** = razılıq / bildirmə
- **verify** = həqiqətlik yoxlaması

- "**Confirm** the order." (bəli, istəyirəm)
- "**Verify** the payment." (ödəniş düzdürmü)

---

## Fərqli Kontekstlərdə Müqayisə

### User signup flow

- User fills form → **validate** input (format uyğundurmu?)
- Send email → user clicks link → **verify** email (həqiqi poçt)
- User logs in → **confirm** login (təsdiq)

### CI/CD

- **Test** the code (unit tests)
- **Check** the build output (fail edib?)
- **Verify** the deployment (doğru getdi?)
- **Validate** config files (schema uyğun?)

## Test

Hansı söz daha uyğun?

1. Please ______ your email by clicking the link. (həqiqətlik)
2. The form ______ s user input. (qaydalar)
3. Did you ______ the tests before deploying? (yoxlamaq)
4. Can you ______ the meeting tomorrow? (razılıq)
5. Quick ______ — is the server up? (sadə)

**Cavablar:** 1. verify, 2. validates, 3. test, 4. confirm, 5. check

---

## İnterview Nümunələri

- "I wrote unit **tests** to **verify** the logic."
- "We **validate** all user inputs to prevent injection."
- "The CI pipeline runs **tests** and **verifies** the build."
- "Users **confirm** their email before accessing the dashboard."

## Azərbaycanlı Səhvləri

- ✗ Please verify your form before submit. (daha çox validate)
- ✓ Please **validate** your form before submit.

- ✗ Can you confirm if tests pass? (yoxlamaq — check)
- ✓ Can you **check** if tests pass?
