# Hot / Warm / Boiling / Scorching — İsti Sifətləri

## Səviyyə
A2-B1

---

## Əsas Cədvəl (İntensivlik)

| Söz | Dərəcə | Kontekst |
|-----|--------|----------|
| **warm** | ilıq (xoş) | rahat isti |
| **mild** | mülayim | hava |
| **hot** | isti | ümumi |
| **boiling** | qaynayan | çox isti |
| **scorching** | yandıran | ekstrem |
| **sweltering** | bunaldıcı | bunalıq |

> **Qısa qayda:**
> - **warm** = xoş isti (rahat)
> - **hot** = isti (normal yaxşı isti)
> - **boiling / scorching** = çox isti (dözülməz)

---

## İntensivlik

```
freezing < cold < cool < mild < warm < hot < boiling < scorching < sweltering
   çox soyuq                 rahat    isti     çox isti     yandıran
```

---

## 1. Warm — İlıq (Rahat)

Xoş hiss. Dözülməz deyil.

### Nümunələr

- **Warm** coffee.
- **Warm** weather.
- **Warm** clothes.
- **Warm** welcome.
- Feel **warm**.

### Emosional

- **Warm** hug.
- **Warm** smile.
- **Warm** person. (mehriban)
- **Warmly** welcomed.

### Tech / abstract

- **Warm** cache (hələ dolu).
- **Warm** up the system.

---

## 2. Hot — İsti

Normal isti. Bəzən dözüləsi olur.

### Nümunələr

- It's **hot** today.
- **Hot** coffee.
- **Hot** weather.
- Too **hot**!
- **Hot** food.

### Üç məna

#### A. Temperatura

- **Hot** water.
- **Hot** summer.

#### B. Acı (biber)

- **Hot** sauce.
- **Hot** pepper.

#### C. Slang: cəlbedici, populyar

- That's **hot**! (populyar)
- **Hot** topic.
- **Hot** new framework.

### Warm vs Hot

- **warm** = rahat isti
- **hot** = isti / daha çox

- **Warm** bath (xoş).
- **Hot** bath (bir az isti, bəlkə çox).

---

## 3. Boiling — Qaynayan

Çox isti. Dözülməz.

### Nümunələr

- **Boiling** hot!
- **Boiling** water.
- I'm **boiling**. (çox istiyəm)

### Struktur

- **boiling** hot (güclü forma)

### Hot vs Boiling

- **hot** = isti
- **boiling** = çox isti (ekstrem)

- "It's **hot** today." (normal)
- "It's **boiling** today!" (çox isti)

---

## 4. Scorching — Yandıran

Çox-çox isti. Fiziki / hava üçün.

### Nümunələr

- **Scorching** heat.
- **Scorching** sun.
- **Scorching** day.

### Kontekst

- Yay, səhra
- 40°C+ temperatura

---

## 5. Sweltering — Bunaldıcı

İsti + rütubətli. Çox pisdir.

### Nümunələr

- **Sweltering** heat.
- **Sweltering** weather.
- **Sweltering** in the car.

---

Şəxsi səhifə: [cold-cool-chilly-freezing.md](cold-cool-chilly-freezing.md)

## Test

Hansı söz uyğundur?

1. My coffee is ______ , not hot. (rahat)
2. It's absolutely ______ today — 42°C! (çox isti)
3. **Hot** showers in winter are nice. (normal)
4. ______ welcome! (emosional)
5. ______ summer in the desert. (yandıran)

**Cavablar:**
1. warm
2. boiling / scorching
3. Hot
4. Warm
5. Scorching

## Temperature Cədvəli

| Celsius | İngiliscə |
|---------|-----------|
| < 0°C | freezing |
| 0-10°C | cold |
| 10-18°C | cool |
| 18-24°C | mild / warm |
| 24-30°C | warm / hot |
| 30-40°C | hot / boiling |
| > 40°C | scorching / sweltering |

## Azərbaycanlı Səhvləri

- ✗ Warm day in August. (→ hot)
- ✓ **Hot** day in August. (istidir!)

- ✗ I'm very boiling from the sun. (struktur qəribdir)
- ✓ I'm **boiling**! / It's **boiling** today.
