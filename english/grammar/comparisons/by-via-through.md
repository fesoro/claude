# By / Via / Through — Vasitə Prepositionları

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Kontekst |
|-----|----------|
| **by** | metod / agent / nəqliyyat |
| **via** | vasitə / marşrut |
| **through** | proses / vasitə |

> **Qısa qayda:**
> - **by** = üsul / metod (ən ümumi)
> - **via** = marşrut / rəsmi vasitə
> - **through** = proses / intermediary

→ Related: [by-vs-with.md](by-vs-with.md), [through-across-over.md](through-across-over.md)

---

## 1. By — Metod / Agent

### Metod

- Pay **by** credit card.
- Travel **by** train.
- Send **by** email.

### Agent (passive)

- Written **by** me.
- Approved **by** manager.

### V-ing ilə

- Learn **by** doing.
- Win **by** working hard.

→ Ətraflı: [by-vs-with.md](by-vs-with.md), [to-vs-for.md](to-vs-for.md)

---

## 2. Via — Marşrut / Vasitə

Latın mənşəli. Rəsmi.

### Route

- Fly to Paris **via** London.
- Travel **via** the coast road.

### Communication method

- Contact **via** email.
- Connect **via** VPN.
- Payment **via** PayPal.

### Through vs Via

- **via** = spesifik marşrut / rəsmi
- **through** = proses / intermediary

- "Flew **via** Istanbul." (marşrut — durdu)
- "Drove **through** Istanbul." (şəhərdən keçdi)

### Tech kontekst

- **Via** API.
- Authenticate **via** OAuth.
- Deploy **via** GitHub Actions.
- Connect **via** SSH.

---

## 3. Through — Proses / Vasitə

İçərisindən keçmək + abstract istifadə.

### Physical (→ [through-across-over.md](through-across-over.md))

- Walk **through** the park.

### Abstract — proses

- Learn **through** experience.
- Grow **through** challenges.

### Intermediary

- Found it **through** a friend.
- Booked **through** an agent.
- Applied **through** LinkedIn.

### Through vs By

- **by** = direct method
- **through** = intermediary

- "Paid **by** card." (direct method)
- "Paid **through** PayPal." (intermediary)

---

## Müqayisə Nümunəsi

### "He got the job ..."

- **by** networking. (method/action — what he did)
- **via** a referral. (rəsmi route)
- **through** a friend. (intermediary)

### "Connected ..."

- **by** WiFi. (method)
- **via** VPN. (route/system)
- **through** a proxy. (intermediary)

---

## Tech Kontekstində

### By

- Deploy **by** pushing to main. ✓ (action)
- Authenticated **by** password. ✓

### Via

- Access **via** API. ✓ (interface)
- Deploy **via** CI/CD. ✓
- Connect **via** SSH. ✓
- Load balanced **via** nginx. ✓

### Through

- Request goes **through** the gateway. ✓ (intermediary)
- Debugged **through** logs. ✓
- Learn **through** docs. ✓

## Common Collocations

### By

- **by email**
- **by phone**
- **by mistake**
- **by chance**
- **by the way**
- **by default**

### Via

- **via email** (formal)
- **via phone**
- **via the app**
- **via satellite**

### Through

- **through email** (possible — less common)
- **through the system**
- **through training**
- **through research**

## When to Use Which?

### By (most common, default)

- Method / how
- "How did you..."
- Agent in passive

### Via (formal, route)

- Specific route (flight, driving)
- Rəsmi channel
- Tech platforms

### Through (process, intermediary)

- Abstract proses
- Intermediary (friend, agent)
- Learning / experience

---

## Bonus: Per / Through (Rəsmi)

### Per

Rəsmi, "görə".

- **Per** your request.
- **Per** the documentation.

### Formal alternative

- "As **per** our conversation..." (formal)
- "**Per** the instructions..."

Tech-də nadir, amma legal / formal docs-da.

---

## Tech Nümunələr

### Authentication

- "Authenticated **by** password." ✓
- "Authenticated **via** OAuth." ✓
- "Authenticated **through** the SSO provider." ✓

All three work! Difference:
- **by** = method
- **via** = system/interface
- **through** = intermediary

### Request flow

- Request goes **through** the load balancer, **via** HTTPS, sent **by** the client.

---

## Phrasal / Combined

### Go through

- **Go through** training (undergo).
- **Go through** the docs (review).

### Go by

- "The bus **goes by** here." (passes).
- "**Go by** the rules." (follow).

### Go via

- "Train **goes via** Bakı." (route).

---

## Formal Writing

### Email

- "Payment **via** credit card." ✓
- "Reachable **by** email." ✓

### Technical docs

- "Communicate **via** REST API." ✓
- "Processed **through** the queue." ✓

### Legal

- "**Per** the agreement..." ✓ (formal)
- "**By** signing below..." ✓

---

## Azərbaycanlı Səhvləri

- ✗ Send it through email. (rare → by / via)
- ✓ Send **by** email. / Send **via** email.

- ✗ Come by Istanbul. (route → via)
- ✓ Come **via** Istanbul.

- ✗ Got job via friend. (intermediary → through)
- ✓ Got the job **through** a friend.

## Xatırlatma

| Söz | Bir sözdə |
|-----|-----------|
| **by** | metod / agent |
| **via** | marşrut / rəsmi vasitə |
| **through** | proses / intermediary |

**Simple rules:**
- Default → **by**
- Route / platform → **via**
- Intermediary / process → **through**

**Tech:**
- **By** method (pay by card)
- **Via** system (via API)
- **Through** intermediary (through proxy)
