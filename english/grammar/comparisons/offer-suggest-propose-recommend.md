# Offer / Suggest / Propose / Recommend — Təklif Fellər

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Kontekst | Struktur |
|-----|----------|----------|
| **offer** | direkt təklif (vermək) | offer + N / to + V |
| **suggest** | fikir təklif | suggest + V-ing / that |
| **propose** | rəsmi təklif | propose + N / that |
| **recommend** | tövsiyə (başqasına) | recommend + N / V-ing |

> **Qısa qayda:**
> - **offer** = vermək təklifi
> - **suggest** = fikir təklif et (neytral)
> - **propose** = rəsmi təklif (plan, evlilik)
> - **recommend** = tövsiyə et (təcrübədən)

→ Related (partly covered): [suggest-recommend-advise.md](suggest-recommend-advise.md)

---

## 1. Offer — Təklif Etmək (Vermək)

Kimsəyə bir şey vermək / etmək təklif.

### Struktur

- **offer** + N
- **offer** + to + V
- **offer** + N + N (double object)

### Nümunələr

- I **offered** him a job.
- She **offered** to help.
- Can I **offer** you a coffee?
- **Offer** free support.

### İsim

- A job **offer**.
- Counter**offer**.
- Special **offer**.

### Offer vs Suggest

- **offer** = vermək / etmək (konkret)
- **suggest** = fikir

- "I **offered** to drive." (mən edəcəm)
- "I **suggested** we drive." (fikir verdim)

---

## 2. Suggest — Təklif Etmək (Fikir)

Fikir verirsən, amma sən icra etməyə məcbur deyilsən.

### Struktur

- **suggest** + V-ing
- **suggest** + that + clause
- **suggest** + N

### ⚠ Yanlış struktur

- ✗ suggest me to go
- ✓ **suggest** (to me) that I go
- ✓ **suggest** going

**Suggest** direct object + infinitive qəbul etmir!

### Nümunələr

- I **suggest** trying again.
- She **suggested** that we refactor.
- May I **suggest** an alternative?
- **Suggest** a meeting time.

---

## 3. Propose — Rəsmi Təklif

Rəsmi təklif / plan / evlilik.

### Struktur

- **propose** + N
- **propose** + to + V
- **propose** + that + clause

### Nümunələr

- **Propose** a new design.
- He **proposed** to her. (evlilik!)
- I **propose** we start early.
- **Propose** a toast.

### Propose vs Suggest

- **propose** = rəsmi, yazılı, hazır plan
- **suggest** = fikir, informal

- "I **suggest** we meet tomorrow." (fikir)
- "I **propose** we postpone the meeting." (rəsmi qərar)

### Tech kontekst

- **Propose** a solution (design doc).
- **Propose** a new architecture.
- **Proposal** = təklif sənədi.

### Evlilik

- He **proposed** (to her). = təklif etdi.
- **Marriage proposal**.

---

## 4. Recommend — Tövsiyə (Təcrübədən)

Bildiyin / təcrübəndən başqalarına tövsiyə.

### Struktur

- **recommend** + N
- **recommend** + V-ing
- **recommend** + that + clause

### ⚠ Yanlış

- ✗ recommend me to try
- ✓ **recommend** trying
- ✓ **recommend that** I try

Same as "suggest" — no direct object + infinitive.

### Nümunələr

- I **recommend** this book.
- He **recommends** using TypeScript.
- She **recommended** that I apply.

### Suggest vs Recommend

- **suggest** = fikir (dəyərləndirmə zəif)
- **recommend** = tövsiyə (təcrübə əsasında)

- "I **suggest** trying Python." (fikir)
- "I **recommend** Python." (təcrübəm var, emin)

### Tech interview

- "I **recommend** microservices for this case." ✓ (ekspert səslənir)

## Müqayisə — Bir Cümlə

### Offer (vermək)

"I **offer** you a new job."
→ Konkret iş təklifi.

### Suggest (fikir)

"I **suggest** you apply."
→ Fikir: müraciət et.

### Propose (rəsmi)

"I **propose** a partnership."
→ Rəsmi iş təklifi.

### Recommend (tövsiyə)

"I **recommend** this company."
→ Təcrübəmdən: yaxşı şirkətdir.

| Feil | + N | + V-ing | + to + V | + that |
|------|-----|---------|----------|--------|
| **offer** | ✓ | ✗ | ✓ | ✗ |
| **suggest** | ✓ | ✓ | ✗ | ✓ |
| **propose** | ✓ | ✓ | ✓ | ✓ |
| **recommend** | ✓ | ✓ | ✗ | ✓ |

**Diqqət:** Yalnız **propose** "propose to do" strukturunu alır.

## Azərbaycanlı Səhvləri

- ✗ I suggested him to leave.
- ✓ I **suggested that he leave**.
- ✓ I **suggested** he **leave**.

(Subjunctive: "he leave" — no -s!)

- ✗ I recommended him to try.
- ✓ I **recommended** that he **try**. / **recommended trying**.

- ✗ I propose to change the design.
- ✓ I **propose** changing the design. / **propose that we change**...

(Although "propose to V" is technically correct too — "propose to build a bridge" — it's less common)

---

## Subjunctive Form (Advanced)

"Suggest", "recommend", "propose" + that + subject + **base form** (no -s):

- I suggest that he **go**. (not goes!)
- I recommend that she **take** the exam.
- I propose that the meeting **be** rescheduled.

Bu forma rəsmi yazıda istifadə olunur.
