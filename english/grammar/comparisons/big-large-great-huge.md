# Big / Large / Great / Huge / Enormous — Böyük Sifətləri

## Səviyyə
A2-B1

---

## Əsas Cədvəl

| Söz | Nüans | Tonu |
|-----|-------|------|
| **big** | informal, ümumi | gündəlik |
| **large** | formal, ölçü/kəmiyyət | yazılı, rəsmi |
| **great** | böyük + əhəmiyyətli | emotional, əla |
| **huge** | çox böyük | intensive |
| **enormous** | nəhəng | daha güclü |
| **massive** | çox böyük + ağır | britsh, impressive |
| **giant** | nəhəng (sifət) | colloquial |

> **Qısa qayda:**
> - **big** = ümumi böyük
> - **large** = rəsmi böyük (ölçü)
> - **great** = əla / əhəmiyyətli (emosional)
> - **huge / enormous / massive / giant** = çox böyük

---

## 1. Big — Gündəlik Böyük

Ən çox işlədilir. İnformal.

### Nümunələr

- A **big** house.
- **Big** decision.
- **Big** problem.
- **Big** mistake.
- **Big** idea.
- **Big** company.
- My **big** brother.

### İfadələr

- **big deal** = böyük məsələ
- **big picture** = ümumi mənzərə
- **big time** = çox
- **big shot** = mühüm adam
- **big data** = böyük verilənlər

---

## 2. Large — Rəsmi / Ölçü

Ölçü, kəmiyyət üçün. Daha rəsmi.

### Nümunələr

- A **large** building.
- **Large** population.
- **Large** dataset.
- **Large** quantity.
- **Large** number of users.

### Big vs Large

Əksər hallarda sinonim, lakin:

| Kontekst | Söz |
|----------|-----|
| gündəlik | big |
| rəsmi yazı | large |
| ölçülərdə | large |
| emosional | big |

- "A **big** problem." ✓ (gündəlik)
- "A **large** number of issues." ✓ (rəsmi)
- ~~A large problem~~ — gərgin səslənir

### T-shirt size, coffee size

- Small / Medium / **Large** / XL → Large istifadə olunur

---

## 3. Great — Əla / Əhəmiyyətli

İki ana məna:

### A. Çox böyük / əhəmiyyətli (abstract)

- **Great** success.
- **Great** idea!
- **Great** achievement.
- **Great** power.
- The **Great** Wall of China.

### B. Əla / çox yaxşı

- **Great** job!
- This is **great**.
- **Great** to see you.
- **Great** news.

### Great vs Big

- **big** = fiziki ölçü
- **great** = əhəmiyyət / keyfiyyət

- A **big** company → ölçü
- A **great** company → əla (reputation)

---

## 4. Huge — Çox Böyük

**Big**-dən böyük. Emosional.

### Nümunələr

- A **huge** crowd.
- **Huge** success.
- **Huge** impact.
- **Huge** difference.
- **Huge** fan of the team.

### Big vs Huge

- big = böyük
- huge = çox böyük (emotional)

- A **big** improvement. (qeyd)
- A **huge** improvement. (impressed)

---

## 5. Enormous — Nəhəng

**Huge**-dən bir az güclü. Yazılı / rəsmi.

### Nümunələr

- **Enormous** amount of data.
- **Enormous** effort.
- **Enormous** challenge.
- **Enormous** benefit.

### Huge vs Enormous

Çox yaxın, lakin:
- **huge** = gündəlik, emosional
- **enormous** = daha rəsmi, yazılı

---

## 6. Massive — Çox Böyük + Ağır

British English favorite. Impressive.

### Nümunələr

- **Massive** update.
- **Massive** success.
- **Massive** building.
- **Massive** impact.

### British slang

- "That's **massive**!" = super!

---

## 7. Giant — Nəhəng (Colloquial)

### Sifət kimi

- A **giant** company.
- **Giant** step forward.
- Tech **giants** (FAANG).

### İsim kimi

- Tech **giants** = Google, Apple, etc.

---

## İntensivlik Ardıcıllığı

```
big < large ≤ huge ≤ massive ≈ enormous < giant / gigantic
```

Amma gündəlik söhbətdə "big" və "huge" kifayətdir.

4. That was a ______ mistake. (gündəlik)
5. Google is a tech ______ . (colloquial)

**Cavablar:** 1. huge/great, 2. large/enormous, 3. massive/enormous, 4. big, 5. giant

## Tech / İş Kontekstində

- A **big** refactor. ✓ (gündəlik)
- A **large** codebase. ✓ (rəsmi)
- A **huge** performance improvement. ✓ (emotional)
- A **massive** update. ✓ (British-friendly)
- Tech **giants**: Google, Meta, Amazon. ✓

---

## Azərbaycanlı Səhvləri

- ✗ A large problem. (emosional — big)
- ✓ A **big** problem.

- ✗ Huge amount of data. (OK, amma formal yazıda → enormous/large)
- ✓ An **enormous** / **large** amount of data.

---

## Xatırlatma

- **big** = ümumi (daily speak)
- **large** = rəsmi / ölçü
- **great** = əla / əhəmiyyətli (emosional)
- **huge** = çox böyük (gündəlik)
- **enormous / massive** = daha güclü, rəsmi
