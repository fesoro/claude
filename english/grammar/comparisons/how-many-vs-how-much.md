# How Many vs How Much — Sayılan vs Sayılmayan

## Səviyyə
A2-B1 (əsas qayda, amma tələ var!)

---

## Əsas Fərq

| İfadə | İstifadə |
|-------|----------|
| **how many** | countable (sayılan) |
| **how much** | uncountable (sayılmayan) + qiymət |

> **Qısa qayda:**
> - **how many** = sayılan şeylər (books, people)
> - **how much** = sayılmayan (water, time, money)
> - **how much** = həm də qiymət sorğusu

---

## 1. How Many — Countable

Sayıla bilən isimlər.

### Nümunələr

- **How many** books?
- **How many** people came?
- **How many** users?
- **How many** bugs?

### Plural noun ilə

- **how many** + plural noun

- **How many** cars? (plural)
- **How many** cups?

### Cavab

- "Two." / "A few." / "Several." / "Many."

---

## 2. How Much — Uncountable

Sayıla bilməyən isimlər (su, vaxt, pul).

### Nümunələr

- **How much** water?
- **How much** time?
- **How much** money?
- **How much** data?

### Singular noun ilə

- **how much** + uncountable noun

- **How much** coffee? (uncountable)
- **How much** homework?

### Cavab

- "A little." / "Some." / "A lot." / "None."

---

## 3. How Much — Qiymət Sorğusu

"Nə qədərdir?" — bu məqsədlə.

### Nümunələr

- **How much** is it?
- **How much** does it cost?
- **How much** for this?
- **How much** is rent?

Buradan sadə qayda: pul həmişə "much".

---

## Kritik Siyahı (Tech + Daily)

### Countable (→ how many)

- users, customers, clients
- bugs, issues, tickets
- PRs, commits
- tests
- servers, instances
- files, folders
- lines (of code) → ambiguous, "lines" say.

### Uncountable (→ how much)

- time, money, work
- traffic, bandwidth
- data, information
- memory, storage
- knowledge, experience
- feedback
- code (bütöv kod, not lines)

---

## 4. Tricky Cases

### "Data"

- AmE: "**How much** data?" (uncountable)
- BrE formal: "**How many** data?" (plural — data is datum pl.)

Gündəlikdə: **how much data**.

### "People"

- **People** = plural countable → **how many**!
- ~~how much people~~ ✗
- ✓ **How many** people?

### "Work"

- **Work** = uncountable usually → **how much**
- **How much** work do you have?

But:
- **Work** (country works, publications) = countable
- **How many** works of Shakespeare?

### "Information"

- **Information** = uncountable → **how much**
- **How much** information?
- ~~how many informations~~ ✗

### "Advice"

- **Advice** = uncountable → **how much**
- **How much** advice?

## Sayılan vs Sayılmayan — Ümumi Siyahı

### Countable (sayıla bilən)

people, books, cars, tests, bugs, users, apples, questions, ideas, jobs, laptops, windows

### Uncountable (sayıla bilməz)

water, milk, time, money, information, advice, knowledge, traffic, love, music, news, work, homework, furniture, luggage

### Diqqət — Tələlər

- **news** = uncountable (~~a news~~)
- **advice** = uncountable
- **furniture** = uncountable (~~furnitures~~)
- **equipment** = uncountable
- **information** = uncountable

## Struktur Qaydası

### How many/much + noun + verb + subject?

- **How many** users do you have?
- **How much** memory do we need?
- **How much** does it cost?

### Short questions

- **How many**?
- **How much**?

- "I have apples." — "**How many**?"
- "I need coffee." — "**How much**?"

---

## Approximate Answers

### For how many

- a few, some, several
- many, lots of
- dozens of, hundreds of

### For how much

- a little, some
- a lot of, plenty of
- tons of (informal)

### Universal (both)

- none
- a lot of
- plenty of
- loads of (informal)

---

## Tech Interview Nümunələri

- "**How many** users does the app have?" → "10M active users."
- "**How much** traffic do you handle?" → "1B requests/day."
- "**How many** engineers in the team?" → "8 engineers."
- "**How much** downtime did we have?" → "2 minutes."

## Azərbaycanlı Səhvləri

- ✗ How much bugs are there?
- ✓ **How many** bugs are there? (sayılan!)

- ✗ How many memory we have?
- ✓ **How much** memory we have?

- ✗ How many traffics?
- ✓ **How much** traffic? (traffic = uncountable)

- ✗ How many peoples? (people zatən plural)
- ✓ **How many** people?

- ✗ How many informations? (uncountable)
- ✓ **How much** information?

## Related Sözlər

### A lot / A lot of

İkisində də işləyir:
- **A lot of** users (countable)
- **A lot of** time (uncountable)

### Few vs Little

- **few** = sayılan (few bugs)
- **little** = sayılmayan (little time)

→ [few-little-much-many.md](few-little-much-many.md)

### Some / Any

- Hər ikisində işləyir.

## "How Long" / "How Often" Bonus

### How long

Müddət.
- **How long** does the build take?
- **How long** have you worked here?

### How often

Tezlik.
- **How often** do you deploy?
- **How often** do standups happen?

### How far

Məsafə.
- **How far** is the office?

---

## Xatırlatma

| Question | Noun tipi |
|----------|-----------|
| **How many** | countable (users, bugs) |
| **How much** | uncountable (time, money, data) |
| **How much** (price) | pul, qiymət |

**Test metodu:** Noun-a "-s" qoya bilərsən? → **how many**. Yox? → **how much**.

**Tech diqqət:** data, memory, traffic, storage = **how much**.
