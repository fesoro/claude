# On vs Upon — Daily vs Rəsmi

## Səviyyə
B1

---

## Əsas Fərq

| Söz | Tonu |
|-----|------|
| **on** | ən çox istifadə (daily) |
| **upon** | rəsmi / ədəbi (formal) |

> **Qısa qayda:**
> - **on** = standart (hər yerdə)
> - **upon** = rəsmi "on" (xüsusi ifadələr)
> - **Çox ehtiyatlı:** upon sadəcə köhnə "on" deyil!

---

## 1. On — Standart (Daily)

Hər yerdə istifadə.

### Fiziki

- Book **on** the table.
- **On** the floor.

### Zaman

- **On** Monday.
- **On** my birthday.

### Mövzu

- Article **on** AI.
- Book **on** Python.

### Tech

- Click **on** the button.
- Run **on** Linux.

→ Ətraflı istifadələr: [in-on-at-place.md](in-on-at-place.md), [in-on-at-time.md](in-on-at-time.md)

---

## 2. Upon — Rəsmi / Xüsusi İfadələr

**Upon** çox spesifik kontekstlərdə istifadə olunur — çox az.

### A. Rəsmi "on"

- **Upon** arrival, please register.
- **Upon** completion.
- **Upon** request.

### B. "As soon as" mənası

- **Upon** hearing the news...
- **Upon** finishing, we celebrated.

### C. Fixed phrases

- Once **upon** a time. (story opening)
- **Upon** my word. (old-fashioned)
- **Upon** reflection.

---

## 3. Fixed Upon Phrases

### Common Business / Legal

- **Upon** request (xahişlə)
- **Upon** arrival (gələndə)
- **Upon** completion (bitirikdə)
- **Upon** approval (təsdiqlə)
- **Upon** receipt (alınanda)
- **Upon** delivery (çatdırılanda)
- **Upon** cancellation (ləğv olduqda)

### Contract language

- "Payment is due **upon** receipt."
- "**Upon** termination of employment..."

### Legal documents

Çox işlənir.

---

## 4. "Once Upon a Time"

Nağıl başlanğıcı (ədəbi).

- **Once upon a time**, there was a princess.

Yalnız ədəbi. Gündəlikdə yox.

---

## 5. Upon + V-ing

"As soon as" strukturu.

### Nümunələr

- **Upon arriving**, he called.
- **Upon hearing** the news, she smiled.
- **Upon reflection**, I agree.

### Alternative

- "**On** arrival" ✓ (daha standart)
- "**Upon** arrival" ✓ (rəsmi yazı)

---

## 6. Verbs + On / Upon

### Fixed expressions

- **rely on** someone (not upon)
- **depend on** / **upon** (both OK, on more common)
- **agree on** / **upon** (both OK)
- **insist on** / **upon** (both OK)

### ⚠ Use "on" for daily

- **rely on** (standart)
- **depend on** (standart)

### "Upon" alternative

Çox formal / poetic yazıda "rely upon, depend upon."

Gündəlik: **on** istifadə et.

---

## On vs Upon — İstifadə Cədvəli

| Kontekst | On | Upon |
|----------|----|----|
| Fiziki (table) | ✓ | ✗ |
| Zaman (Monday) | ✓ | ✗ |
| Mövzu (article on AI) | ✓ | ✗ |
| Rəsmi "upon arrival" | ✓ | ✓ |
| "As soon as" (upon V-ing) | ✓ (on V-ing) | ✓ |
| Once upon a time | ✗ | ✓ (fixed) |
| Verbs (rely, depend) | ✓ | ✓ (formal) |

## Tech / İş Kontekstində

### Daily (on)

- "Click **on** the button." ✓
- "Run **on** Docker." ✓
- "Article **on** Kubernetes." ✓
- "Rely **on** cache." ✓

### Formal yazı (upon)

- "**Upon** receipt of the request..." ✓ (rəsmi email)
- "Action required **upon** approval." ✓
- "**Upon** completion of the task..." ✓

### Legal tech docs

- "**Upon** termination..."
- "**Upon** signing the agreement..."

---

## Interview Kontekstində

### Formal:

- "**Upon** receiving feedback, I..." ✓
- "**Upon** completion of the project..." ✓

### Casual:

- "**On** receiving feedback, I..." ✓ (same)

**Hər ikisi OK — upon formal.**

---

## "Upon" Nə Vaxt İstifadə Edim?

### Use "upon"

✓ Rəsmi yazı (contracts, legal)
✓ Fixed phrases (once upon a time)
✓ Ədəbi (poetic)
✓ "Upon V-ing" = "As soon as"

### Use "on"

✓ Hər yerdə — daily, tech, casual.
✓ Day-to-day communication.
✓ Standart writing.

**Qızıl qayda:** Şübhədə → **on**.

---

## Azərbaycanlı Səhvləri

- ✗ Upon the table. (daily → on)
- ✓ **On** the table.

- ✗ Upon Monday. (daily → on)
- ✓ **On** Monday.

- ✗ Article upon AI. (mövzu → on)
- ✓ Article **on** AI.

**Upon sadəcə "fancy on" deyil — spesifik kontekst!**

---

## "Upon" Overuse

Bəzi insanlar bütün yerdə "upon" işlədir — çox rəsmi görünür, pretentious.

### ✗ Too much upon

- "Click **upon** the button." (qərib)
- "Based **upon** the data." (OK amma "on" daha yaxşı)

### ✓ Natural

- "Click **on** the button."
- "Based **on** the data."

---

## Fixed Expressions — Memory

Bu bir neçəsini yadda saxla:

1. **Once upon a time** — nağıl
2. **Upon arrival** — gələndə
3. **Upon request** — xahişlə
4. **Upon completion** — bitirikdə
5. **Upon receipt** — alınanda
6. **Upon approval** — təsdiqlə

Digər yerlərdə — **on**.

---

## Xatırlatma

| Söz | Nə vaxt? |
|-----|----------|
| **on** | hər yerdə (daily) |
| **upon** | rəsmi yazı + fixed phrases |

**Qızıl qayda:** 
- Daily → **on**
- Formal writing → **upon** (fixed phrases)
- Şübhədə → **on**

**Tech:** Əksər hallarda **on** istifadə et.
