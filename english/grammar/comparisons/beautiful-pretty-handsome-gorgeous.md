# Beautiful / Pretty / Handsome / Gorgeous / Cute — Gözəllik Sifətləri

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Kim / nə? | Dərəcə |
|-----|-----------|--------|
| **cute** | uşaqlar, heyvanlar, yüngül | şirin |
| **pretty** | qadınlar / şeylər | orta gözəllik |
| **beautiful** | hər kəs / hər şey | güclü |
| **handsome** | kişilər (ənənə) | güclü |
| **gorgeous** | çox gözəl | çox güclü |
| **stunning** | nəfəskəsici | ən güclü |
| **attractive** | cəlbedici | ümumi |

> **Qısa qayda:**
> - **cute** = şirin (uşaqlar, kiçik şeylər)
> - **pretty** = gözəl (yüngül, qadınlar)
> - **beautiful** = gözəl (universal)
> - **handsome** = yaraşıqlı (kişilər)
> - **gorgeous / stunning** = çox gözəl

---

## 1. Cute — Şirin

Kiçik, şirin, sevimli. Uşaqlar, heyvanlar, kiçik şeylər.

### Nümunələr

- A **cute** baby.
- **Cute** puppy.
- **Cute** dress.
- How **cute**!

### Kontekst

- kiçik uşaqlar
- heyvanlar (bala)
- kiçik paltar / aksesuar
- informal emosional

### ⚠ Diqqət

**Cute** gənc kişi / qadına deyilə bilər (maraqlı kontekst):
- A **cute** guy. (gənc, sevimli — flirty)

---

## 2. Pretty — Gözəl (Yüngül)

Qadınlar, qız uşaqlar, gözəl şeylər.

### Nümunələr

- A **pretty** girl.
- **Pretty** flowers.
- **Pretty** dress.
- **Pretty** scenery.

### Pretty vs Beautiful

- **pretty** = sevimli, xoş görünən (yüngül)
- **beautiful** = tam gözəl (güclü)

- A **pretty** girl. (sevimli)
- A **beautiful** woman. (tam gözəl)

### Pretty həm də zərf

- **pretty** = olduqca (zərf)
- **pretty** good = kifayət yaxşı
- **pretty** easy = olduqca asan

---

## 3. Beautiful — Gözəl (Universal)

Hər kəs və hər şey üçün. Güclü.

### Nümunələr

- A **beautiful** woman.
- A **beautiful** sunset.
- **Beautiful** music.
- **Beautifully** designed.
- **Beautiful** view.

### Universal istifadə

- Insanlar ✓
- Yerlər ✓
- Sənət ✓
- Musiqi ✓
- Təbiət ✓

### Gender

Ənənəvi olaraq "beautiful" qadınlar üçün, "handsome" kişilər üçün. Amma modern ingiliscədə "beautiful man" da OK.

---

## 4. Handsome — Yaraşıqlı (Kişilər)

Ənənəvi kişi gözəlliyi.

### Nümunələr

- A **handsome** man.
- **Handsome** actor.
- Tall, dark, and **handsome**.

### Modern istifadə

Bəzən qadınlar üçün də istifadə edilir (güclü, xarizmatik):
- A **handsome** woman. (güclü cəzbedicilik)

### Abstract istifadə

- A **handsome** reward. (böyük mükafat)
- A **handsome** profit.

---

## 5. Gorgeous — Çox Gözəl

Beautiful-dan güclü. Emosional.

### Nümunələr

- You look **gorgeous**!
- A **gorgeous** dress.
- **Gorgeous** view.
- Absolutely **gorgeous**.

### Pretty vs Beautiful vs Gorgeous

- **pretty** = sevimli
- **beautiful** = gözəl
- **gorgeous** = heyrətamiz gözəl

---

## 6. Stunning — Nəfəskəsici (Ən Güclü)

İnsanı şok edəcək qədər gözəl.

### Nümunələr

- **Stunning** view!
- A **stunning** dress.
- **Absolutely stunning**.
- **Stunning** performance.

### Kontekst

- heyrətamiz gözəllik
- çox yaxşı performans

### Stunning vs Gorgeous

Demək olar eyni. **Stunning** biraz formal.

---

## 7. Attractive — Cəlbedici (Neytral)

Ümumi cazibə. Neytral.

### Nümunələr

- An **attractive** person.
- **Attractive** design.
- **Attractive** offer.

### Attractive vs Beautiful

- **beautiful** = fiziki gözəllik
- **attractive** = cəlbedici (ümumi — şəxsiyyət + görünüş + fikir)

- **Attractive** salary. ✓ (cəlbedici təklif)
- ~~Beautiful salary~~ ✗

---

## Bonus Sözlər

### Lovely

Şirin + gözəl. British-də çox.
- A **lovely** day.
- **Lovely** meeting you.

### Adorable

Çox şirin (cute-dən güclü).
- **Adorable** kitten.

### Charming

Cəlbedici şəxsiyyət.
- A **charming** man.

### Elegant

Zövqlü gözəllik.
- **Elegant** dress.

### Good-looking

Informal "handsome / beautiful".
- A **good-looking** guy.

## Tech / İş Kontekstində

### Beautiful (rarely used)

- **Beautiful** code. ✓ (təmiz)
- **Beautifully** designed API. ✓

### Elegant (tech favorite)

- **Elegant** solution. ✓ (clever + təmiz)
- **Elegant** code. ✓

### Attractive

- **Attractive** offer. ✓ (cəlbedici)
- **Attractive** salary. ✓
- **Attractive** benefits. ✓

### Clean (tech-də gözəllik)

- **Clean** code. (gözəl, aydın)
- **Clean** architecture.

---

## İnterview Kontekstində

Fiziki gözəllik ilə söhbət etmə. Amma:

- "An **attractive** opportunity." ✓ (cəlbedici)
- "An **elegant** solution." ✓ (kod)
- "A **beautiful** codebase." ✓ (təmiz)

---

## Azərbaycanlı Səhvləri

- ✗ A beautiful man. (handsome daha təbii)
- ✓ A **handsome** man. (ənənəvi)

Modern olmaq istəyirsənsə:
- ✓ A **beautiful** man. ✓ (modern, genderless)

- ✗ A cute mountain. (dağa → beautiful / majestic)
- ✓ A **beautiful** / **majestic** mountain.

- ✗ A pretty man. (qərib — "yaraşıqlı" demirsən)
- ✓ A **handsome** / **good-looking** man.

---

## Complimenting Etiquette

### Professional

- "That's a **beautiful** presentation." ✓
- "You look **great** today." ✓ (safe)
- "Nice haircut!" ✓ (specific)

### ✗ Avoid at work

- ~~You're gorgeous~~ (too personal)
- ~~You're cute~~ (too personal)

---

## Gender Note

Modern ingiliscədə "beautiful", "gorgeous" kişilər üçün də adi haldır. "Handsome" qadınlar üçün güclü bir complimentdir.
