# Increase / Rise / Grow / Improve — Artmaq, Böyümək, Yaxşılaşmaq

## Səviyyə
B1-B2

---

## Əsas Cədvəl

| Söz | Məna | Keçiricilik | Kontekst |
|-----|------|-------------|---------|
| **increase** | artmaq | transitiv + intransitiv | ümumi, rəqəm |
| **rise** | qalxmaq | yalnız intransitiv | formal, qiymət, səviyyə |
| **grow** | böyümək | əsasən intransitiv | tədricən, zaman içrə |
| **improve** | yaxşılaşmaq | transitiv + intransitiv | keyfiyyət (miqdar deyil!) |

> **Qısa qayda:**
> - **increase** = artmaq (miqdar / rəqəm)
> - **rise** = qalxmaq (formal, yalnız intransitiv)
> - **grow** = böyümək (tədricən, zaman içrə)
> - **improve** = yaxşılaşmaq (daha çox deyil, daha **yaxşı**)

---

## 1. Increase — Artmaq (Miqdar)

Ən universal söz. Həm "biz artırdıq" (transitiv), həm "artdı" (intransitiv) mənasında.

### Struktur

- **increase** + N → "we increased the budget" (transitiv — biz artırdıq)
- N + **increase** → "the budget increased" (intransitiv — artdı)
- **increase by** + % / rəqəm → dəyişmə miqdarı
- **increase to** + dəyər → hədəf dəyər

### Nümunələr

| İfadə | Nümunə | Azərbaycanca |
|-------|--------|-------------|
| increase + N | "We increased the limit to 1000 req/s." | limiti artırdıq |
| N + increase | "Response time increased by 30%." | cavab müddəti artdı |
| increase by | "Errors increased by 5x after deploy." | 5 dəfə artdı |
| increase to | "Traffic increased to 10k RPS." | 10k RPS-ə çatdı |
| significant increase | "A significant increase in memory usage." | yaddaş istifadəsi artdı |

### Tech kontekst

- "CPU usage **increased** after the migration."
- "We **increased** the connection pool size."
- "Request count **increased by** 40% overnight."

---

## 2. Rise — Qalxmaq (Formal, Yalnız Intransitiv)

**Yalnız intransitiv** — subyekt özü qalxır, başqası qaldırmır.

"we rose the price" ✗ — rise intransitiv olduğuna görə bu yanlışdır!
"we raised the price" ✓ — raise = transitiv feli

### Struktur

- N + **rise** → "prices rose" (keçmiş — rose)
- N + **is rising** → "costs are rising" (davam edir)
- N + **has risen** → "the level has risen" (nəticə var)

### Nümunələr

| İfadə | Nümunə | Azərbaycanca |
|-------|--------|-------------|
| prices rose | "Server costs rose sharply in Q4." | xərclər artdı |
| is rising | "Traffic is rising steadily." | artmaqdadır |
| has risen | "Error rate has risen to 5%." | artıb (nəticə) |
| rise in | "A rise in latency after update." | gecikmə artımı |

### Rise vs Raise (mühüm fərq!)

| | Rise | Raise |
|-|------|-------|
| Növ | intransitiv | transitiv |
| Mənası | özü qalxır | kimisə/nəyisə qaldırır |
| Nümunə | "Costs rose." | "We raised prices." |

- ✗ "We **rose** the salary." (yanlış — intransitiv!)
- ✓ "We **raised** the salary." (düzgün)
- ✓ "Salaries **rose** this year." (düzgün)

### Tech kontekst

- "Latency **rose** after the config change."
- "Memory usage **has risen** 20% this week."
- "Error rates **are rising** — check the logs."

---

## 3. Grow — Böyümək (Tədricən, Zaman İçrə)

Tədricən inkişaf. Bitkinin böyüməsi kimi — zaman içrə, üzvi şəkildə.

### Struktur

- N + **grow** → "the team is growing"
- **grow by** + % → "grew by 200%"
- **grow into** → "grew into a platform"
- **grow from** X **to** Y → başlanğıc → son

### Nümunələr

| İfadə | Nümunə | Azərbaycanca |
|-------|--------|-------------|
| is growing | "Our user base is growing fast." | istifadəçi bazası böyüyür |
| grew by | "Revenue grew by 3x last year." | 3 dəfə böyüdü |
| grow into | "The startup grew into a platform." | platforma oldu |
| grow from | "Grew from 10 to 500 employees." | böyüdü |
| growing demand | "Growing demand for real-time APIs." | artan tələb |

### Tech kontekst

- "Our microservices architecture **grew** from a monolith."
- "The codebase has **grown** too large to maintain."
- "Tech debt **grows** when you skip refactoring."

---

## 4. Improve — Yaxşılaşmaq (Keyfiyyət!)

**Ən vacib fərq:** improve = daha çox deyil, daha **yaxşı**.
Miqdar artmır — keyfiyyət yüksəlir.

### Struktur

- **improve** + N → "we improved performance" (transitiv)
- N + **improve** → "performance improved" (intransitiv)
- **improvement in** + area → "improvement in quality"

### Nümunələr

| İfadə | Nümunə | Azərbaycanca |
|-------|--------|-------------|
| improve performance | "We improved query performance by 50%." | performans yaxşılaşdı |
| improve quality | "Code quality improved after refactoring." | keyfiyyət yüksəldi |
| improve skills | "I want to improve my English." | inkişaf etdirmək |
| room for improvement | "There's room for improvement in the API." | yaxşılaşma üçün yer var |
| significant improvement | "Significant improvement in load time." | böyük irəliləyiş |

### Improve vs Increase (ən çox qarışdırılan!)

| | Improve | Increase |
|-|---------|----------|
| Sual | Daha yaxşı? | Daha çox? |
| Nümunə | "Quality improved." | "Quantity increased." |
| Fokus | keyfiyyət | miqdar |

- "We need to **improve** performance." → daha sürətli, daha etibarlı (yaxşılaşdır)
- "We need to **increase** performance." → bu cümlə texniki yazıda nadir — adətən "improve" işlənir
- "We need to **increase** throughput." → miqdar artımı (request sayı, MB/s)

### Tech kontekst

- "We **improved** query time from 2s to 200ms."
- "Code readability **improved** after the review."
- "We're working to **improve** the developer experience."
- "Test coverage **improved** from 40% to 85%."

## Tez-tez Yanlış İşlənənlər

| ❌ Yanlış | ✅ Düzgün |
|-----------|-----------|
| "We rose the price." | "We **raised** the price." |
| "The app improved to 10k users." | "The app **grew to** 10k users." |
| "Errors are improving." | "Error rate is **decreasing**." |
| "We increased the code quality." | "We **improved** the code quality." |
| "Performance is growing." | "Performance is **improving**." |
| "The budget is improving." | "The budget is **increasing**." |
| "Latency increased to a better level." | "Latency **improved** / **decreased**." |
| "We need to grow our response time." | "We need to **reduce** / **improve** response time." |

## Məşq Tapşırıqları

### 1. Doğru sözu seçin (increase / rise / grow / improve)

1. Server costs ______ by 40% last month. (miqdar artdı)
2. We need to ______ our test coverage. (yaxşılaşdır)
3. The startup ______ from 5 to 200 employees. (tədricən böyüdü)
4. Error rates ______ after the deployment. (qalxdı — formal)
5. Response time ______ significantly after the refactor. (yaxşılaşdı)
6. We ______ the timeout limit to 30 seconds. (biz artırdıq)
7. Tech debt ______ when you don't refactor. (böyüyür, tədricən)
8. The number of API calls ______ to 1 million per day. (artdı)
9. Code readability ______ after the review. (yaxşılaşdı)
10. Memory usage ______ sharply during peak hours. (qalxdı — formal)

**Cavablar:** 1. increased, 2. improve, 3. grew, 4. rose, 5. improved, 6. increased, 7. grows, 8. increased, 9. improved, 10. rose

### 2. Boşluğu doldurun

1. "We need to ______ the performance of our API — it's too slow." (yaxşılaşdır)
2. "Traffic has ______ by 300% since the product launch." (artdı)
3. "The codebase ______ too large to maintain without documentation." (böyüdü)
4. "Deployment frequency ______ after we adopted CI/CD." (yaxşılaşdı / artdı — hər ikisi olar)
5. "Prices ______ again — we need to optimize cloud costs." (qalxdı — formal)
6. "We ______ database connection limits to handle the load." (biz artırdıq)

**Cavablar:** 1. improve, 2. increased/grown, 3. grew, 4. improved/increased, 5. rose, 6. increased
