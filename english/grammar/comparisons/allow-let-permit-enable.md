# Allow / Let / Permit / Enable — İcazə Vermək Fellər

## Səviyyə
B1 (strukturlar vacib!)

---

## Əsas Cədvəl

| Söz | Kontekst | Tonu |
|-----|----------|------|
| **let** | informal | gündəlik, ən çox |
| **allow** | rəsmi | neytral/rəsmi |
| **permit** | çox rəsmi | rəsmi yazı, legal |
| **enable** | texnoloji / imkan | tech |

> **Qısa qayda:**
> - **let** = icazə ver (informal)
> - **allow** = icazə ver (neytral)
> - **permit** = icazə ver (rəsmi)
> - **enable** = imkan ver (tech / mümkün etmək)

---

## 1. Let — İnformal İcazə

Ən çox gündəlik söhbətdə.

### Struktur (diqqət!)

- **let** + someone + V (infinitive without "to")

### Nümunələr

- Let me **help** you.
- Let her **go**.
- Let's **start**.
- Let me **know**.
- My boss let me **leave** early.

### ⚠ Yanlış struktur

- ✗ Let me **to go**.
- ✓ Let me **go**.

"let" to infinitive istəmir!

### Forma

- let → let → let (heç dəyişmir!)
  - Yesterday, she **let** me in.
  - I've **let** you down.

---

## 2. Allow — Rəsmi İcazə

### Struktur (ikisi də var)

- **allow** + someone + to + V
- **allow** + V-ing (ümumi qayda)

### Nümunələr

- They **allow** us to work from home.
- Smoking is not **allowed**.
- The policy **allows** remote work.
- **Allow** me to introduce myself.

### Let vs Allow

- **let** → no "to" (let me go)
- **allow** → needs "to" (allow me to go)

- Let me **do** it. ✓
- Allow me **to do** it. ✓

Məna eynidir, struktur fərqli!

### Passive — çox istifadə olunur

- Smoking is not **allowed** here.
- Pets are **allowed**.
- You're **allowed** to leave.

---

## 3. Permit — Çox Rəsmi

Rəsmi / legal / yazılı.

### Struktur

- **permit** + someone + to + V
- **permit** + V-ing

### Nümunələr

- No cameras **permitted**.
- You are **permitted** to enter.
- Parking is not **permitted**.
- **Permit** me to say...

### İsim kimi — icazə sənədi

- Work **permit**.
- Driving **permit**.
- Building **permit**.

### Allow vs Permit

Demək olar eyni, lakin:
- **allow** = neytral
- **permit** = rəsmi / legal

- "Dogs aren't **allowed**." (daily)
- "Cameras aren't **permitted**." (sign / rəsmi)

Rəsmi yazıda **permit** üstün.

---

## 4. Enable — İmkan Vermək (Tech)

**Diqqət:** "icazə" deyil, "**mümkün etmək**".

### Nümunələr

- This feature **enables** fast search.
- The API **enables** developers to build apps.
- **Enable** JavaScript in your browser.
- **Enable** two-factor authentication.

### Struktur

- **enable** + N (feature-i işə sal)
- **enable** + someone + to + V (imkan ver)

- **Enable** dark mode. (feature)
- The tool **enables** us **to debug** faster. (imkan)

### Allow vs Enable

- **allow** = icazə (subyekt izn verir)
- **enable** = imkan (obyekt mümkün edir)

- "The manager **allowed** me to leave." (icazə)
- "The tool **enabled** me to debug faster." (imkan)

### Tech kontekst

- **Enable** feature flag.
- **Enabled** / **disabled** button.
- Feature toggle: enabled / disabled.

---

## Struktur Müqayisəsi (KRİTİK!)

| Feil | Struktur | Nümunə |
|------|----------|--------|
| **let** | let + sb + V | Let me **go**. |
| **allow** | allow + sb + to + V | Allow me **to go**. |
| **permit** | permit + sb + to + V | Permit me **to go**. |
| **enable** | enable + sb + to + V | Enable me **to go**. |

**Yalnız "let" "to" istəmir!**

## Tech / İş Kontekstində

- The system **allows** 100 concurrent users. ✓
- **Enable** HTTPS in production. ✓
- We **let** users customize their profile. ✓
- **Enabling** feature flags via config. ✓
- Pets are **allowed** in the office. ✓

---

## Azərbaycanlı Səhvləri

- ✗ Let me to go. (to yox!)
- ✓ **Let me go.**

- ✗ Allow me go. (to var!)
- ✓ **Allow me to go.**

- ✗ This enables to work faster. (subyekt lazım)
- ✓ This **enables us to** work faster.

---

## Phrasal Forms

### Let

- **let in** = daxil et
- **let out** = sər
- **let down** = üzgün et / məyus et
- **let go** = burax
- **let off** = buraxmaq (sərbəst et)

### Allow

- **allow for** = nəzərə almaq
  - **Allow for** some delay.

### Permit

- Çox phrasal yoxdur — rəsmidir.

### Enable

- **enable** feature X
- **disable** feature X (əksi)

## Sign Language (gündəlik işarələr)

- "No smoking **permitted**" ✓
- "Pets **allowed**" ✓
- "No photography **permitted**" ✓
