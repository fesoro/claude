# Hard / Difficult / Tough / Complex / Complicated — "Çətin" Sifətləri

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Nüans | Tonu |
|-----|-------|------|
| **hard** | çətin + zəhmətli | informal |
| **difficult** | çətin | neytral/rəsmi |
| **tough** | çətin + güclü | emotional, informal |
| **complex** | çox hissəli | rəsmi / tex |
| **complicated** | qarışıq | daily / daha neqativ |
| **challenging** | çətin + maraqlı | pozitiv |
| **tricky** | hiyləgər çətin | informal |

> **Qısa qayda:**
> - **hard / difficult** = çətin (hard = informal)
> - **tough** = çətin + emosional güc
> - **complex** = çox hissəli (neytral)
> - **complicated** = qarışıq (negativ çalar)
> - **challenging** = motivasion çətin
> - **tricky** = hiyləgər çətin

---

## 1. Hard — Informal Çətin

Ən çox gündəlik söhbətdə. **difficult**-dan daha informal.

### İki mənası

- **Hard** work (zəhmətli)
- **Hard** surface (bərk)

### Nümunələr

- It's **hard** to say.
- A **hard** question.
- **Hard** work.
- **Hard** day.
- **Hard** to believe.

### İfadələr

- **hard-working** = çalışqan
- **hard time** = çətin dövr
- **hard to please** = çətin razı salınan

### Hard vs Difficult

Əksər hallarda sinonim:
- "A **hard** question." = "A **difficult** question."

Fərq: **hard** gündəlik, **difficult** rəsmi.

---

## 2. Difficult — Neytral / Rəsmi Çətin

Yazılı, interview, rəsmi söhbət.

### Nümunələr

- **Difficult** task.
- **Difficult** situation.
- **Difficult** decision.
- **Difficult** customer.
- It's **difficult** to estimate.

---

## 3. Tough — Emotional Çətin

Emosional güc tələb edir. Adam + vəziyyət.

### Nümunələr

- A **tough** decision.
- **Tough** times.
- **Tough** boss.
- **Tough** call (çətin qərar).
- Hang in there, it's **tough**.

### Insan sifəti kimi

- He's a **tough** guy. = güclü / dayanaqlı
- **Tough** negotiator.

### Hard vs Tough

- **hard** = çətin (intellektual)
- **tough** = çətin (emosional / fiziki güc)

- A **hard** exam. (intellektual çətin)
- A **tough** year. (emosional çətin)

---

## 4. Complex — Çox Hissəli (Neytral)

Bir şeyin çox komponenti var. Neytral ton.

### Nümunələr

- **Complex** system.
- **Complex** problem.
- **Complex** algorithm.
- **Complex** relationship.
- **Complex** architecture.

### Complex vs Complicated

| Söz | Nüans |
|-----|-------|
| **complex** | çox hissəli ama anlaşılan |
| **complicated** | qarışıq və zəhmətli |

- A **complex** system (neytral — detallı).
- A **complicated** situation (negativ — baş ağrıdan).

- "This codebase is **complex**." (neytral — böyük)
- "This codebase is **complicated**." (negativ — qarışıq)

---

## 5. Complicated — Qarışıq (Negativ)

Qarışıq, zəhmətli, anlaması çətin.

### Nümunələr

- It's **complicated**.
- A **complicated** relationship.
- **Complicated** process.
- Don't make it **complicated**.

### Facebook status

- "It's **complicated**." = münasibət qarışıqdır

### Complex (yaxşı) vs Complicated (pis)

- "We built a **complex** system." (yaxşı: təfərrüatlı)
- "We built a **complicated** system." (pis: qarışıq)

---

## 6. Challenging — Çətin + Maraqlı (Pozitiv!)

Çətin amma dəyərli. İnterview-da istifadə et.

### Nümunələr

- A **challenging** project.
- **Challenging** role.
- **Challenging** bug.
- I'm looking for a **challenging** position.

### Hard vs Challenging

- **hard** = çətin (bəzən neqativ)
- **challenging** = çətin + motivasiya (pozitiv)

### Interview qızıl sözü

"What are you looking for?"
→ "A **challenging** role where I can grow."

---

## 7. Tricky — Hiyləgər Çətin

Zahiri sadə, lakin gizli çətinliklər. Informal.

### Nümunələr

- A **tricky** question.
- **Tricky** bug.
- **Tricky** situation.
- It's **tricky** to debug.

### Tricky vs Hard

- **hard** = birbaşa çətin
- **tricky** = gözlənilməyən çətinlik

- A **hard** math problem. (birbaşa çətin)
- A **tricky** edge case. (gizli problem)

## Test

Hansı söz daha uyğun?

1. Our ______ architecture includes 20 services. (neytral, çox hissəli)
2. That was a ______ bug — took 3 days. (gözlənilməz, hiyləgər)
3. I'm looking for a ______ role. (interview, pozitiv)
4. This is a ______ decision to make. (emosional)
5. Don't make the code so ______ . (qarışıq, negativ)

**Cavablar:** 1. complex, 2. tricky, 3. challenging, 4. tough/hard, 5. complicated

---

## İnterview Nümunələri

- "I thrive on **challenging** problems." ✓
- "We built a **complex** distributed system." ✓
- "I faced a **tough** decision about architecture." ✓
- "Debugging was **tricky** due to race conditions." ✓
- "The legacy code is **complicated**." ✓ (neqativ istəyirsən)

---

## Azərbaycanlı Səhvləri

- ✗ Our system is complicated. (əksi — rəsmi yazıda → complex)
- ✓ Our system is **complex**.

- ✗ It was a hard decision. (daha emosional → tough)
- ✓ It was a **tough** decision.

- ✗ I'm looking for a hard role. (pozitiv tərzdə → challenging)
- ✓ I'm looking for a **challenging** role.

---

## Xatırlatma

- **hard** = informal çətin
- **difficult** = rəsmi çətin
- **tough** = emosional çətin
- **complex** = çox hissəli (neytral/pozitiv)
- **complicated** = qarışıq (neqativ)
- **challenging** = interview qızıl sözü (pozitiv)
- **tricky** = hiyləgər çətin
