# Clever / Smart / Intelligent / Wise / Brilliant — Ağıllı Sifətləri

## Səviyyə
B1

---

## Əsas Cədvəl

| Söz | Nüans | Kontekst |
|-----|-------|----------|
| **smart** | ağıllı (daily) | ümumi, informal |
| **clever** | bacarıqlı, zirək | pratik çıxışlar |
| **intelligent** | ağıllı (rəsmi) | rəsmi, elmi |
| **wise** | müdrik | təcrübə / qərar |
| **brilliant** | parlaq ağıllı | heyran |
| **bright** | zəki (gənc) | təhsil kontekst |

> **Qısa qayda:**
> - **smart** = ağıllı (gündəlik)
> - **clever** = zirək, hiyləgər ağıllı
> - **intelligent** = rəsmi ağıllı
> - **wise** = müdrik (təcrübə)
> - **brilliant** = super ağıllı

---

## 1. Smart — Gündəlik Ağıllı

Ən çox işlənən. İnformal və pozitiv.

### Nümunələr

- She's really **smart**.
- A **smart** decision.
- **Smart** kid.
- That's **smart** thinking.
- **Smart** move.

### Tex kontekst

- **smart** phone
- **smart** contract
- **smart** TV
- **smart** home

Burada "smart" = "ağıllı" (technologic mənada).

### Diffenent regions

- **Smart** (AmE) = ağıllı
- **Smart** (BrE) = yaxşı geyinmiş + ağıllı

---

## 2. Clever — Zirək / Bacarıqlı

Sürətli düşünür, problem həll edir. Bəzən hiyləgər çalarla.

### Nümunələr

- **Clever** solution.
- **Clever** trick.
- She's **clever**.
- A **clever** design.
- **Clever** idea.

### Pozitiv çalar

- Problem həllində → clever solution
- Kod texnikası → clever code

### Negativ çalar (ehtiyatlı ol!)

- **Too clever** = hiyləgər, qaydalarla oynayan
- "Don't be **too clever**." = çox hiyləgər olma

### Clever vs Smart

- **smart** = ümumi ağıllı
- **clever** = yaradıcı problem həll

- She's **smart** in general. (bütün sahələrdə)
- **Clever** use of caching. (spesifik trick)

---

## 3. Intelligent — Rəsmi Ağıllı

Smart-dan rəsmi. Yazılı, elmi.

### Nümunələr

- An **intelligent** person.
- **Intelligent** design.
- Artificial **intelligence** (AI).
- Highly **intelligent**.

### Smart vs Intelligent

- **smart** = daily, informal
- **intelligent** = rəsmi, yazılı

Əksər hallarda interchange olur, lakin rəsmi yazıda **intelligent** daha təbii.

---

## 4. Wise — Müdrik

Təcrübədən gələn ağıl. Qərar vermə.

### Nümunələr

- A **wise** decision.
- **Wise** advice.
- A **wise** old man.
- **Wise** investment.

### Smart vs Wise

- **smart** = ağıllı (intellektual)
- **wise** = müdrik (həyat təcrübəsi)

- She's **smart**. (ağıllıdır)
- She's **wise**. (təcrübəlidir, yaxşı qərar verir)

### İfadələr

- **wise up** = ayıl (phrasal)
- **wise** words
- street-**wise** (küçə təcrübəsi)

---

## 5. Brilliant — Parlaq Ağıllı

Heyrətlənmiş reaksiya. Çox güclü.

### Nümunələr

- **Brilliant** idea!
- A **brilliant** student.
- **Brilliant** engineer.
- **Brilliant** mind.

### British slang

- "That's **brilliant**!" = super! (coffee was brilliant = əla idi)

Britlər "brilliant"-ı "great" kimi istifadə edir.

---

## 6. Bright — Zəki (Gənc İnsan)

Xüsusilə gənclər və uşaqlar üçün. Gələcəyi parlaq görünən.

### Nümunələr

- A **bright** student.
- **Bright** young engineer.
- **Bright** future.
- Very **bright** kid.

### Smart vs Bright

- **smart** = hər yaş
- **bright** = gənc, potensial

---

## Bonus Sözlər

### Genius

İsim və sifət. Ən yüksək səviyyə.
- A **genius** idea.
- She's a **genius**.

### Sharp

Zehni iti. Informal.
- **Sharp** mind.

### Savvy

Informal, pratik bilik.
- Tech-**savvy**.
- Business-**savvy**.

### Knowledgeable

Çox şey bilən (bilik).
- He's very **knowledgeable** about AWS.

## Hansı Söz Hansı Kontekstdə?

### Interview özünü təqdim

- "I'm a **smart** developer." ✓ (ok)

- "I make **wise** architectural decisions." ✓

### Kolleqaya tərif

- "That's a **clever** solution!" ✓
- "**Brilliant** idea!" ✓ (British)
- "**Smart** move." ✓

### Qərar haqqında

- "A **wise** decision." ✓
- "A **smart** choice." ✓
