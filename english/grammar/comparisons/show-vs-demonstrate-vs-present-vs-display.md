# Show / Demonstrate / Present / Display — Göstərmək Fellər

## Səviyyə
B1-B2

---

## Əsas Cədvəl

| Söz | Kontekst | Rəsmilik |
|-----|----------|---------|
| **show** | ümumi, gündəlik | casual |
| **demonstrate** | nümunə ilə sübut et | rəsmi |
| **present** | hazırlanmış, auditoriyaya | rəsmi, strukturlu |
| **display** | ekranda / vizual görünüş | texniki / UI |

> **Qısa qayda:**
> - **show** = göstər (ümumi, gündəlik)
> - **demonstrate** = nümunə ilə sübut et (formal, interview)
> - **present** = auditoriyaya formal təqdim et
> - **display** = ekranda/interfeysdə göstər (UI/tech)

---

## 1. Show — Ümumi Göstərmək (Casual)

Ən çox işlənən. Hər kontekstdə istifadə oluna bilər.

### Nümunələr

- Let me **show** you how it works.
- The error **shows** in the logs.
- **Show** me the code.
- The graph **shows** the traffic spike.
- I'll **show** you the shortcut.

### Kontekst

- Gündəlik söhbət, casual
- Həmkarlarla: "Let me **show** you what I found." ✓
- Həm fiziki, həm abstrakt: "**Show** the modal", "**show** respect"

### Struktur

- **show** + sb + sth
- **show** + sth + to + sb

- **Show** me the error. ✓
- **Show** the error to me. ✓

---

## 2. Demonstrate — Nümunə ilə Sübut Et

Yalnız göstərmək deyil — **sübut etmək**, açıqlamaq, nümunə vermək.
İnterviewda, prezentasiyada, texniki izahda çox güclü söz.

### Nümunələr

- Can you **demonstrate** your approach?
- The tests **demonstrate** the code is correct.
- Let me **demonstrate** how the caching works.
- I can **demonstrate** the concept with a simple example.
- This **demonstrates** why immutability matters.

### Əsas fərq: Show vs Demonstrate

- **show** = görünür etmək
- **demonstrate** = nümunə ilə sübut etmək

- "Let me **show** you the feature." (görüntü)
- "Let me **demonstrate** how the feature works." (necə işlədiyini göstər + izah)

### Interview kontekstdə — çox güclü

- "Can you **demonstrate** this on a real example?" ✓
- "I'll **demonstrate** the algorithm step by step." ✓
- "This solution **demonstrates** scalability." ✓

### İsim: Demonstration

- A live **demonstration**.
- A **demo** (qısaldılmış forması).

---

## 3. Present — Formal Auditoriyaya Təqdim Et

Hazırlanmış məzmunu formal olaraq tamaşaçıya/komandaya göstərmək.
Struktur, hazırlıq və auditoriya — üçü birlikdə.

### Nümunələr

- I'll **present** the architecture to the team.
- She **presented** her findings to the board.
- **Present** the solution clearly.
- He **presented** three options for the stakeholders.
- We're **presenting** the new API design next week.

### Əsas fərq: Show vs Present

- **show** = göstər (informal, hazırlıq tələb etmir)
- **present** = formal, auditoriyaya, hazırlanmış

- "Let me **show** you the diagram." (casual)
- "I'll **present** the architecture at the sprint review." (formal, hazırlanmış)

### Kontekst: Nə vaxt Present?

- System design müzakirəsi: "I'll **present** my approach." ✓
- Stakeholder meeting: "We **presented** the new roadmap." ✓
- Interview: "Let me **present** the solution." ✓

### İsim: Presentation

- A **presentation** to the board.
- A **PowerPoint presentation**.
- "My **presentation** tomorrow covers..." ✓

### "Present" başqa mənalar

- **present** = hal-hazırda (adj/adv): "at **present**", "**present** state"
- **present** = hədiyyə (isim): a **present** for you
- Eyni yazılış, fərqli məna — diqqət!

---

## 4. Display — Ekranda / Vizual Göstər (UI/Tech)

Texniki və UI kontekst üçün. Ekranda göstərmək, interfeysdə görünmək.

### Nümunələr

- **Display** an error message.
- The chart **displays** the traffic data.
- **Display** the modal on button click.
- The screen **displays** the current status.
- **display: none** in CSS.

### Əsas fərq: Show vs Display

- **show** = ümumi (insan-insan, yaxud UI)
- **display** = texniki (ekran, interface, vizual komponent)

- "**Show** me the results." (casual, insan)
- "**Display** the results in a table." (UI/tech)

### Tech kontekstdə — çox işlənir

- CSS: `display: flex`, `display: none`, `display: block`
- React/Vue: "**display** the loading spinner"
- Error handling: "**display** an error toast"
- Dashboard: "**display** metrics in real time"

### İsim: Display

- A **display** unit (monitor).
- On **display** (görünür yerdə).
- **Display** settings.

## Müqayisə Cədvəli

| | Show | Demonstrate | Present | Display |
|-|------|-------------|---------|---------|
| Rəsmilik | casual | rəsmi | çox rəsmi | texniki |
| Auditoriya | 1 nəfər | kiçik qrup | böyük qrup | UI/user |
| Hazırlıq | yox | bəzən | bəli | yox |
| İnterviewda | bəli | çox | bəli | az |
| UI/Frontend | bəli | nadir | nadir | çox |
| Sübut hissi | az | çox | bəli | yox |

## Tez-tez Yanlış İşlənənlər

| ❌ Yanlış | ✅ Düzgün |
|-----------|-----------|
| Display me the error. | **Show** me the error. |
| Let me present how the function works. (casual) | Let me **show** / **demonstrate** how it works. |
| I'll demonstrate the results in a chart. | I'll **display** the results in a chart. |
| The screen shows the CSS display property. | The screen **displays** the CSS output. |
| Can you show the algorithm's correctness? | Can you **demonstrate** the algorithm's correctness? |
| Present the modal when clicked. | **Display** / **Show** the modal when clicked. |
| I demonstrated my findings to 1 person. | I **showed** my findings to 1 person. |
| Display me what you found. | **Show** me what you found. |

## Test

Hansı söz uyğundur?

1. ______ me the code — I want to understand it quickly. (casual)
2. The dashboard ______ real-time traffic data. (UI/texniki)
3. Can you ______ your approach during the interview? (nümunə ilə sübut)
4. Next Monday I'll ______ the new architecture to the team. (formal, hazırlanmış)
5. The chart ______ a clear downward trend. (vizual)
6. Let me ______ you how to set up the dev environment. (casual, gündəlik)
7. She ______ three options to the board. (formal auditoriya)
8. ______ an error message if the input is invalid. (UI)

**Cavablar:** 1. Show, 2. displays, 3. demonstrate, 4. present, 5. displays/shows, 6. show, 7. presented, 8. Display

## Tech / İş Kontekstində

### Show

- "Let me **show** you the production logs." ✓
- "The metrics **show** a memory leak." ✓

### Demonstrate

- "I can **demonstrate** the algorithm in Python." ✓
- "The benchmark **demonstrates** a 3x improvement." ✓
- "Can you **demonstrate** your approach?" (interview) ✓

### Present

- "I'll **present** the findings to the CTO." ✓
- "We're **presenting** the new API design next week." ✓
- System design interview: "Let me **present** my solution." ✓

### Display

- `display: flex` — CSS ✓
- "**Display** an error toast on failure." ✓
- "The UI **displays** the user's subscription status." ✓

## Azərbaycanlı Səhvləri

- ✗ Display me the results. (display → ekran üçün, subyekt)
- ✓ **Show** me the results. / **Display** the results on screen.

- ✗ I'll present how the function works. (casual kontekst — present too formal)
- ✓ I'll **show** / **demonstrate** how the function works.

- ✗ Demonstrate the error message. (UI → display)
- ✓ **Display** the error message.
