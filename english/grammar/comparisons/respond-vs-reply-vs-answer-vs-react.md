# Respond / Reply / Answer / React — Cavab Vermək Felləri

## Səviyyə
A2-B1

---

## Niyə Çətindir?

Azərbaycancada hamısı "cavab vermək" və ya "reaksiya göstərmək" kimi tərcümə olunur. Amma ingilis dilində hər birinin öz konteksti var: **answer** sual üçündür, **reply** mesaj üçündür, **respond** rəsmi reaksiya üçündür, **react** isə emosional/ani reaksiya üçündür. Bunları qarışdırmaq natural səslənməni pozur.

---

## Əsas Cədvəl

| Söz | Kontekst | Tonu | Struktur |
|-----|----------|------|---------|
| **answer** | sual / zəng / qapı | ümumi | answer + N (direkt) |
| **reply** | yazılı mesaj / email | yazılı | reply + to + N |
| **respond** | rəsmi cavab / reaksiya | rəsmi | respond + to + N |
| **react** | emosional / ani reaksiya | fərqli məna | react + to + N |

> **Qısa qayda:**
> - **answer** = sual cavabla, zəngi götür, qapını aç
> - **reply** = mesaja, emailə, comment-ə cavab yaz
> - **respond** = rəsmi cavab ver, reaksiya göstər (situasiyaya da)
> - **react** = ani/emosional reaksiya (cavab deyil — hiss!)

---

## 1. Answer — Sual, Zəng, Qapı

Ən ümumi söz. Birbaşa obyekt alır ("to" olmadan).

### Struktur

- **answer** + N → direkt obyekt (to olmadan!)
- **answer** + question / phone / door

### Nümunələr

| İfadə / Kontekst | Nümunə | Azərbaycanca |
|------------------|--------|-------------|
| answer a question | "Can you answer my question about the DB schema?" | sualı cavabla |
| answer the phone | "She didn't answer the phone during standup." | zəngi götür |
| answer the door | "Can you answer the door? Someone's here." | qapını aç |
| answer an email | "I'll answer your email after the meeting." | emailə cavab ver |
| answer a ticket | "Please answer the support ticket first." | ticketi cavabla |

### ⚠ Yanlış struktur

- ✗ "Answer to my question." (to yox — direkt obyekt!)
- ✓ "**Answer** my question."

---

## 2. Reply — Yazılı Mesaja Cavab

Yazılı mesajlar üçün: email, Slack, comment, text.

### Struktur

- **reply** + to + N → (to mütləqdir!)
- **reply** (intransitiv, tək başına) → "I haven't replied yet."

### Nümunələr

| İfadə / Kontekst | Nümunə | Azərbaycanca |
|------------------|--------|-------------|
| reply to an email | "I haven't replied to the recruiter yet." | emailə cavab vermədim |
| reply to a PR comment | "Please reply to the PR comments before merging." | commenta cavab ver |
| reply to a Slack message | "He replied to the thread an hour ago." | thread-ə cavab verdi |
| quick reply | "Send a quick reply — yes or no." | qısa cavab |
| reply all | "Don't reply all to company-wide emails." | hamıya cavab verme |

### ⚠ Yanlış struktur

- ✗ "Reply my email." (to lazımdır!)
- ✓ "Reply **to** my email."

### Reply vs Answer

- **answer** = sual cavablandırmaq (ağızdan da, yazılıdan da), zəng götürmək
- **reply** = yalnız yazılı mesaja cavab

- "I **answered** his call." ✓ (zəngi götürdüm)
- "I **replied** to his Slack message." ✓ (yazılı mesaj)

---

## 3. Respond — Rəsmi Cavab / Reaksiya

Daha formal. Həm "cavab vermək", həm "reaksiya göstərmək" mənasında.

### Struktur

- **respond** + to + N → (to mütləqdir!)
- **respond** + with + N → "responded with silence"

### Nümunələr

| İfadə / Kontekst | Nümunə | Azərbaycanca |
|------------------|--------|-------------|
| respond to feedback | "Please respond to the code review feedback." | feedbacka cavab ver |
| respond to a crisis | "The team responded to the outage immediately." | situasiyaya reaksiya |
| respond to an email | "I'll respond to your email by EOD." | rəsmi cavab |
| server responds | "The server responded with 503." | server cavab verdi |
| respond positively | "Users responded well to the new UI." | müsbət reaksiya |

### ⚠ Yanlış

- ✗ "Respond my email." (to lazımdır!)
- ✓ "Respond **to** my email."

### Reply vs Respond

Hər ikisi yazılı cavab üçün işlənir, lakin:
- **reply** = birbaşa, mesaja spesifik cavab
- **respond** = rəsmi, situasiya da daxil

- "**Reply to** the email." (emailə cavab göndər)
- "**Respond to** the outage." (situasiyaya reaksiya — email mümkün deyil)
- "The API **responded** with 200." (teknik kontekstdə respond standart)

### Tech kontekst (vacib!)

- **response** — texniki yazıda standart: API response, HTTP response, server response
- ~~server reply~~ — texniki kontekstdə işlənmir
- "Response time" ✓ / ~~"reply time"~~ ✗

---

## 4. React — Emosional / Ani Reaksiya

**Diqqət:** bu "cavab vermək" deyil — **reaksiya göstərməkdir**. Bəzən emosional, bəzən ani.

### Struktur

- **react** + to + N

### Nümunələr

| İfadə / Kontekst | Nümunə | Azərbaycanca |
|------------------|--------|-------------|
| react to news | "How did the team react to the layoffs?" | necə reaksiya verdi |
| react to a change | "Users reacted negatively to the new pricing." | mənfi reaksiya |
| react emotionally | "Don't react emotionally in code reviews." | emosional reaksiya etmə |
| react quickly | "The system reacted to the spike in seconds." | sürətli reaksiya |
| how did he react? | "How did your manager react to the delay?" | necə qarşıladı |

### Respond vs React

| | Respond | React |
|-|---------|-------|
| Düşünülmüş? | bəli | xeyr — ani |
| Emosional? | xeyr | tez-tez bəli |
| Nümunə | "He responded calmly." | "He reacted angrily." |

- "She **responded** to the criticism professionally." (düşünülmüş, sakit)
- "She **reacted** with frustration." (ani, emosional)

### ⚠ Tech-də React = framework adı!

- **React** (JavaScript library) — bu feil "react" ilə bağlantısı yoxdur
- "We use **React** for the frontend." ≠ reaksiya

---

## Struktur Müqayisəsi (KRİTİK!)

| Feil | Direkt obyekt? | "to" lazımdır? | Nümunə |
|------|----------------|-----------------|--------|
| **answer** | ✓ bəli | ✗ xeyr | "Answer **the question**." |
| **reply** | ✗ xeyr | ✓ bəli | "Reply **to** the email." |
| **respond** | ✗ xeyr | ✓ bəli | "Respond **to** the issue." |
| **react** | ✗ xeyr | ✓ bəli | "React **to** the news." |

**Yalnız "answer" birbaşa obyekt alır — digərləri "to" istəyir!**

## Tez-tez Yanlış İşlənənlər

| ❌ Yanlış | ✅ Düzgün |
|-----------|-----------|
| "Answer to my question." | "**Answer** my question." |
| "Reply my email." | "Reply **to** my email." |
| "Respond my message." | "Respond **to** my message." |
| "He answered to the criticism." | "He **responded to** the criticism." |
| "Server reply time" | "Server **response** time" |
| "React to the email quickly." | "**Reply to** / **Respond to** the email quickly." |
| "I reacted his question." | "I **answered** his question." |
| "How did he answer to the news?" | "How did he **react to** the news?" |

## Məşq Tapşırıqları

### 1. Doğru sözu seçin (respond / reply / answer / react)

1. "Please ______ my question about the database schema." (sual)
2. "I haven't ______ to the recruiter's email yet." (yazılı mesaj)
3. "How did the team ______ to the new deployment process?" (emosional/fikir)
4. "The server ______ with a 500 error." (texniki)
5. "Can you ______ the phone? I'm in a meeting." (zəng)
6. "She ______ to the PR comments before merging." (yazılı cavab)
7. "We need to ______ to the client's complaint formally." (rəsmi)
8. "Users ______ positively to the dark mode feature." (reaksiya)
9. "Please ______ the door — someone knocked." (qapı)
10. "I'll ______ to your Slack message after standup." (yazılı mesaj)

**Cavablar:** 1. answer, 2. replied, 3. react, 4. responded, 5. answer, 6. replied, 7. respond, 8. reacted, 9. answer, 10. reply

### 2. Cümləni tamamlayın

1. "Sorry for the late ______ — I was on vacation." (yazılı cavab, isim)
2. "The API ______ time is under 100ms." (texniki, isim)
3. "Don't ______ emotionally to critical feedback in code reviews." (ani reaksiya)
4. "I'll ______ your email by EOD." (rəsmi cavab)
5. "He ______ to all ten interview questions confidently." (sualları cavabladı)
6. "The monitoring system ______ to the spike within 3 seconds." (ani reaksiya / texniki)

**Cavablar:** 1. reply, 2. response, 3. react, 4. respond to / answer, 5. answered, 6. responded / reacted
