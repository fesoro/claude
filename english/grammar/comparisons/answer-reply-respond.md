# Answer / Reply / Respond / React — Cavab Vermək Fellər

## Səviyyə
A2-B1

---

## Əsas Cədvəl

| Söz | Kontekst | Tonu |
|-----|----------|------|
| **answer** | sual / zəng / qapı | ümumi |
| **reply** | mesaj / email | yazılı |
| **respond** | rəsmi cavab / reaksiya | rəsmi |
| **react** | emosional reaksiya | fərqli məna |

> **Qısa qayda:**
> - **answer** = sual cavablandır (ümumi)
> - **reply** = mesaja cavab (yazılı)
> - **respond** = rəsmi cavab / reaksiya
> - **react** = emosional / sürətli reaksiya

---

## 1. Answer — Cavab Vermək (Ümumi)

Sual, zəng, qapı, məktub — hər cür.

### Struktur

- **answer** + N (direct object)
- **answer** + question / phone / door

### Nümunələr

- **Answer** my question.
- She didn't **answer** the phone.
- Please **answer** the door.
- **Answer** all 10 questions.
- I can't **answer** that.

### İsim

- The **answer** is 42.
- Short **answer**: yes.
- Detailed **answer**.

### Answer the phone / door

Fərqli məna:
- **answer the phone** = telefon götür (zəng)
- **answer the door** = qapı aç (qonaq gəlib)

---

## 2. Reply — Cavab (Yazılı / Formal)

Adətən mesajlara cavab.

### Struktur

- **reply** + to + N
- **reply** + N (direct)

**Diqqət:** "to" istəyir!

### Nümunələr

- **Reply to** my email.
- She **replied** to the message.
- I haven't **replied** yet.
- **Reply** "yes" if you agree.

### ⚠ Yanlış struktur

- ✗ Reply my email.
- ✓ **Reply to** my email.

"Reply" direkt obyekti qəbul etmir — "to" lazımdır.

### İsim

- Quick **reply**.
- **Reply** all (bütün cavabla).

### Reply vs Answer

- **answer** = ümumi (sual, zəng)
- **reply** = yazılı mesaja

- I **answered** the call. ✓ (zəng)
- I **replied** to the email. ✓ (yazılı)

---

## 3. Respond — Rəsmi Cavab / Reaksiya

Rəsmi. Həmçinin "reaksiya".

### Struktur

- **respond** + to + N (həmişə to!)

### Nümunələr

- Please **respond** to this urgent matter.
- The server **responded** slowly.
- He **responded** with anger.
- Users **respond** well to the new design.

### ⚠ Yanlış

- ✗ Respond my email.
- ✓ **Respond to** my email.

### Reply vs Respond

Oxşar, lakin:
- **reply** = daha birbaşa, mesaja
- **respond** = rəsmi, reaksiya da daxil

- "**Reply to** the email ASAP." (cavab göndər)
- "The server **responded** with 404." (reaksiya)

### Tech kontekst

- API **response** (mütləq respond)
- HTTP **response** code
- Server **response** time
- ~~Server reply~~ (istifadə olunmur, response standart)

### İsim

- Response time.
- No **response** yet.
- Positive **response**.

---

## 4. React — Reaksiya (Emosional / Sürətli)

"Cavab" deyil, "reaksiya".

### Struktur

- **react** + to + N

### Nümunələr

- How did she **react**?
- Don't **react** emotionally.
- Users **reacted** positively.
- The market **reacted** to the news.

### Respond vs React

- **respond** = düşünülmüş cavab
- **react** = anidən reaksiya (bəzən emosional)

- "He **responded** calmly." (sakit düşüncə ilə)
- "He **reacted** angrily." (anında əsəb)

### Tech: React = framework adı!

- **React** (JavaScript library)
- Respond-dan fərqli kontekst.
