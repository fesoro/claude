# Efficient vs Effective — Resursla vs Nəticə ilə

## Səviyyə
B1 (tech/iş kontekstində KRİTİK)

---

## Əsas Fərq

| Söz | Məna |
|-----|------|
| **efficient** | az resursla çox iş (fast/optimized) |
| **effective** | işləyir, nəticə verir (works) |

> **Qısa qayda:**
> - **efficient** = "**How**?" → resurs azdır (vaxt, pul, enerji)
> - **effective** = "**Does it work?**" → məqsədə çatır

---

## 1. Efficient — Resursla Qənaətlidir

"Az enerji / vaxt / pul ilə çox iş".

### Nümunələr

- An **efficient** algorithm.
- **Efficient** use of time.
- **Efficient** team.
- This code is **efficient**.
- Fuel-**efficient** car.
- **Efficient** process.

### Sinonim

- optimized
- streamlined
- productive (per unit resource)

### Tech kontekst

- **Efficient** code → az CPU / memory
- **Efficient** algorithm → O(n) vs O(n²)
- **Efficient** query → indexed, fast
- **Efficient** rendering

---

## 2. Effective — İşləyir / Nəticə Verir

"Məqsədə çatdırır" — sürətli və ya ya da yox, fərqi yox.

### Nümunələr

- **Effective** strategy.
- **Effective** communication.
- **Effective** leader.
- An **effective** solution.
- The ad was **effective**.

### Sinonim

- successful
- working
- productive (achieves goal)
- impactful

---

## 3. Kritik Fərq

### Klassik nümunə: Parça kəsmək

- **Efficient way**: Laser kəsici (sürətli, dəqiq, az material israfı)
- **Effective way**: Bıçaqla (yavaş, amma kəsir, işləyir)

Hər ikisi kəsir (= effective).
Laser daha **efficient** (az resurs).

---

## Hansı Vacibdir?

### "Efficient but not effective"

- Kodun sürətlidir, amma yanlış nəticə verir.
- Fast but wrong.

### "Effective but not efficient"

- Kod işləyir, amma yavaşdır.
- Works but slow.

### Best: "Efficient AND effective"

- Düzgün nəticə + sürətli / qənaətcil.

## Tech / İş Kontekstində

### Efficient

- **Efficient** database query. ✓ (indexed, fast)
- **Efficient** memory usage. ✓
- **Efficient** algorithm. ✓
- Time-**efficient**. ✓
- **Efficient** team processes. ✓

### Effective

- **Effective** incident response. ✓ (məqsədə çatır)
- **Effective** code review. ✓ (bug tutur)
- An **effective** architecture. ✓ (problem həll edir)
- **Effective** marketing. ✓

---

## Performance Review / Interview

### Tərif

- "She's an **efficient** worker — finishes tasks quickly."
- "He's an **effective** leader — gets results."

### Both together

- "Our team is **efficient** and **effective**." ✓ (sürətli + nəticə)

---

## Azərbaycanlı Səhvləri

- ✗ The meeting was efficient. (məqsədə çatdımı? → effective)
- ✓ The meeting was **effective**.
  (Amma əgər "tez başa çatdı, vaxtı qənaət etdik" deyirsən → **efficient**)

- ✗ An effective algorithm (O(log n) demək istəyirsən → efficient)
- ✓ An **efficient** algorithm.

---

## Related Sözlər

### Efficiency (isim)

- **Efficiency** = məhsuldarlıq
- Improve **efficiency**.

### Effectiveness (isim)

- **Effectiveness** = təsirlilik
- Measure **effectiveness**.

### Efficacy (isim, rəsmi)

- **Efficacy** = tibbi / elmi təsir
- Vaccine **efficacy**.

### Inefficient / Ineffective

- **inefficient** = resurs israfı
- **ineffective** = işləmir

---

## Bir Cümlə, İki Sual

"Was the meeting good?"

- "It was **effective**" = Məqsədə çatdıq, nəticə əldə etdik.
- "It was **efficient**" = Qısa idi, vaxt aldı deyildi.

Meetings istəyirsən: **efficient AND effective**.

---

## Xatırlatma

| Sual | Söz |
|------|-----|
| Resurs az istifadə edir? | **efficient** |
| Məqsədə çatır? | **effective** |
| Sürətli + dəqiq? | **efficient** |
| İşləyir? | **effective** |

**Qızıl qayda:**
- **Effective** = does the right thing.
- **Efficient** = does things right (with minimal resources).
