# Sifətlərin Sırası (Order of Adjectives)

## Səviyyə
B1-B2

## Niyə Vacibdir?

İngilis dilindəki sifət sırası sabitdir. Yanlış sıra ana dilli ingilis danışanına qəribə səslənir.

---

## Sifət Sırası

```
1. Determiner  →  a, the, my, this, some
2. Observation →  beautiful, awful, lovely, interesting
3. Size        →  big, small, tall, long, tiny
4. Shape       →  round, square, triangular
5. Age         →  old, young, new, ancient
6. Color       →  red, blue, dark, light
7. Origin      →  French, Azerbaijani, Italian, Chinese
8. Material    →  wooden, silver, leather, cotton
9. Purpose     →  sleeping (bag), running (shoes), cooking (pot)
10. NOUN
```

---

## Əzbərləmə Üsulu: **OSASCOMP**

**O**bservation → **S**ize → **A**ge → **S**hape → **C**olor → **O**rigin → **M**aterial → **P**urpose

## Nümunələr

| Nümunə | Sıra |
|--------|------|
| a **beautiful big** house | observation + size |
| a **big old** table | size + age |
| a **small round** clock | size + shape |
| a **lovely old** Italian villa | observation + age + origin |
| a **beautiful black** leather bag | observation + color + material |
| a **big old brown** wooden box | size + age + color + material |
| some **nice new** running shoes | observation + age + purpose |
| a **tiny little** brown mouse | size + size (hər ikisi kiçiklik) + color |

---

## Ən Çox İşlənən Kombinasiyalar

### İki sifət
- a **big** red apple (size + color)
- a **beautiful** young woman (observation + age)
- a **long** black dress (size + color)
- an **old** wooden chair (age + material)

### Üç sifət
- a **lovely little** old cottage (observation + size + age)
- a **big** black **leather** jacket (size + color + material)
- a **small** round **silver** coin (size + shape + material)

---

## Praktik Qayda

Praktikada bir cümlədə 3-dən çox sifət nadir hallarda işlənir. Ən çox **2 sifət** cütlüyü istifadə edilir.

**Ən çox işlənən cütlər:**
- size + color: a **big red** ball
- observation + size: a **beautiful big** house
- age + material: an **old wooden** table
- size + material: a **large leather** sofa

---

## AND — Eyni kateqoriyadan iki sifət

**Eyni kateqoriyadan** iki sifət (məsələn, iki rəng) "and" ilə birləşdirilir:

- a **black and white** photo (iki rəng)
- a **red and blue** flag
- a **warm and friendly** atmosphere (hər ikisi observation)

## Azərbaycanlıların Tez-tez Etdiyi Səhvlər

| Səhv | Düzgün | İzah |
|------|--------|-------|
| a red big car | a **big red** car | size → color |
| an Italian old car | an **old Italian** car | age → origin |
| a wooden old table | an **old wooden** table | age → material |
| a beautiful Italian old villa | a **beautiful old Italian** villa | observation → age → origin |
