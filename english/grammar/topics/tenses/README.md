# Zamanlar (Tenses) — Ümumi Baxış

İngilis dilində **12 əsas zaman** vardır. Bunlar 3 qrup üzrə bölünür: indiki, keçmiş, gələcək — hər birinin 4 aspekti var.

## Zamanların Tam Cədvəli

| | Simple | Continuous | Perfect | Perfect Continuous |
|---|--------|------------|---------|-------------------|
| **Present** | [Present Simple](present-simple.md) | [Present Continuous](present-continuous.md) | [Present Perfect](present-perfect.md) | [Present Perfect Continuous](present-perfect-continuous.md) |
| **Past** | [Past Simple](past-simple.md) | [Past Continuous](past-continuous.md) | [Past Perfect](past-perfect.md) | [Past Perfect Continuous](past-perfect-continuous.md) |
| **Future** | [Future Simple](future-simple.md) | [Future Continuous](future-continuous.md) | [Future Perfect](future-perfect.md) | — |

## Hər Zamanın Qısa Xülasəsi

| Zaman | Quruluş | Əsas işlənmə |
|-------|---------|-------------|
| **Present Simple** | do/does + V1 | adət, həqiqət, cədvəl |
| **Present Continuous** | am/is/are + V-ing | indi baş verir, müvəqqəti |
| **Present Perfect** | have/has + PP | keçmiş → indiki əlaqə |
| **Present Perfect Cont.** | have/has been + V-ing | davam edən proses |
| **Past Simple** | V2 / did + V1 | tamamlanmış keçmiş |
| **Past Continuous** | was/were + V-ing | keçmiş fon hərəkəti |
| **Past Perfect** | had + PP | keçmişdəki keçmiş |
| **Past Perfect Cont.** | had been + V-ing | keçmişdə davam etmiş |
| **Future Simple** | will + V1 | spontan qərar, proqnoz |
| **Future Continuous** | will be + V-ing | gələcəkdə davam edəcək |
| **Future Perfect** | will have + PP | gələcəkdə tamamlanmış |

## Hansından Başlamaq Lazımdır?

**A1 üçün:** Present Simple → Past Simple → Future (Will/Going to)
**A2 üçün:** Present Continuous → Present Perfect
**B1 üçün:** Past Continuous → Past Perfect → Perfect Continuous
**B2+ üçün:** Bütün zamanları müqayisəli öyrənin → [tense-comparison.md](tense-comparison.md)

## Zaman İfadələri (Time Markers)

| Zaman | Açar sözlər |
|-------|-------------|
| Present Simple | always, usually, often, sometimes, never, every day |
| Present Continuous | now, at the moment, currently, today, still |
| Present Perfect | just, already, yet, ever, never, since, for, recently |
| Past Simple | yesterday, ago, last week, in 2020, when |
| Past Continuous | while, when, at that time, as |
| Past Perfect | before, after, by the time, already (keçmişdə) |
| Future Simple | tomorrow, next week, soon, I think, probably |
| Future Perfect | by + zaman (by 2030, by next year) |

---

## Ən Çox İşlənən Zamanlar

1. **Present Simple** — gündəlik həyat, ən çox işlənən
2. **Past Simple** — danışıq dilində keçmişi ifadə etmək
3. **Present Perfect** — keçmişin indikiylə əlaqəsi
4. **Present Continuous** — indiki an, yaxın planlar
5. **Future with Will/Going to** — gələcək planlar, qərarlar
