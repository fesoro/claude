# Modal Fellər (Modal Verbs)

## Səviyyə
B1 (əsas) → C1 (irəliləmiş)

## Modal Fellər Nədir?
Modal fellər **köməkçi fellərdir** — əsas felin mənasını dəyişir: imkan, zərurət, icazə, ehtimal, məsləhət bildirir. Bunların **özü -s, -ed almır**, hər zaman əsas fellə birlikdə işlənir.

**Əsas modallar:** can, could, may, might, must, have to, should, ought to, will, would, need, dare

---

## Xülasə Cədvəli

| Modal | Azərbaycanca | Əsas Məna |
|-------|--------------|-----------|
| **can** | bacarır, bilir | İmkan, bacarıq |
| **could** | bacarırdı, bilərdi | Keçmiş imkan, nəzakətli xahiş |
| **may** | ola bilər | Rəsmi icazə, ehtimal |
| **might** | ola bilər (az ehtimal) | Zəif ehtimal |
| **must** | lazımdır, olmalıdır | Zərurət, əminlik |
| **have to** | məcburdur | Xarici zərurət |
| **should** | lazımdır, tövsiyə | Məsləhət, öhdəlik |
| **ought to** | lazımdır | Məsləhət (should kimi) |
| **will** | edəcək | Gələcək, könüllülük |
| **would** | edərdi | Keçmiş vərdiş, nəzakətli xahiş |
| **need** | lazım | Zərurət (əsasən mənfi) |

## Modal Fellərin Quruluşu

```
(+) şəxs + modal + V (əsas)
(-) şəxs + modal + not + V
(?) Modal + şəxs + V?
```

**Qeyd:** Modal fellərdən sonra **infinitive without "to"** gəlir:
- ✅ She **can swim**.
- ❌ She **can to swim**.

**İstisna:** have to, ought to, need to — "to" ilə işlənir.
