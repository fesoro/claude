## Quick Decision Guide

```
enjoy / avoid / finish / mind / suggest / keep
→ GERUND (verb + -ing)

want / need / decide / hope / plan / refuse / manage
→ INFINITIVE (to + verb)

after prepositions (in, at, of, about, to*)
→ GERUND

remember / forget / stop / try / go on / regret / mean
→ BOTH — məna dəyişir! (verbs-both.md-ə bax)
```

> *"looking forward to **seeing**" — "to" burada prepozisiyadır!
