# Şərt Cümləsi (Conditionals)

## Səviyyə
B1 (1-ci, 2-ci) → B2 (3-cü) → C1 (Mixed)

## Şərt Cümlələri Nədir?
"Əgər ... olsa, ... olar" quruluşlu cümlələrdir. İngilis dilində 5 növü var.

---

## Növlərin Xülasəsi

| Növ | Quruluş | Məna | Nümunə |
|-----|---------|------|--------|
| **Sıfır** | If + Present Simple, Present Simple | Daimi həqiqət | If you heat ice, it melts. |
| **1-ci** | If + Present Simple, will + V | Real gələcək | If it rains, I'll stay home. |
| **2-ci** | If + Past Simple, would + V | Xəyali/qeyri-real hal | If I had money, I would travel. |
| **3-cü** | If + Past Perfect, would have + V3 | Keçmiş xəyal | If I had studied, I would have passed. |
| **Mixed** | Müxtəlif kombinasiyalar | Vaxtlar arası | If I had slept, I would be fine now. |
