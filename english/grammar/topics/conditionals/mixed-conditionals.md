# Mixed Conditionals — Qarışıq Şərt Cümləsi

## Səviyyə
C1

## Qısa Xülasə
Şərt və nəticə fərqli vaxtlara aiddir — keçmiş şərt + indiki nəticə, və ya indiki şərt + keçmiş nəticə.

---

## Növ 1: Keçmiş Şərt → İndiki Nəticə

**Formula: If + Past Perfect, would + V**

| Azərbaycanca | İngilis |
|--------------|---------|
| O gün oxusaydım, indi həkimolardım. | If I **had studied** medicine, I **would be** a doctor now. |
| Akkumulyatoru doldursaydı, indi şarjı olardı. | If she **had charged** her phone, it **would work** now. |

---

## Növ 2: İndiki Şərt → Keçmiş Nəticə

**Formula: If + Past Simple, would have + V3**

| Azərbaycanca | İngilis |
|--------------|---------|
| Əgər cəsur olsaydım (amma deyiləm), o gün danışardım. | If I **were** braver, I **would have spoken** up that day. |

---

## Xülasə Kartı

```
MIXED CONDITIONALS
Keçmiş şərt + İndiki nəticə:
  If + Past Perfect, would + V
  "If I had studied medicine, I would be a doctor now."

İndiki şərt + Keçmiş nəticə:
  If + Past Simple, would have + V3
```
