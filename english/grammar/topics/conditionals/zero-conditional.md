# Zero Conditional — Sıfır Şərt Cümləsi

## Səviyyə
A2-B1

## Qısa Xülasə
Həmişə doğru olan həqiqətlər, elmi faktlar, ümumi qanunlar üçün.

**Formula: If + Present Simple, Present Simple**

---

## Nümunələr

| Azərbaycanca | İngilis |
|--------------|---------|
| Suyu qızdırsan, qaynayır. | If you heat water, it boils. |
| Yağış yağırsa, torpaq islanır. | If it rains, the ground gets wet. |
| Sürtgəci bassan, düzəlir. | If you press the button, it works. |

> **When** ilə əvəz edilə bilər: **When** you heat water, it boils.

---

## 1-ci Şərtdən Fərqi

| Sıfır | 1-ci |
|-------|------|
| Həmişə doğru | Konkret vəziyyətdə doğru |
| If you heat ice, it **melts**. | If it rains tomorrow, I **will stay** home. |

---

## Xülasə Kartı

```
SIFIR ŞƏRT
If + Present Simple, Present Simple
Həmişə doğru həqiqətlər
```
