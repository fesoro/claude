# Second Conditional — 2-ci Şərt Cümləsi

## Səviyyə
B1

## Qısa Xülasə
İndiki və ya gələcəkdəki **qeyri-real, xəyali** şərtlər. "Əgər ... olsaydı (amma deyil), ... olardı."

**Formula: If + Past Simple, would + V**

---

## 1. Quruluş

| Nümunə |
|--------|
| If I **had** more money, I **would travel** the world. |
| If she **lived** closer, we **would meet** more often. |
| If I **were** you, I **would apologize**. |

> **"If I were you"** — were bütün şəxslər üçün (formal): If I **were**, If he **were**, If she **were**  
> (was da işlənir danışıq dilində, amma were daha düzgündür)

---

## 2. Nümunələr

| Azərbaycanca | İngilis |
|--------------|---------|
| Pulum olsaydı, maşın alardım. | If I **had** money, I **would buy** a car. |
| Sənin yerindəolsaydım, üzr istəyərdim. | If I **were** you, I **would apologize**. |
| O daha çox çalışsaydı, uğur qazanardı. | If she **worked** harder, she **would succeed**. |
| Uçmağı bilsəydim, oraya gedərdim. | If I **could** fly, I **would go** there. |

---

## 3. 1-ci vs. 2-ci Şərt

| 1-ci Şərt | 2-ci Şərt |
|-----------|-----------|
| Real, mümkün | Xəyali, qeyri-real |
| If it rains, I'll stay. (yağa bilər) | If I were rich, I'd travel. (amma deyiləm) |

---

## 4. Azərbaycanlıların Tez-tez Etdiyi Səhvlər

- ❌ If I **would have** money, I **would** travel.
- ✅ If I **had** money, I **would** travel. (şərt hissəsində "would" yoxdur!)

---

## Məşq

1. If she _______ (be) taller, she _______ (play) basketball.
2. If I _______ (know) his number, I _______ (call) him.
3. What _______ you _______ (do) if you _______ (win) the lottery?

**Cavablar:** 1. were / would play 2. knew / would call 3. would you do / won

---

