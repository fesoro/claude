# First Conditional — 1-ci Şərt Cümləsi

## Səviyyə
B1

## Qısa Xülasə
Real, mümkün gələcək şərtlər üçün. "Əgər bu baş versə, o baş verəcək."

**Formula: If + Present Simple, will + V**

---

## 1. Quruluş

| Cümlə | Nümunə |
|-------|--------|
| If + [şərt], [nəticə] | If it rains, I **will** stay home. |
| [Nəticə] if + [şərt] | I **will** stay home if it rains. |

> **Qeyd:** Şərt hissəsindəki "if" ilə **will işlənmir** — Present Simple işlənir!  
> ❌ If it **will rain**, I'll stay home.  
> ✅ If it **rains**, I'll stay home.

---

## 2. Nümunələr

| Azərbaycanca | İngilis |
|--------------|---------|
| Hava yaxşı olsa, parka gedərik. | If the weather **is** nice, we **will go** to the park. |
| Tez çıxsan, qatara yetişərsən. | If you **leave** early, you **will catch** the train. |
| O gəlməsə, mən gedərəm. | If she **doesn't come**, I **will leave**. |
| Ağlasan, hamı görər. | If you **cry**, everyone **will see**. |

---

## 3. Will Əvəzinə Başqaları

| Forma | Nümunə |
|-------|--------|
| can | If you study, you **can** pass. |
| might | If it rains, we **might** cancel. |
| should | If you finish, you **should** call me. |
| may | If he comes, she **may** leave. |

---

## 4. Unless = If Not

"Unless" = "əgər ... olmasaydı / -masaydı"

| Azərbaycanca | İngilis |
|--------------|---------|
| Dərmanı içməsən, sağalmayacaqsan. | **Unless** you take your medicine, you won't get better. |

---

## 5. Azərbaycanlıların Tez-tez Etdiyi Səhvlər

- ❌ If it **will rain**, I'll stay.
- ✅ If it **rains**, I'll stay. (şərt hissəsində will yoxdur!)

---

## Məşq

1. If you _______ (study) hard, you _______ (pass) the exam.
2. She _______ (miss) the bus if she _______ (not/hurry).
3. If he _______ (call), tell him I'm busy.

**Cavablar:** 1. study / will pass 2. will miss / doesn't hurry 3. calls

---

## Xülasə Kartı

```
1-Cİ ŞƏRT CÜMLƏSİ
If + Present Simple → will + V
Real, mümkün gələcək şərt
```
