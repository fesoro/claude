# Inversion (Söz sırasının dəyişməsi)

Inversion — subyekt ilə köməkçi fel yerini dəyişdirdikdə yaranır. Formal dil, ədəbi mətnlər və IELTS/C1-C2 yazısında çox işlənir.

---

## 1. Negative Adverbials (Mənfi başlanğıc)

Cümlə mənfi mənalı bir ifadə ilə başlayanda inversion tətbiq olunur.

| Trigger | Structure | Example |
|---------|-----------|---------|
| **Never** | Never + aux + S + V | Never have I seen such a mess. |
| **Rarely** | Rarely + aux + S + V | Rarely do we get such opportunities. |
| **Seldom** | Seldom + aux + S + V | Seldom did she complain. |
| **Hardly** | Hardly + aux + S + V... when | Hardly had I arrived when it started raining. |
| **Scarcely** | Scarcely + aux + S + V... when | Scarcely had she left when he called. |
| **No sooner** | No sooner + aux + S + V... than | No sooner had I sat down than the phone rang. |

**Normal → Inverted:**
- I have never seen this before. → **Never have I seen this before.**
- She rarely makes mistakes. → **Rarely does she make mistakes.**

---

## 2. "Not only... but also"

**Structure:** Not only + aux + S + V, but S also + V

| Example | Azerbaijani |
|---------|-------------|
| Not only did he lie, but he also cheated. | O yalnız yalan demədi, həm də aldatdı. |
| Not only is she smart, but she is also hardworking. | O həm ağıllıdır, həm də çalışqandır. |

---

## 3. "Only" expressions

| Trigger | Example |
|---------|---------|
| Only when | Only when I saw her did I understand. |
| Only after | Only after the meeting did we find out. |
| Only then | Only then did he realize his mistake. |
| Only by | Only by working hard can you succeed. |
| Only if | Only if you study will you pass. |

---

## 4. "So/Such... that" and "So + adjective"

**Structure:** So + adj/adv + aux + S + V

| Example | Azerbaijani |
|---------|-------------|
| So tired was she that she fell asleep immediately. | O qədər yorğun idi ki, dərhal yuxuya getdi. |
| So great was the pressure that he quit. | Təzyiq o qədər böyük idi ki, o istefa verdi. |

---

## 5. Conditional Inversion (if-siz şərt)

Formal yazıda "if" buraxıla bilər — inversion işlənir:

| With "if" | Without "if" (Inverted) |
|-----------|------------------------|
| If I were you... | **Were I** you... |
| If he had known... | **Had he** known... |
| If this should happen... | **Should this** happen... |

---

## 6. "Nor / Neither"

**Structure:** Nor/Neither + aux + S + V

| Example | Azerbaijani |
|---------|-------------|
| I don't smoke. Nor do I drink. | Siqaret çəkmirəm. İçki də içmirəm. |
| She can't swim. Neither can he. | O üzə bilmir. O da bilmir. |

---

## Quick Summary

```
Negative start:     Never / Rarely / Seldom + aux + S + V
Not only:           Not only + aux + S + V, but S also + V
Only expressions:   Only when/after/then + aux + S + V
So + adj:           So tired + aux + S + V
Conditional:        Were I / Had he / Should this + ...
Neither/Nor:        Nor + aux + S + V
```

> Inversion — cümləyə rəsmi, dramatik ton verir. Danışıq dilindən qaçınılır.
