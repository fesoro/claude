# There & It Constructions

---

## 1. "There is / There are" — Varlığı bildirmək

**Structure:** There + be + noun (+ place/time)

| Example | Azerbaijani |
|---------|-------------|
| **There is** a book on the table. | Masanın üstündə bir kitab var. |
| **There are** three people waiting. | Üç nəfər gözləyir. |
| **There was** a problem yesterday. | Dünən bir problem var idi. |
| **There will be** a meeting tomorrow. | Sabah görüş olacaq. |
| **There has been** a change in plans. | Planda dəyişiklik oldu. |
| **Is there** any milk left? | Süd qaldı? |
| **There isn't** enough time. | Vaxt kifayət etmir. |

### With modal verbs:
| Example | Azerbaijani |
|---------|-------------|
| There **must be** a mistake. | Mütləq bir səhv var. |
| There **might be** a solution. | Bir həll ola bilər. |
| There **should be** more options. | Daha çox seçim olmalıdır. |

### Common mistakes:
| Wrong | Correct |
|-------|---------|
| There is many people. | There **are** many people. |
| It is a book on the table. | **There is** a book on the table. |
| There have a problem. | There **is** a problem. |

## 2. "It" — Formal subyekt (Dummy Subject)

Real subyekti cümlənin sonuna atmaq üçün "it" formal subyekt kimi işlənir.

### a) Hava, vaxt, məsafə

| Example | Azerbaijani |
|---------|-------------|
| **It** is raining. | Yağış yağır. |
| **It** is 5 o'clock. | Saat 5-dir. |
| **It** is Monday. | Bazar ertəsidir. |
| **It** is 10 km from here. | Buradan 10 km məsafədədir. |

### b) "It + be + adj + to-infinitive"

**Structure:** It is + adj + (for sb) + to + verb

| Example | Azerbaijani |
|---------|-------------|
| **It is** important **to** study regularly. | Müntəzəm oxumaq vacibdir. |
| **It is** difficult **for me to** understand. | Mənim üçün başa düşmək çətindir. |
| **It is** easy **to** make mistakes. | Səhv etmək asandır. |
| **It was** kind **of you to** help. | Kömək etmək sənin lütfkarlığın idi. |

### c) "It + be + adj + that-clause"

| Example | Azerbaijani |
|---------|-------------|
| **It is** clear **that** he lied. | Aydındır ki, o yalan dedi. |
| **It is** surprising **that** she passed. | Təəccüblüdür ki, o keçdi. |
| **It is** likely **that** prices will rise. | Güman ki, qiymətlər qalxacaq. |
| **It is** essential **that** you be on time. | Vaxtında olmaq vacibdir. |

### d) "It seems / appears / turns out"

| Example | Azerbaijani |
|---------|-------------|
| **It seems that** he forgot. | Görünür, o unutdu. |
| **It appears that** there was an error. | Görünür, bir xəta var idi. |
| **It turns out that** she was right. | Məlum oldu ki, o haqlı idi. |
| **It looks like** it will snow. | Qar yağacaq kimi görünür. |

### e) "It takes" — vaxt / səy

| Example | Azerbaijani |
|---------|-------------|
| **It takes** 2 hours to get there. | Ora çatmaq 2 saat çəkir. |
| **It took** me all day to finish. | Bitirmək mənə bütün günü apardı. |
| **It will take** time to recover. | Sağalmaq vaxt alacaq. |

---

## 3. "There" vs "It" — Fərq

| There | It |
|-------|----|
| Bir şeyin **varlığını** bildirir | Bir şeyin **xüsusiyyətini** bildirir |
| There is a problem. (Bir problem var.) | It is a serious problem. (Bu ciddi bir problemdir.) |
| There are 5 people here. | It is crowded here. |

---

## Quick Reference

```
There is/are:       Varlığı bildirmək
It is + adj + to:   It is hard to learn.
It is + adj + that: It is clear that...
It seems/appears:   It seems that...
It takes:           It takes 2 hours to...
Weather/time:       It is raining / It is 5 o'clock.
```
