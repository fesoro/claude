# İsmlər (Nouns)

## Səviyyə
A1 (əsas) → B2 (mürəkkəb)

## İsmlər Nədir?
İsmlər şeylərin, insanların, yerlərin, anlayışların adıdır: book, teacher, Baku, happiness.

---

## İsm Növləri

| Növ | Nümunə |
|-----|--------|
| **Sayıla bilən (Countable)** | book, car, apple, teacher |
| **Sayıla bilməyən (Uncountable)** | water, money, information, advice |
| **Xüsusi ad (Proper)** | Baku, John, Azerbaijan |
| **Ümumi ad (Common)** | city, person, country |
| **Mücərrəd (Abstract)** | love, happiness, freedom |
| **Toplanma (Collective)** | team, family, group |
