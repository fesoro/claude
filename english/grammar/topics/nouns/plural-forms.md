# Cəm Formaları (Plural Forms)

## Səviyyə
A1

---

## Əsas Qaydalar

| Qayda | Nümunə |
|-------|--------|
| Əksər ismlər: + s | book → books, car → cars |
| -s, -sh, -ch, -x, -z ilə bitən: + es | bus → buses, match → matches, box → boxes |
| Samit + y: y → ies | city → cities, baby → babies |
| Sait + y: + s | day → days, key → keys |
| -f / -fe: f → ves | leaf → leaves, wife → wives |
| -o: + s və ya + es | photo → photos, tomato → tomatoes |

## Azərbaycanlıların Tez-tez Etdiyi Səhvlər

- ❌ peoples (ümumi mənada) → ✅ people (cəmi artıq var)
- ❌ childs → ✅ children
- ❌ informations → ✅ information (uncountable)
- ❌ advices → ✅ advice (uncountable)
- ❌ womans → ✅ women
