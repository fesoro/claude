# Fellər (Verbs)

## Səviyyə
A1-C2

---

## Fell Növləri

| Növ | İzah | Nümunə |
|-----|------|--------|
| **Regular (Düzgün)** | -ed ilə keçmiş zaman | work → worked |
| **Irregular (Qeyri-düzgün)** | Xüsusi keçmiş forma | go → went → gone |
| **Stative (Vəziyyət)** | Davam edən hal bildirmir | know, like, want |
| **Action (Hərəkət)** | Davam edən hal bildirir | run, eat, write |
| **Transitive** | Obyekt alır | She **read** a book. |
| **Intransitive** | Obyekt almır | He **slept**. |
| **Linking** | Subyekti predikatla birləşdirir | She **seems** happy. |
| **Phrasal** | Zərflə birlikdə | give up, look after |
