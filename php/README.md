# PHP / Laravel Backend Knowledge Base

Bu folder PHP backend developer üçün strukturlaşdırılmış öyrənmə materiallarını əhatə edir — fundamental PHP biliklərindən enterprise arxitekturasına qədər.

**Hədəf**: 5+ il təcrübəli PHP/Laravel developer-lər üçün dərin biliklər.

## Hansını seçim?

```

  → topics/ (Junior section)

Interview hazırlığı?
  → interview/ (01-13 arası, sonra use-cases/)

Konkret pattern öyrənmək istəyirsən?
  → topics/ (level-ə görə axtarış)

Real produksiyon problemi var?
  → use-cases/ (01-60)

DB arxitekturası öyrənmək?
  → db-designs/ (şirkət adına görə axtarış)
```
