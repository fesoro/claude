## Oxuma Yolları (Reading Paths)

### PHP Fundamentals Path (Junior → Middle)
`01-08` (dil əsasları) → `09-13` (magic methods, enums, attributes) → `14-17` (PSR-7/14/middleware, autoloading) → `18-23` (Composer, service container)

### Laravel Developer Path (Middle)
`20-21` (service provider, Laravel internals) → `22-23` (Symfony DI, container comparison) → `24-27` (Telescope, Auth, Livewire, Artisan) → `28` (package development) → `37-42` (Horizon, GraphQL, notifications, scheduler)

### PHP Expert Path (Senior → Architect)
`29-30` (PHP 8.3/8.4, streams) → `44-46` (JIT, process model, OPcache) → `47-49` (profiling, memory, FPM) → `50-51` (Fibers, long-running) → `55-57` (Octane, Swoole, execution models)

### Interview Hazırlığı Path
`interview/01-03` (core + fundamentals + DB) → `interview/04-05` (queues + advanced Laravel) → `interview/06` (internals + memory) → `interview/07-09` (ecosystem + modern PHP + async)
