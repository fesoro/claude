# Soft Skills for Senior+ Engineers

Senior müsahibəsində texniki bilik kifayət etmir. Recruiter-lər və hiring manager-lər bu xüsusiyyətləri axtarır:

- **Kommunikasiya** — qeyri-texniki adamlarla danışa bilmək, yazılı aydınlıq
- **Liderlik** — junior-ları böyütmək, texniki qərarların sürücüsü olmaq
- **Etibar** — tərəddüdsüz qərar vermək, amma feedback qəbul etmək
- **Konflikt idarəsi** — razılaşmamaq, amma hörmət saxlamaq
- **Görünürlük** — rəhbərlik, stakeholder-lər ilə effektiv danışmaq

Bu folder bunların hər birini addım-addım, real template, skript və İngilis frazalar vasitəsilə öyrədir.
